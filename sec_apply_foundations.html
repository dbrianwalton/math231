Section 5.1 Foundational Principles

Applications of the derivative use knowledge about the derivative function to create conclusions about the original function. The derivative is a calculation that is specific to individual points. Most of the applications focus on properties over an interval or possibly the entire domain. We need some additional tools that allow us to cross the bridge between point-wise numerical values to interval-wide behavior. Calculus provides some important and powerful theorems that provide this bridge.

This section introduces these theorems.

  • Intermediate Value Theorem: Continuous functions pass through intermediate values.

  • Extreme Value Theorem: Continuous functions defined on closed intervals have maximum and minimum values.

  • Fermat's Theorem: At a local extreme value, a function either has a slope of zero or no derivative at all.

  • Mean Value Theorem: A continuous and differentiable function on a closed interval must have an interior point where the instantaneous rate is equal to the average rate of change over the interval as a whole.

The other sections in the chapter will look at how these theorems are used in specific applications.

Subsection 5.1.1 Intermediate Value Theorem

The intermediate value theorem is a theorem about continuous functions. When we defined continuity, it was a property that functions had at individual points. To extend the idea of point-wise continuity to an interval, we require continuity at each point within the interval. For end points, we require one-sided continuity based on a limit coming from values within the interval.

Definition 5.1.1. Continuity on Intervals.

A function \(f\) is continuous on an open interval \((a,b)\) if \(f\) is continuous at every point \(c \in (a,b)\text{,}\)

\begin{equation*} \lim_{x \to c} f(x) = f(c)\text{.} \end{equation*}

We can include the left end point \(a\) if \(f\) is right-continuous at \(a\text{,}\)

\begin{equation*} \lim_{x \to a^+} f(x) = f(a)\text{.} \end{equation*}

We can include the right end point \(b\) if \(f\) is left-continuous at \(b\text{,}\)

\begin{equation*} \lim_{x \to b^-} f(x) = f(b)\text{.} \end{equation*}

The property of continuity is directly related to a property of connectedness. A function that is continuous on an interval has a graph that is connected. The intermediate value theorem captures this property by looking at how a graph must pass through all intermediate values.

Theorem 5.1.2. Intermediate Value Theorem (IVT).

If a function \(f\) is continuous on a closed interval \([a,b]\text{,}\) then for every value \(L\) between \(f(a)\) and \(f(b)\text{,}\) there exists some \(c \in (a,b)\) so that \(f(c)=L\text{.}\)

A visual interpretation of the IVT would be to consider the end points of the function creating points \((a,f(a))\) and \((b,f(b))\text{.}\) Now, imagine putting any horizontal line at a height \(y=L\) with \(L\) between \(f(a)\) and \(f(b)\text{.}\) The IVT guarantees that any possible continuous function \(f\) that goes through these two points must cross the horizontal line at some intermediate point.

There are two important applications.

The first is when trying to solve an equation \(f(x)=L\text{.}\) In most actual applications, there is not a way to find the exact solution. But if \(f\) is continuous on an interval where \(f(a)\) and \(f(b)\) are on opposite sides of \(L\text{,}\) such as \(f(a) \lt L \lt f(b)\text{.}\) The IVT guarantees that a solution exists, and the interval \([a,b]\) is said to bracket the solution. If we repeatedly subdivide the interval, always choosing a subinterval that brackets the solution, we may not find the exact solution, but we can find an approximation that is as close as we desire.

The second application results from understanding how a function can cross a particular value. If the function is continuous, it must actually pass through the value of interest. If you know that there is no solution, then it must occur at a discontinuity where the function jumps across the value. This is the justification for doing sign analysis.

Sign analysis is concerned about where a function is positive or negative. These values are separated by the value \(L=0\text{.}\) The IVT says that a function can switch from positive to negative only by passing through points where \(f(x)=0\) or where \(f(x)\) is not continuous. This is why we mark all points on the number line where \(f(x)=0\) (roots) or where \(f(x)\) is not continuous (such as undefined points).

These points determine intervals inside of which we know there are no additional roots or discontinuities. When we test values of \(f(x)\) at individual points inside such an interval, we know that every other point in the same interval must have the same sign as the point we tested.

Subsection 5.1.2 Extreme Value Theorem

The extreme value theorem is concerned with a function's maximum and minimum values. It is possible for a function not to have any maximum value, such as when it has a vertical asymptote and the function increases without bound. Alternatively, it might be that for every value the function achieves, there is always another value that is even higher. In both cases, the function has no actual maximum value.

Definition 5.1.3. Global Extreme Values.

A function \(f\) with domain \(D\) has a maximum value of \(M\) if there is some value \(x_M \in D\) with \(f(x_M) = M\) and for every other \(x \in D\text{,}\) \(f(x) \le M\text{.}\)

A function \(f\) with domain \(D\) has a minimum value of \(m\) if there is some value \(x_m \in D\) with \(f(x_m) = m\) and for every other \(x \in D\text{,}\) \(f(x) \ge m\text{.}\)

Because not all functions have extreme values, we want some conditions that will guarantee their existence. The extreme value theorem tells us that a function continuous on a closed interval provides such a guarantee.

Theorem 5.1.4. Extreme Value Theorem.

Suppose a function \(f\) is restricted to a domain that is a closed interval \([a,b]\text{.}\) If \(f\) is continuous on \([a,b]\text{,}\) then \(f(x)\) has a minimum and a maximum value on \([a,b]\text{.}\)

The reason that the interval must be closed is that when an end-point is not included, say the right end point \(b\text{,}\) we might have a limit \(\displaystyle \lim_{x \to b^-} f(x) = M\) that would be the maximum but the function never actually has that value, \(f(x) \lt M\) for all \(x\) in the domain.

Subsection 5.1.3 Derivatives at Extreme Values

In addition to considering the highest and lowest value of a function across the entire domain, we are also interested in local or relative extreme values. Rather than considering the entire domain, local extreme values are the highest or lowest values among points immediately to the left or right.

Definition 5.1.5. Local Extreme Values.

A function \(f\) has a local or relative maximum at \(x=c\) if there exists \(\delta \gt 0\) so that for all \(x \in (c-\delta,c+\delta)\text{,}\) \(f(x)\) is defined and \(f(x) \le f(c)\text{.}\)

A function \(f\) has a local or relative minimum at \(x=c\) if there exists \(\delta \gt 0\) so that for all \(x \in (c-\delta,c+\delta)\text{,}\) \(f(x)\) is defined and \(f(x) \ge f(c)\text{.}\)

Derivatives give us information about slope at a point by finding the limit of slopes connecting nearby points. The consequence of this is that a slope also gives information about those nearby points. The idea of the derivative had started before calculus was formally developed by Newton and Leibniz. Fermat realized a connection between what we now call derivatives and local extremes.

Theorem 5.1.6. Fermat's Theorem.

If \(f(x)\) has a local extreme at \(x=c\) and is differentiable at \(x=c\text{,}\) then \(f'(c)=0\text{.}\)

Proof.

Suppose that \(f(x)\) has a local maximum at \(x=c\text{.}\) Then for \(c-\delta x \lt c\text{,}\) we know \(f(x) \lt f(c)\) and \(x \lt c\) so that

\begin{equation*} \frac{f(x)-f(c)}{x-c} \gt 0\text{.} \end{equation*}

Using a limit as \(x \to c^-\text{,}\) we find

\begin{equation*} \frac{df}{dx}\Big|_{c^-} = \lim_{x \to c^-} \frac{f(x)-f(c)}{x-c} \ge 0 \end{equation*}

or does not exist. Meanwhile \(c \lt x \lt c+\delta\text{,}\) we know \(f(x) \lt f(c)\) and \(x \gt c\) so that

\begin{equation*} \frac{f(x)-f(c)}{x-c} \lt 0\text{.} \end{equation*}

Using a limit as \(x \to c^+\text{,}\) if it exists, we find

\begin{equation*} \frac{df}{dx}\Big|_{c^+} = \lim_{x \to c^+} \frac{f(x)-f(c)}{x-c} \le 0\text{.} \end{equation*}

If \(f'(c)\) exists, the one-sided derivatives must agree, which can only occur if \(f'(c)=0\text{.}\) If the one-sided derivatives do not exist or if they disagree, then \(f'(c)\) itself does not exist.

Fermat's theorem gives us a way to find all points where a local extreme might occur. We only need to find points where \(f'(x)=0\) or where \(f'(x)\) does not exist (where \(f\) is not differentiable). This is important enough that mathematicians assign these points a special name.

Definition 5.1.7. Critical Points.

A function \(f(x)\) has a critical point at \(x=a\) in the domain of \(f\) if \(f'(a)=0\) or \(f'(a)\) does not exist.

Notice that the critical point is property of \(f(x)\) and is based on the derivative \(f'(x)\text{.}\) We don't say critical point of \(f'(x)\) unless we are talking about points where \(f''(x)=0\) or \(f''(x)\) does not exist.

Subsection 5.1.4 Mean Value Theorem

The average rate of change measures the slope of a line joining two points at the ends of an interval. The derivative measures the instantaneous rate of change at an individual point. The mean value theorem provides a connection between these two ideas.

Before discussing the mean value theorem, we first consider Rolle's theorem. This theorem considers a function that is continuous and differentiable on a closed interval so that the end points are at the same \(y\)-value. Because it is continuous on a closed interval, the extreme value theorem guarantees that this function has both a maximum and a minimum value. If both extremes are at the end points, then the function must be constant on the entire interval. Otherwise, at least one of the extremes must be at an interior point. But at an interior extreme point where the function is differentiable, we must have \(f'(x)=0\text{.}\)

Theorem 5.1.8. Rolle's Theorem.

Suppose \(f(x)\) is a function that is continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) If \(f(a)=f(b)\text{,}\) then there must exist \(c \in (a,b)\) where \(f'(c)=0\text{.}\)

Rolle's theorem is often used to show that \(f'(x)\) must have a zero somewhere between two points \(a\) and \(b\) where \(f(a)=f(b)\text{,}\) usually at two roots of \(f(x)\text{.}\) If a function leaves a \(y\)-value \(f(a)\) and later returns to the same \(y\)-value at \(f(b)\text{,}\) then there must be a point between them where the graph turns around with a horizontal tangent.

The mean value theorem puts Rolle's theorem on a tilt. Instead of guaranteeing a horizontal tangent, the mean value theorem guarantees that there will be a point with a tangent line whose slope matches the slope joining the end-points.

Theorem 5.1.9. Mean Value Theorem.

Suppose \(f(x)\) is a function that is continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) There must exist \(c \in (a,b)\) where

\begin{equation*} f'(c) = \frac{f(b)-f(a)}{b-a}\text{.} \end{equation*}
