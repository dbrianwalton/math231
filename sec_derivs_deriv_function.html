Section 4.3 The Derivative Function

So far, we have talked about the derivative as the slope of the tangent line for a function at a particular point. Because a function theoretically could have a tangent line at each point in the domain, we really should think about the derivative as a function. The most common notation for a derivative of a function is to put a prime on the name of the original function using an apostrophe. For a function named \(f\text{,}\) the derivative would be named \(f'\text{.}\) For a function named \(\mathrm{brian}\text{,}\) the derivative would be named \(\mathrm{brian}'\text{.}\)

In this section, we focus on three core outcomes related to the idea of a derivative as a function.

  • Use the definition of the derivative for a variable point to find the derivative for a polynomial.

  • Use the definition of the derivative for a variable point to find the derivative for a simple algebraic function involving either a quotient or a square root.

  • Based on the graph of a function \(y=f(x)\text{,}\) sketch a reasonable graph of the derivative \(y=f'(x)\text{.}\)

Subsection 4.3.1 Derivative Functions and Polynomials

When we used the definition of the derivative to find the derivative at a point (Subsection 4.2.3), we started by identifying the value of \(f(a)\) and a formula for \(f(a+h)\) in terms of \(h\) so that we could write down a formula for the slope of the secant line joining the points. In that approach, the value \(a\) referred to a specific number.

When calculating the derivative as a function using the definition, we want to consider an arbitrary value in the domain. Instead of a specific value \(a\text{,}\) we will use a variable \(x\) as a placeholder for this arbitrary value. We will compute expressions for \(f(x)\) and \(f(x+h)\) and again calculate a formula for the slope of the secant line joining these points, after which we will work to evaluate the limit as \(h \to 0\text{.}\)

For polynomials, this calculation is fairly straight forward. The main complication is properly expanding the formula for \(f(x+h)\text{,}\) as it is easy for students to forget to multiply out powers properly. The second complication is when you subtract \(f(x)\) from \(f(x+h)\) that you need to subtract the overall formula. When students forget to write this formula inside grouping parentheses, it is too easy to forget that we need to add the opposite of each term, not just the first term.

We will illustrate this with a typical example.

Example 4.3.1. Learning Objective D-5.

Use the definition of the derivative to compute the exact formula \(g'(x)\) for \(g(x)=x^2-2x\text{.}\) (Note that this formula also appeared in the piecewise function in Example 4.2.12.)

Solution.

The basic pattern for using the definition of the derivative will be used:

  • Identify the formula for \(g(x)\text{.}\)

  • Compute \(g(x+h)\) using function substitution.

  • Write down the formula for \(\displaystyle \frac{\Delta g}{\Delta x} = \frac{g(x+h)-g(x)}{h}\text{.}\)

  • Use algebra to rewrite this in order to cancel away division by \(h\) (finding a continuous formula at \(h=0\)).

  • Use continuity at \(h=0\) to evaluate the limit as \(h \to 0\) of the slope of the secant line.

The resulting formula coming after this limit will be the formula for the derivative function.

Here are the steps for this problem. First, find the \(y\)-values for the two points on the secant line.

\begin{align*} g(x) &= x^2 -2x\\ g(x+h) &= (x+h)^2 - 2(x+h)\\ &= (x+h)(x+h) - 2(x+h)\\ &= x^2+2xh+h^2 - 2x - 2h \end{align*}

Second, write down the formula for the slope of the secant line and then simplify the result.

\begin{align*} \frac{\Delta g}{\Delta x} &= \frac{g(x+h)-g(x)}{h}\\ &= \frac{(x^2+2xh+h^2 - 2x - 2h) - (x^2-2x)}{h}\\ &= \frac{\cancel{x^2}+2xh+h^2 - \cancel{2x} - 2h - \cancel{x^2} + \cancel{2x}}{h}\\ &= \frac{2xh+h^2 - 2h}{h} \end{align*}

Notice that each term in the numerator has a common factor of \(h\text{.}\) We factor it out in order to cancel:

\begin{align*} \frac{\Delta g}{\Delta x} &= \frac{2xh+h^2 - 2h}{h}\\ &= \frac{\cancel{h}(2x+h-2)}{\cancel{h}}\\ &= 2x+h-2 \end{align*}

Lastly, the derivative is defined as the limit of this secant line slope as \(h \to 0\text{.}\) Because the final formula removed the discontinuity at \(h=0\text{,}\) we can use substitution to find the limit.

\begin{align*} g'(x) &= \lim_{h \to 0} \frac{\Delta g}{\Delta x}\\ &= \lim_{h \to 0} [2x+h - 2]\\ &= 2x + 0 - 2\\ &= 2x - 2 \end{align*}

Therefore, we have shown, using the definition of the derivative, that for \(g(x)=x^2-2x\text{,}\) we have \(g'(x) = 2x-2\text{.}\)

The example above showed that for \(g(x)=x^2-2x\text{,}\) we have \(g'(x)=2x-2\text{.}\) This formula can be used now to find the slope of the tangent line at any point. For example, we have \(g'(2) = 2(2)-2 = 2\text{.}\) The tangent line at \(x=2\) has slope 2. Similarly, we have \(g'(3)=2(3)-2 = 4\text{,}\) so that the tangent line at \(x=3\) has slope 4. This one formula allows us to find the slope of any tangent line for \(g(x)\text{.}\)

To find the equation of the tangent line, we would use the point-slope formula, \(y=m(x-a)+b\text{.}\) This requires knowing the point \((a,b)\) and the slope \(m\text{.}\) The derivative has provided the slope. The original function is used to find the \(y\)-value of the point. At \(x=2\text{,}\) we have \(g(2)=2^2-2(2)=0\text{,}\) so our point would be \((2,0)\text{.}\) Using the slope \(g'(2)=2\text{,}\) the tangent line would be

\begin{equation*} y=2(x-2)+0 = 2(x-2)\text{.} \end{equation*}

At \(x=3\text{,}\) we have \(g(3)=3^2-2(3) = 3\) with a corresponding point \((3,3)\) and tangent line

\begin{equation*} y=6(x-3)+3\text{.} \end{equation*}

We will have a learning outcome that uses the tangent line later (D-8).

Subsection 4.3.2 Derivative Functions of Algebraic Formulas

We are almost ready to learn differentiation, a method to find derivative formulas using derivative rules instead of the raw definition of a derivative. However, before we get there, we will look at two additional examples of simple algebraic functions that we can find using the derivative's definition. This way we will be able to compare whether differentiation finds correct formulas for a variety of examples. In addition, the methods used in these additional examples will be used to show why the derivative rules in differentiation are valid.

One of our learning objectives is to apply the definition of a derivative to one of these types of formulas. The setup process is identical. We do need to be careful that we use proper algebra in the context of either roots or quotients before we cancel away \(h\) in the slope formula.

The first example is a formula that involves a simple quotient. A rational function is a formula that is the quotient of two polynomials. The main challenge here is (1) writing down the appropriate formula for the slope of the secant line and (2) simplifying the slope quotient by finding a valid common denominator. In addition, it will be helpful to remember that dividing by \(h\) is the same as multiplying by \(\frac{1}{h}\) in order to express our slope formula as a simple quotient rather than a quotient of a quotient. If those steps are done properly, we should be able to factor out \(h\) and cancel it to find a continuous version of the slope formula prior to finding a limit.

Example 4.3.2. Learning Objective D-6 (Rational Functions).

Use the definition of the derivative to compute the exact formula \(f'(x)\) for \(\displaystyle f(x)=\frac{2x}{3x+1}\text{.}\)

Solution.

We will repeat the same steps to find the derivative using the definition as before.

  • Identify the formula for \(f(x)\text{,}\)

    \begin{equation*} f(x) = \frac{2x}{3x+1}\text{.} \end{equation*}

  • Compute \(f(x+h)\) using function substitution.

  • Write down the formula for \(\displaystyle \frac{\Delta f}{\Delta x} = \frac{f(x+h)-f(x)}{h}\text{.}\)

  • Use algebra to rewrite this in order to cancel away division by \(h\) (finding a continuous formula at \(h=0\)).

  • Use continuity at \(h=0\) to evaluate the limit as \(h \to 0\) of the slope of the secant line.

First, we compute

\begin{equation*} f(x+h) = \frac{2(x+h)}{3(x+h)+1}\text{.} \end{equation*}

Notice that each instance of \(x\) was replaced by \((x+h)\text{.}\) We will rewrite this as

\begin{equation*} f(x+h) = \frac{2x+2h}{3x+3h+1}\text{.} \end{equation*}

Next, we write down the formula for \(\displaystyle \frac{\Delta f}{\Delta x} = \frac{f(x+h)-f(x)}{h}\text{,}\)

\begin{equation*} \frac{\Delta f}{\Delta x} = \frac{\frac{2x+2h}{3x+3h+1}-\frac{2x}{3x+1}}{h}\text{.} \end{equation*}

This is where we have a quotient of a quotient. We want to rewrite this so that the difference on the top is combined to a single quotient, and instead of dividing by \(h\text{,}\) we multiply by \(\frac{1}{h}\text{.}\) The first fraction has a denominator of \(3x+3h+1\) and the second has a denominator of \(3x+1\text{.}\) The temptation is to add \(3h\) to get a common denominator, but this is not valid.

The only way we can leave a quotient's value unchanged is to multiply the top and bottom by the same thing. In this case, we will create a new common denominator that is the product of the two original denominators. This is the motivation for writing

\begin{align*} \frac{\Delta f}{\Delta x} &= \frac{\frac{2x+2h}{3x+3h+1}-\frac{2x}{3x+1}}{h}\\ &= \frac{1}{h} \cdot \left( \frac{(2x+2h)(3x+1)}{(3x+3h+1)(3x+1)} - \frac{2x(3x+3h+1)}{(3x+1)(3x+3h+1)}\right)\\ &= \frac{1}{h} \cdot \left( \frac{(6x^2+2x+6xh+2h)-(6x^2+6xh+2x)}{(3x+3h+1)(3x+1)}\right)\\ &= \frac{1}{h} \cdot \left( \frac{2h}{(3x+3h+1)(3x+1)}\right) \end{align*}

At this point the factor of \(h\) on the right cancels division by \(h\) on the left, leaving

\begin{equation*} \frac{\Delta f}{\Delta x} = \frac{2}{(3x+3h+1)(3x+1)}\text{,} \end{equation*}

which is continuous at \(h=0\text{.}\)

The derivative \(f'(x)\) will be the limit of the slope of the secant line as \(h \to 0\text{.}\)

\begin{align*} f'(x) &= \lim_{h \to 0} \frac{\Delta f}{\Delta x}\\ &= \lim_{h \to 0} \frac{2}{(3x+3h+1)(3x+1)}\\ &= \frac{2}{(3x+3(0)+1)(3x+1)}\\ &= \frac{2}{(3x+1)^2} \end{align*}

That is, we have found that for \(\displaystyle f(x)=\frac{2x}{3x+1}\text{,}\) we have a derivative \(\displaystyle f'(x)=\frac{2}{(3x+1)^2}\text{.}\)

Any simple quotient will follow this basic pattern. The other example that we will want to explore occurs with square roots. In this type of problem, our temptation is going to be to try to cancel terms that appear in different square roots. For example, if we have \(\sqrt{x+1}-\sqrt{x}\text{,}\) the temptation I am referring to is to think that we can cancel \(\sqrt{x}\) from the first term with \(\sqrt{x}\) in the second term. But algebra does not allow us to separate the \(x+1\) that is inside the square root.

Instead, we will find a new representation, again using multiplication. The process we use is called rationalization using multiplication by a conjugate. The word conjugate refers to expressions that are sums or differences of square roots. You might have learned in algebra about a complex conjugate of a number like \(2+3i\) where the coefficient of \(i\) (in this case 3) has its sign changed to \(2-3i\text{.}\) The symbol \(i\) represents \(\sqrt{-1}\text{,}\) so we were dealing with a sum/difference involving a square root.

A formula like \(\sqrt{x+1} - \sqrt{x}\) can be rewritten by multiplying and dividing by the same expression, in this case the conjugate, which uses the sum of the same terms in the difference:

\begin{equation*} \sqrt{x+1}-\sqrt{x} = \frac{(\sqrt{x+1}-\sqrt{x})(\sqrt{x+1}+\sqrt{x})}{\sqrt{x+1}+\sqrt{x}}\text{.} \end{equation*}

Why would we do this? Every time we multiply conjugates, we get the difference of the squares of the terms: \((a-b)(a+b) = a^2 - b^2\text{.}\) So the numerator of this new formula we have a difference of squares of square roots, meaning the numerator will no longer have any square roots:

\begin{align*} \sqrt{x+1}-\sqrt{x} &= \frac{(\sqrt{x+1}-\sqrt{x})(\sqrt{x+1}+\sqrt{x})}{\sqrt{x+1}+\sqrt{x}}\\ &= \frac{(\sqrt{x+1})^2-(\sqrt{x})^2}{\sqrt{x+1}+\sqrt{x}\\ &= \frac{(x+1)-x}{\sqrt{x+1}+\sqrt{x}\\ &= \frac{1}{\sqrt{x+1}+\sqrt{x}\text{.} \end{align*}

That is, we have eliminated the difference of square roots in the numerator by introducing a sum of square roots in the denominator.

This same strategy will be used to simplify the slope of a secant line based on a function involving a square root.

Example 4.3.3. Learning Objective D-6 (Square Root Functions).

Use the definition of the derivative to compute the exact formula \(f'(x)\) for \(f(x)=\sqrt{4x-1}\text{.}\)

Solution.

Notice how we will use the same general steps to find the slope of the secant line joining \((x,f(x))\) and \((x+h,f(x+h))\text{,}\) simplify that formula to a continuous version, and then evaluate the limit in order to find the derivative.

\begin{align*} f(x) &= \sqrt{4x-1}\\ f(x+h) &= \sqrt{4(x+h)-1} = \sqrt{4x+4h-1}\\ \frac{\Delta f}{\Delta x} &= \frac{f(x+h)-f(x)}{h}\\ &= \frac{\sqrt{4x+4h-1} - \sqrt{4x-1}}{h}\\ &= \frac{(\sqrt{4x+4h-1} - \sqrt{4x-1})(\sqrt{4x+4h-1} + \sqrt{4x-1})}{h(\sqrt{4x+4h-1}+\sqrt{4x-1})}\\ &= \frac{(\sqrt{4x+4h-1})^2 - (\sqrt{4x-1})^2}{h(\sqrt{4x+4h-1}+\sqrt{4x-1})}\\ &= \frac{(4x+4h-1) - (4x-1)}{h(\sqrt{4x+4h-1}+\sqrt{4x-1})}\\ &= \frac{4\cancel{h}}{\cancel{h}}(\sqrt{4x+4h-1}+\sqrt{4x-1})}\\ &= \frac{4}{\sqrt{4x+4h-1}+\sqrt{4x-1}}\\ f'(x) &= \lim_{h \to 0} \frac{\Delta f}{\Delta x}\\ &= \lim_{h \to 0} \frac{4}{\sqrt{4x+4h-1} + \sqrt{4x-1}}\\ &= \frac{4}{\sqrt{4x+4(0)-1} + \sqrt{4x-1}}\\ &= \frac{4}{\sqrt{4x-1} + \sqrt{4x-1}}\\ &= \frac{4}{2\sqrt{4x-1}} = \frac{2}{\sqrt{4x-1}} \end{align*}

Subsection 4.3.3 Graphs of Derivative Functions

Because the derivative function \(f'(x)\) measures the slope of tangent lines of another function \(f(x)\text{,}\) we can visualize the graph of \(f'(x)\) by recognizing how the slope of \(f(x)\) itself is changing.

When creating a graph of \(f'(x)\text{,}\) we should pay particular attention to the following features:

  • Where \(f(x)\) is horizontal, the slope of the tangent line is zero. At such a point, \(f'(x)=0\) so that the graph of \(f'(x)\) touches the \(x\)-axis.

  • Where \(f(x)\) has a positive slope, the value of \(f'(x)\) itself is positive so the graph of \(f'(x)\) is above the \(x\)-axis.

  • Where \(f(x)\) has a negative slope, the value of \(f'(x)\) itself is negative and the graph of \(f'(x)\) is below the \(x\)-axis.

  • Where \(f(x)\) is concave up, the slope is increasing, so the graph of \(f'(x)\) will be an increasing function.

  • Where \(f(x)\) is concave down, the slope is decreasing, so the graph of \(f'(x)\) will be an decreasing function.

  • Inflection points where \(f(x)\) changes concavity will be local extreme values for \(f'(x)\text{.}\)

Overall, the idea is to imagine going from left to right on the graph of \(y=f(x)\) and pay attention to the slope of that graph. At each \(x\)-value, we note the value of the slope and record that as the value of \(f'(x)\text{.}\) If the \(f(x)\) graph is concave up, that means the slope is increasing and our \(f'(x)\) function should be rising. If the \(f(x)\) graph is concave down, \(f'(x)\) should be falling. And the closer the graph of \(f(x)\) is to horizontal, the closer the \(f'(x)\) graph should be getting to the \(x\)-axis (\(f'(x)=0\)).

Let us start with a dynamic illustration. The figure below shows a graph of \(y=f(x)\) on the left. A point on the graph is draggable, and at that point, the graph also shows the corresponding tangent line. The graph on the left shows the derivative function \(f'(x)\text{.}\) As you drag from left to right, more of the graph of \(y=f'(x)\) will be shown. Pay attention to how each of the features described above come into play.

Notice how on the far left and far right edges of the graph, the tangent line is almost horizontal so the \(f'(x)\) graph is close to zero on these same edges. Similarly, the graph \(y=f(x)\) has horizontal tangents at its local extreme values at \(x=-2\) and \(x=2\text{,}\) so the \(f'(x)\) graph crosses the axis at these points. On the interval \((-\infty,-2)\text{,}\) the graph of \(f(x)\) has a tangent line with negative slope, so \(f'(x)\) is below the axis on this same interval. You should also notice that the \(f'(x)\) graph has its own local minimum at about \(x \approx -3.5\text{.}\) This is a consequence of the graph \(y=f(x)\) having an inflection point changing from concave down on the far left to concave up. The corresponding point is where the tangent line has the most negative slope.

When doing this by hand, I recommend:

  • Identify where \(f'(x)=0\text{.}\) Mark this on your graph as points on the \(x\)-axis.

  • Determine the intervals where \(f'(x)\) is positive or negative.

  • Identify extreme slopes (inflection points). Mark approximate points on your graph giving the approximate slope at these points. These will become extreme values for \(f'(x)\text{.}\)

  • Determine what happens at the edge of your graph. Is the slope becoming unbounded? Or is the slope approaching zero?

  • Draw a graph joining the points you created earlier with the correct end behavior and correct signs.

Subsection 4.3.4 Tangent Lines and Linear Approximations

In the paragraphs following Example 4.3.1, we emphasized how the derivative function calculates the slope of a tangent line and introduced the idea of using that information to write down the equation of a line. In this subsection, we want to explore this idea in a little more depth.

Recall that a tangent line is the best linear function to approximate a function through a given point. The derivative is the tool that helps us defined how to get the slope of this linear approximation. The resulting linear function is called linearization of the original function around the point where the tangent line was found. Why do we ever want a linearization?

There are many examples where variables in data are related according to complicated relationships. Finding those formulas might be a very complex task. But if we can find a linearization, which only requires knowing an initial value (the point) and the rate of change (the slope), then at least for nearby points we have a simple formula to make calculated predictions.

In physics, the force of resistance to change provided by an elastic object like a spring is actually a complex nonlinear relation. However, like any differentiable function, for small changes from its equilibrium position, the linearization will be quite close to this nonlinear function. This is why physics typically represents springs using a linear force, known as Hooke's law, \(F=-kx\) where \(x\) is the displacement from equilibrium. If we represented \(x\) as a position and a particular position as equilibrium \(x_0\text{,}\) the equation would intead be \(F = -k(x-x_0)\text{.}\) The spring constant \(k\) is actually the derivative or rate of change of the more complex nonlinear force function with respect to displacement.

Finding a linear approximation is nothing more than repeating Learning Objective P-2 (Example 1.1.10). The only difference is that now we know where the slope is coming from. It is coming from the derivative. For the purpose of this learning objective, it might be assumed that the derivative formula has already been found and your task is to use the formula.

Example 4.3.8. Learning Objective D-8 (Formula).

Specific heat is a physical quantity measured for many substances. It is actually a derivative \(\frac{dQ}{dT}\) of the energy of heat \(Q\) (Joules) in a single gram of the substance with respect to the temperature \(T\) (degrees Celsius). An engineer has done analysis on some specific heat data for sapphire and has determined that for the values of interest,

\begin{equation*} \frac{dQ}{dT} \approx 0.7180 + 0.0024 T - 0.000005 T^2\text{.} \end{equation*}

After adding \(Q=4\) Joules to a gram sapphire, the temperature is now \(T=60\) degrees C. Use a linearization of heat with respect to temperature to approximate the value of \(Q\) that is needed to reach \(T=65\) degrees C.

Solution.

Underneath the discussion about physical variables, we understand from the problem that \(Q\) is a function of \(T\text{,}\) say \(Q=f(T)\text{.}\) The rate of change \(\frac{dQ}{dT}\) that is given represents the derivative \(f'(T)\text{,}\)

\begin{equation*} f'(T) = 0.7180 + 0.0024T - 0.000005 T^2\text{.} \end{equation*}

From the given information, we know that \(f(60) = 4\text{.}\) Using the derivative formula, we can also find

\begin{equation*} f'(60) = 0.7180 + 0.0024(60) - 0.000005 (60)^2 = 0.844\text{.} \end{equation*}

The linearization says that for \(T\) close to \(T=60\text{,}\) we have an approximate relation

\begin{equation*} Q = 4 + 0.844(T-60)\text{.} \end{equation*}

Using \(T=65\text{,}\) we find

\begin{equation*} Q \approx 4 + 0.844(65-60) = 8.22\text{.} \end{equation*}

That is, the total heat added to reach 65 degrees Celsius is approximately 8.22 Joules (from its original state) or 4.22 additional Joules (from its current state).

In this solution, the key features I will be looking for is whether you stated a linearization (the tangent line formula), and then whether you used it to answer the question.

Alternatively, you might also be given information about the function and its derivative in other formats. In the following example, information about a function and its derivative are given in a table, and we are asked to use the information to approximate a function value at a point not in the table. Because a linearization gives a good approximation for small deviations from the original point, we choose the closest point to make the linearization.

Example 4.3.9. Learning Objective D-8 (Data).

A population growing in time is observed every five years to record the population \(P\) and the population growth rate \(\frac{dP}{dt}\)(birth rate minus death rate).

\(t\) (yr) \(P\) (thousands) \(\frac{dP}{dt}\) (thousands/yr)
0 10.0 1.9
5 25.0 4.4
10 56.0 8.1
15 102.8 10.0
20 148.4 7.7
Use an appropriate linearization to estimate the population size for \(t=12\text{.}\)

Solution.

The time \(t=12\) is closest to \(t=10\) in the table. We will use use \(t=10\) for our linearization, at which time we have that \(P=56.0\) (thousands of individuals) and \(\frac{dP}{dt}=8.1\) (thousands per year). The tangent line formula is given by

\begin{equation*} P = 56.0 + 8.1\cdot(t-10)\text{,} \end{equation*}

so that the approximate value at \(t=12\) is given by

\begin{equation*} P = 56.0 + 8.1\cdot(12-10) = 72.2\text{.} \end{equation*}

The linearization predicts a population of about 72.2 thousand individuals.
