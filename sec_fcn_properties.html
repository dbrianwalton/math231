Section 2.4 Function Properties

Mathematics, as a field of study, examines many different types of objects and the patterns that they follow. In order to communicate consistently about these patterns, language is created to describe specific attributes and properties. Words in the context of mathematics are associated with specific definitions. Recall how the idea of inverse functions had a conceptual idea of undoing the operations of one another and a formal definition. The definition was a precise logical statement that we had to precisely verify in order to formally know that our functions were inverses.

This pattern will apply to the properties of other objects in mathematics. We will try to create a common understanding of properties but we will also create formal definitions. The logical consequences for how different properties are related will need to follow based on how the definitions are created. Consequently, definitions can feel cumbersome because they are created as much to facilitate the reasoning in this deductive process as they are to capture natural intuition.

In this section, we will focus on learning the intuitive ideas behind definitions for some of the properties of functions, specifically those properties that are used to describe the shapes of function graphs. Most properties are associated with subsets of the domain. Some properties describe what happens at individual points, and we say those properties occur at a given point. Other properties describe what happens over a subset of the domain, and we say those properties occur on a given set.

The learning outcomes represented in this section focus on developing this skill.

  • Describe the graph of a function using mathematical language to characterize where the function is increasing, decreasing, concave up, and concave down, and where the function has extreme values and inflection points.

  • Sketch the graph of a function that is consistent with a provided description based on monotonicity and concavity.

Subsection 2.4.1 Increasing, Decreasing, or Constant

The first group of attributes are classified as describing the monotonicity of a function. The attributes of interest are increasing, decreasing, and constant, and they are properties that a function has on a set. An intuitive understanding of these attributes can come by thinking about moving along a graph of the function restricted to the set in question from left to right. The function is increasing on the set if the \(y\)-values of the graph always go up as you move left to right. The function is decreasing on the set if the \(y\)-values go down. The function is constant on the set if all \(y\)-values are the same.

When functions are smooth, we can think about the graph as having a definite slope at each point (the derivative). One of the major goals in a first-semester of calculus is to make a connection between the derivative and monotonicity. We can associate positive slopes with increasing functions, negative slopes with decreasing functions, and zero slopes with constant functions, but only when these properties are on intervals and not at individual points. The reasoning behind this connection will come later. We are trying to build a sense of how to think about the properties before we get too far into the details.

The formal definition of monotonicity creates a mathematical statement about our intuition of what happens to a \(y\)-value as we move left-to-right.

Definition 2.4.1. Monotonicity of Functions.

Let \(f\) be a function with domain \(D\text{,}\) and let \(S\) be a subset of \(D\) (usually an interval).

  • “\(f\) is increasing on \(S\)” means: For any \(a,b \in S\) with \(a \lt b\text{,}\) \(f(b) \gt f(a)\text{.}\)

    “\(f\) is decreasing on \(S\)” means: For any \(a,b \in S\) with \(a \lt b\text{,}\) \(f(b) \lt f(a)\text{.}\)

    “\(f\) is constant on \(S\)” means: For any \(a,b \in S\text{,}\) \(f(a) = f(b)\text{.}\)

If no set is included (e.g., \(f\) is increasing), then it is understood to apply to the entire domain. If \(f\) is increasing or \(f\) is decreasing (on the domain), we say \(f\) is monotone.

The following figures illustrate the basic ideas.

In Figure 2.4.3, we can imagine moving from left-to-right in the domain. As we move along the graph, the \(y\)-values steadily rise until we reach \(x=-1\text{.}\) Immediately after this point, the graph starts to decrease, so we say the function is increasing on \((-\infty,-1]\text{.}\) The end-point is included because \(f(-1)\) is greater than all values to the left.

Now, starting at \(x=-1\) and continuing to the right, the graph continues to drop until \(x=2\text{.}\) The sudden change in slope at \(x=0\) is irrelevant. We say that the function is decreasing on \([-1,2]\text{.}\) Again, both end-points are included because even when the graph is highest at \(x=-1\) and lowest at \(x=2\) and decreasing everywhere in between.

Finally, starting at \(x=2\) and continuing to the right, the graph is now rising apparently forever (based on the trend as the graph leaves the viewing window). We say that the function is increasing on \([2,\infty)\text{.}\)

In Figure 2.4.4, we again imagine moving from left-to-right in the domain. As we move along the graph, the \(y\)-values steadily rise until we reach \(x=-1\text{.}\) However, this function has a break or discontinuity. The \(y\)-value at \(x=-1\) is below the values immediately to the left. Consequently, we say the function is increasing on \((-\infty,-1)\text{,}\) not including the end-point.

Starting at \(x=-1\text{,}\) the graph immediately jumps down to a curve which is then rising. We ignore the point at \(x=-1\text{,}\) but continue to the right until \(x=2\) where the graph has another break. The value at \(x=2\) is below the points immediately to the left, so we must not include \(x=2\) in the set where the function is increasing. The function is increasing on \((-1,2)\text{.}\)

Now we start at \(x=2\text{.}\) The graph immediately drops to a curve, which then continues to decrease. Since \(f(2)\) is greater than all of these values \(f(x)\) with \(x \gt 2\text{,}\) we will include the end-point at 2 in our interval for decreasing. The graph continues to decrease until the graph stops at \(x=5\text{.}\) The value at \(x=5\) is lower than points to its left, so this point will also be included. The function is decreasing on \([2,5]\text{.}\)

Subsection 2.4.2 Concavity

While the monotonicity of a function describes how it rises or falls, concavity describes how a function's graph bends or curves. A straight line has no concavity. If a curve bends upward, we say the graph is concave up. If a curve bends downward, we say the graph is concave down.

One way to think about concavity is about a tangent line at a point. The tangent line captures the rate of change of the function at that point. If we took a short segment of the tangent line centered at the point of contact, and this segment would be above the graph immediately to the left and to the right, we would have to bend the segment downward to match our graph. This captures the idea of locally being concave down. If the short segment was below the graph immediately to the left and right, we would have to bend the segment upward to match the graph. This captures the idea of locally being concave up. We just need to see how big we can make an interval to include all the points that locally have such a property.

Another way to think about concavity in terms of a tangent line is by thinking about how the slope of the tangent line changes as you consider points going from left to right. The slope of a function \(f(x)\) at a point \(x=a\) is represented by the symbol \(\frac{df}{dx}\Big|_{a}\text{.}\) Because this slope changes as we consider different points, this defines a new function \(f'(x)\) called the derivative function. We will define these objects more carefully later.

A function is concave up if its slope is increasing, and it is concave down if its slope is decreasing. In calculus, this actually serves as our basic definition. We will only define concavity on intervals so that we are dealing with connected sets.

Definition 2.4.5. Concavity of a Function.

Let \(f(x)\) be a function, let \(S\) be an interval that is a subset of the domain of \(f\text{,}\) and suppose \(f'(x)\) is defined on \(S\) (there is a clear sense of slope on \(S\)).

  • “\(f\) is concave up on \(S\)” means the slope (\(f'(x)\)) is increasing on \(S\text{.}\)

  • “\(f\) is concave down on \(S\)” means the slope (\(f'(x)\)) is decreasing on \(S\text{.}\)

  • “\(f\) is linear on \(S\)” means the slope (\(f'(x)\)) is constant on \(S\text{.}\)

We will return to the definition near the end of the semester when we have defined exactly what \(f'(x)\) means and how to calculate it. For our purposes, we will focus on the interpretation of which direction the curve is bending. An interval of concavity must end as soon as the bending direction switches. We will also only consider intervals on which the graph itself is connected, because we will later learn that \(f'(x)\) can not be defined at a discontinuity. We will also not include points where the slope suddenly changes, again because \(f'(x)\) won't exist at these points.

Definition 2.4.6. Inflection Point.

Let \(f\) be a function. “\(f\) has an inflection point at \(x=a\)” means that \(f\) is continuous at \(x=a\) (not broken), has a tangent line, and the concavity changes at \(x=a\text{.}\)

Example 2.4.7.

The following figures illustrate the basic ideas.

The function in Figure 2.4.8 illustrates several intervals of concavity. We start going left to right. When we first encounter the graph around \(x=-5\text{,}\) we can see that the graph is locally bending upwards. That is, the graph is above a tangent line on both sides. The curve continues to bend upward until the corner at \(x=-1\text{.}\) Assuming the graph has a similar behavior to the left of the viewing window, we would say \(f\) is concave up on \((-\infty,-1)\text{.}\) The point \(x=-1\) is not included because there is no well-defined slope at \(x=-1\text{.}\)

Going from \(x=-1\) to \(x=0\text{,}\) the graph is a straight line. That is, \(f\) has no concavity on \((-1,0)\text{.}\) Instead, we say \(f\) is linear on \([-1,0]\text{.}\)

Now, starting at \(x=0\) and going to the right, we see the function is curving downward. Far to the right, however, we see that the curve begins to curve back upward. We need to identify the point where the curving direction switches. In this example, it occurs at \(x=4\text{.}\) If we could draw a tangent line at \(x=4\text{,}\) the graph is below the tangent line for \(x \lt 4\) and above the tangent line for \(x \gt 4\text{.}\) Consequently, we say \(f\) is concave down on \((0,4]\) and concave up on \([4,\infty)\) (assuming the shape pattern continues outside the window).

The function in Figure 2.4.9 provides an additional example. On the far left, the graph is bending upwards, so we know there will be an interval that is concave up. Looking at the graph at \(x=-2\text{,}\) we can see the graph is now bending downwards, so there will be an interval that is concave down.

We check between these points to identify the inflection point. From a graph, we can only give a reasonable guess. The graph appears to be roughly straight for \(x\)-values close to \(x=3\text{.}\) A tangent line at \(x=3\) appears consistent with having the graph above the line on the left and below the line on the right. So \(x=3\) being the inflection point is a reasonable guess. We conclude that \(f\) is concave up on \((-\infty,3]\text{.}\)

We earlier noted that the graph is concave down around \(x=2\text{.}\) Looking further to the right, we should notice that the graph is concave up around \(x=0\text{.}\) It appears that an inflection point is at or near \(x=-1\text{.}\) So we conclude \(f\) is concave down on \([-3,-1]\text{.}\)

Using similar reasoning, we decide there are inflection points at \(x=1\) and at \(x=3\text{.}\) We further conclude that \(f\) is concave up on \([-1,1]\text{,}\) concave down on \([1,3]\text{,}\) and concave up on \([3,\infty)\text{.}\)

Subsection 2.4.3 Extreme Values

We introduced inflection points as places where a continuous function with a tangent line change concavity. There should be a similar idea for where a function changes monotonicity. Suppose a function is increasing and then at a point \(x=a\) changes to decreasing. If we were to draw that picture, we would realize that \(f(a)\) must be greater than the values to the left (because it is increasing), but also greater than the values to the right (because it is decreasing). That makes the point a maximum value on an interval that extends to the left and right of \(x=a\) (a neighborhood of \(x=a\)).

Continuous functions that change monotonicity have reached an extreme value. However, this value is only guaranteed to be a maximum or minimum on an interval subset of the domain. We call such points local extreme values. They are the peaks or valleys of the function. The mathematical definition focuses only on the function reaching the highest or lowest value within an interval rather than requiring monotonicity.

Definition 2.4.10. Local Extreme Values (Relative Extreme Values).

Let \(f\) represent a function and suppose there is some neighborhood of \(x=a\text{,}\) an interval \((a-\delta, a+\delta)\) with \(\delta \gt 0\text{,}\) that is a subset of the domain of \(f\text{.}\)

  • “\(f\) has a local (relative) maximum at \(x=a\)” means for each \(x \in (a-\delta, a+\delta)\text{,}\) \(f(a) \ge f(x)\text{.}\) (The highest value of \(f\) in the neighborhood occurs at \(x=a\text{.}\))

  • “\(f\) has a local (relative) minimum at \(x=a\)” means for each \(x \in (a-\delta, a+\delta)\text{,}\) \(f(a) \le f(x)\text{.}\) (The lowest value of \(f\) in the neighborhood occurs at \(x=a\text{.}\))

We are also interested in the most extreme values across the entire domain. These extreme values are called global extreme values or absolute extreme values.

Definition 2.4.11. Global Extreme Values (Absolute Extreme Values).

Let \(f\) represent a function with a domain \(D\text{.}\)

  • “\(f\) has a global (absolute) maximum at \(x=a\)” means for each \(x \in D\text{,}\) \(f(a) \ge f(x)\text{.}\)

  • “\(f\) has a global (absolute) minimum at \(x=a\)” means for each \(x \in D\text{,}\) \(f(a) \le f(x)\text{.}\)

Subsection 2.4.4 Describing the Shape

A description of the graph of a function includes identifying the monotonicity and concavity of the function, the inflection points and extreme values, and other important features of the graph. These other features might include characterizing discontinuities or asymptotes, ideas we will encounter later when we discuss limits.

Our learning objectives focus on how we communicate about mathematical objects. Communication occurs in sentences, so we need to practice writing complete sentences that properly use mathematical language in their context. The functions are the objects under study, so they will play the role of the nouns in our sentences. When discussing monotonicity or concavity, the definitions are in terms of subsets of the domain, so the sentences should use a prepositional phrase involving on a set. When discussing inflection points or extreme values, the definitions are in terms of individual points, so the sentences should use a prepositional phrase involving at a value.

Example 2.4.12. Learning Objective F-10.

The graph of a function \(g(x)\) is illustrated in the figure below. Assume that behavior at the edge of the viewing window continues indefinitely. Describe \(g(x)\text{,}\) using separate sentences to describe monotonicity and concavity on intervals chosen as large as appropriate. Estimate the location of inflection points and extreme values if not obvious.

Solution.

First, we describe the monotonicity. Looking at the graph, we see it rises, falls, and rises, so we will have three intervals. The function \(g\) is increasing on \([-4,0]\text{,}\) decreasing on \([0,2]\text{,}\) and increasing on \([2,\infty)\text{.}\) End-points were all included because the function values at those points were the lowest or highest value on their intervals.

\(g\) has a global minimum at \(x=-4\) and a global maximum at \(x=0\) because it appears the right-edge of the graph is leveling out fast enough that the \(y\)-value will not get higher than \(y=4\text{.}\) \(g\) also has a local minimum at \(x=2\text{.}\)

Second, we describe the concavity. On the far left, the graph starts concave up to the right of \(x=-4\text{.}\) To the left of \(x=0\text{,}\) the graph is concave down. We can estimate an inflection point at \(x=-2\text{.}\) So \(g\) is concave up on \((-4,-2]\) and concave down on \([-2,0)\text{.}\) End-points are not included because the functions does not have clear tangent lines at those points.

The graph is clearly concave up around \(x=2\text{.}\) This extends leftward to \(x=0\text{.}\) By the time we get to \(x=4\text{,}\) the graph is concave down. An inflection point can be estimated to be at \(x=3\text{.}\) So \(g\) is concave up on \((0,3]\) and concave down on \([3,\infty)\text{.}\)

The reverse skill is also important. Given a description of the behavior of a function, you should be able to sketch a graph of a function that matches that behavior. There are multiple ways to think about creating a graph. See if the following suggestions help or if you can think of other ways.

One perpective is to think about monotonicity and concavity as pairs of information. There are four basic pairs: increasing and concave up, increasing and concave down, decreasing and concave up, and decreasing and concave down. Each pair corresponds to the shape of the four quadrants of a circle.

Using the information about monotonicity and concavity, we can identify intervals with these pairs of shape information. If there are intervals with constant or linear graphs, we add those as well. We can then construct a graph by joining small arcs of graph shapes together, with each arc matching the shape pair information. If the graph is continuous, the arcs need to join together at their end-points. Otherwise, there could be breaks in the graph.

Example 2.4.18. Learning Objective F-11.

A function \(f\) is increasing on \((-\infty,-2]\text{,}\) on \([0,2]\text{,}\) and on \([4,\infty)\text{.}\) The function is decreasing on \([-2,0]\) and on \([2,4]\text{.}\) \(f\) is concave up on \((-\infty,-1]\) and on \([3,\infty)\) and concave down on \([-1,3]\text{.}\) Draw a graph of a continuous function consistent with this information.

Solution.

The first two sentences give us information about monotonicity, and the last sentence gives us information about concavity. We need to identify intervals of the paired information. One way to get this paired information is to create adjacent number line information showing monotonicity and concavity.

Reading this paired information from left-to-right, we can identify intervals of basic shapes.

  • \((-\infty,-2]\text{:}\) increasing and concave down

  • \([-2,-1]\text{:}\) decreasing and concave down

  • \([-1,0]\text{:}\) decreasing and concave up

  • \([0,1]\text{:}\) increasing and concave up

  • \([1,2]\text{:}\) increasing and concave down

  • \([2,\infty)\text{:}\) decreasing and concave down

We might just draw the corresponding shapes below our earlier number line.

Once we know the individual shapes, we can draw the full graph. Because we were told to draw a graph of a continuous function, we will join the shapes together at the points of change. At \(x=-2\text{,}\) we reach a local maximum; at \(x=0\text{,}\) we reach a local minimum; and at \(x=2\text{,}\) we reach another local maximum. At \(x=-1\) and at \(x=1\text{,}\) we have inflection points because the concavity changes. None of our given information tells us about \(y\)-values of the graph, so we will not show a vertical scale.

A second approach to creating such a graph is to focus first on the monotonicity information to create proposed locations of extreme values. We might imagine a graph consisting of only linear segments for this. Then, we add concavity information by identifying intervals on the subdomain in which we need to add curvature. We replace the linear segments with curved arcs that preserve the monotonicity we originally found.

We repeat the previous example using this alternative approach.

Example 2.4.22. Learning Objective F-11.

A function \(f\) is increasing on \((-\infty,-2]\text{,}\) on \([0,2]\text{,}\) and on \([4,\infty)\text{.}\) The function is decreasing on \([-2,0]\) and on \([2,4]\text{.}\) \(f\) is concave up on \((-\infty,-1]\) and on \([3,\infty)\) and concave down on \([-1,3]\text{.}\) Draw a graph of a continuous function consistent with this information.

Solution.

We use the monotonicity information first. The graph is increasing on \((-\infty,-2]\text{,}\) so our preliminary graph will use an increasing line that stops at \(x=-2\text{.}\) This point will become our local maximum. The graph the decreases on \([-2,0]\text{,}\) so we add a new line segment that is decreasing until \(x=0\) for a local minimum. We then draw an increasng line segment between \(x=0\) and \(x=2\) to give a second local maximum. Finishing the preliminary graph, we draw a decreasing line extending to the right from \(x=2\text{.}\) On a paper solution, we might only have drawn the locations of the local extrema.

Now, we turn our attention to concavity. The function needs to be concave down on \((-\infty,-1]\text{.}\) At \(x=-1\text{,}\) our preliminary graph is using the decreasing line. We will identify a \(y\)-value between the maximum at \(x=-2\) and the minimum at \(x=0\) as the location of our inflection point. It can often be chosen to be on the line itself, but it does not need to be. To the left of this point, our preliminary graph is increasing and then decreasing. We convert this to a curved arc that is concave down.

Next, the graph is concave up on \([-1,1]\text{.}\) So we find another point at \(x=1\) between the minimum at \(x=0\) and maximum at \(x=2\text{.}\) The preliminary graph is piecewise linear, decreasing on \([-1,0]\) and increasing on \([0,1]\text{.}\) We convert this to a curved arc that is concave up. To the right of this, the graph is concave down on \([-1,\infty)\text{.}\) The preliminary graph is piecewise linear that we need to convert to a curved arc.
