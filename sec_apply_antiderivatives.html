Section 5.3 Antiderivatives

Having learned that so much of a function's behavior can be described by knowing the derivative (and subsequently the second derivative), it is hopefully not too surprising to learn that knowing the derivative alone can be used to characterize almost all of the function.

One of the major themes in mathematics is the idea of inverse operations. When we start with a function \(g(x)\) and use it to find the derivative \(g'(x)\text{,}\) this action \(\frac{d}{dx}[g(x)] = g'(x)\) is called differentiation. We can almost do the action in reverse. That, is starting with a function \(f(x)\text{,}\) the action used to find a function \(F(x)\) so that our function \(f(x)\) is the derivative of \(F(x)\) is called antidifferentiation and we say that \(F(x)\) is an antiderivative of \(f(x)\text{.}\)

We have two learning outcomes connected to this idea.

  1. Given a graph of a derivative \(g'(x)\text{,}\) describe and sketch a graph of an antiderivative function by using the first and second derivatives tests for monotonicity and concavity.

  2. Given a polynomial \(p(x)\text{,}\) find the antiderivative formula for a polynomial \(f(x)\) so that \(f'(x)=p(x)\) and passes through a given initial point.

Subsection 5.3.1 Overview of Antiderivatives

When we learned about inverse functions, we learned that inverse functions \(f\) and \(f^{-1}\) have the property that the input/output pairing change places:

\begin{equation*} f(x)=y \qquad \Leftrightarrow \qquad f^{-1}(y) = x\text{.} \end{equation*}

If we think of the differentiation operator in a similar way, a function \(f(x)\) and its derivative \(f'(x)\) can be interpreted as an input/output pair:

\begin{equation*} \frac{d}{dx}[f(x)] = f'(x)\text{.} \end{equation*}

The original function \(f(x)\) is the input to the operator and its derivative is the output. An inverse to differentiation should therefore reverse the ordering between these:

\begin{equation*} \frac{d}{dx}^{-1}[f'(x)] = f(x)\text{.} \end{equation*}

Unfortunately, differentiation is not one-to-one and so can not have a true inverse. For example, consider \(f(x) = x^4 - 3x^2 + 4\) and \(g(x)=x^4 - 3x^2 - \pi\text{.}\) Because the formulas involving \(x\) are the same and the derivative of every constant is zero, we have both functions creating the same derivatives, \(f'(x)=g'(x)\)

\begin{gather*} \frac{d}{dx}[x^4-3x^2+4] = 4x^3-6x\\ \frac{d}{dx}[x^4-3x^2-\pi] = 4x^3-6x \end{gather*}

So if we asked to find the antiderivative of \(4x^3-6x\text{,}\) we know at least two possible answers and quickly realize that there are infinitely many possible answers

\begin{equation*} \frac{d}{dx}^{-1}[4x^3-6x] = x^4-3x^2+C\text{,} \end{equation*}

where \(C\) could be any constant you might choose.

Maybe there are others as well that are not just adding a different constant. A natural question arises: Are there other functions that also have the same derivative?

The Mean Value Theorem comes to our rescue and tells us, no. The only antiderivatives that are possible are those that differ by a constant.

Theorem 5.3.1.

Suppose that \(f(x)\) and \(g(x)\) have the same derivative, \(f'(x)=g'(x)\) on an interval \(I\text{.}\) Then there must be a constant \(C\) so that \(g(x)-f(x)=C\text{,}\) or equivalently \(g(x)=f(x)+C\text{.}\)

Proof.

Define the function measuring the difference between \(f(x)\) and \(g(x)\text{,}\)

\begin{equation*} D(x) = g(x)-f(x)\text{.} \end{equation*}

On the interval \(I\text{,}\) \(f\) and \(g\) are differentiable and have the same derivative values, so that we have

\begin{equation*} D'(x) = \frac{d}{dx}[g(x)-f(x)] = g'(x)-f'(x) = 0\text{.} \end{equation*}

The first derivative test for monotonicity 5.2.1, which was an immediate consequence of the Mean Value Theorem, proved that when a function has a derivative equal to zero on an interval, that function must be a constant on that interval, \(D(x)=C\) for some constant \(C\text{.}\) That is \(g(x)-f(x)=C\) or \(g(x)=f(x)+C\text{.}\)

If we can find one antiderivative, then all other antiderivatives can be found by adding a constant. Graphically, this means that antiderivatives are vertical translations (shifts) of one another.

Later in calculus, we will learn that, at least for continuous functions, antiderivatives can always be computed through a process called integration. This fact is the central pillar of calculus called the Fundamental Theorem of Calculus. For this reason, rather than using a notation like \(\frac{d}{dx}^{-1}\) for finding antiderivatives, the notation instead adopts what is called an indefinite integral:

\begin{equation*} \int f(x) \, dx \equiv \frac{d}{dx}^{-1}[f(x)]\text{.} \end{equation*}

One way to think about the integral notation is to remember that a derivative is calculated as a limit of a difference quotient, meaning we find the difference \(f(x+h)-f(x)\) and calculate the quotient of this divided by \(\Delta x = h\text{.}\) In the limit, we think of \(\frac{\Delta y}{\Delta x} \to \frac{dy}{dx}\text{.}\) The inverse operation must involve something like a limit of a product sum. The integral sign \(\int\) is an elongated S representing a sum and the product of \(f(x)\) with \(dx\) is the product. When integration is formally introduced, you will indeed learn that it is defined by calculating a sum of terms calculated as a product of a function value times \(\Delta x\text{,}\) with a limit \(\Delta x \to dx = 0\text{.}\)

Subsection 5.3.2 Graphing an Antiderivative

The behavior of a function is characterized by its derivative. Consequently, if we know \(f'(x)\text{,}\) we can graph \(y=f(x)\) without knowing the actual function \(f(x)\) itself. \(f(x)\) will be an antiderivative of \(f'(x)\text{,}\) which means that we don't know an initial value of \(f(x)\) and that any graph we create could be shifted vertically up or down to get other antiderivatives. In addition, we currently only know where a function is increasing or decreasing, not how much it actual increases or decreases. When we learn about integration, we will learn that we need to calculate areas from the graph of \(f'(x)\) to get this information.

Here is the process that we will follow:

  • Using the formula or graph of \(f'(x)\text{,}\) complete sign analysis of \(f'(x)\) to determine intervals on which \(f(x)\) is increasing or decreasing.

  • (Future) Using integrals, compute the areas of regions of the graph \(f'(x)\) that are positive and negative to determine how much \(f(x)\) will increase or decrease on these intervals.

  • Using the graph of \(f'(x)\) or the formula of \(f''(x)\text{,}\) complete sign analysis of \(f''(x)\) to determine intervals on which \(f(x)\) is concave up or concave down.

  • Determine the shape behavior from the sign analysis.

  • Sketch a continuous graph of \(f(x)\text{.}\)

When we only see a graph of \(f'(x)\text{,}\) the sign analysis of \(f'(x)\) is based on where the graph is above or below the \(x\)-axis. Because \(f''(x)\) is the derivative of \(f'(x)\text{,}\) the sign analysis of \(f''(x)\) is based on the sign of the slope of the graph of \(f'(x)\text{.}\) Where \(f'(x)\) is increasing, we will find \(f''(x) \gt 0\) and where \(f'(x)\) is decreasing, we will find \(f''(x) \lt 0\text{.}\)

Example 5.3.2. Learning Outcome A-3.

Suppose that the graph of \(f'(x)\) is given below. Describe and then graph a function \(f(x)\) consistent with the graph of \(f'(x)\text{.}\)

Solution.

We start by using the information about \(f'(x)\) and \(f''(x)\) to understand where \(f(x)\) is increasing, decreasing, concave up, and concave down.

The signs of \(f'(x)\) determine the intervals of monotonicity. The signs of \(f'(x)\) come from intervals where the graph of \(f'(x)\) is above or below the \(x\)-axis. We see that \(f'(x)=0\) at \(x=-2\text{,}\) \(x=0\text{,}\) and \(x=2\text{.}\) Further, we see that \(f'(x) \gt 0\) on \((-2,0)\) and on \((2,\infty)\text{,}\) while \(f'(x) \lt 0\) on \((-\infty,-2)\) and \((0,2)\text{.}\) This means that \(f(x)\) will be decreasing on \((-\infty,-2]\text{,}\) increasing on \([-2,0]\text{,}\) decreasing on \([0,2]\text{,}\) and increasing on \([2,\infty)\text{.}\) Seeing that \(f'(x) \to 0\) as \(x \to \pm \infty\text{,}\) we realize that \(f(x)\) will become more and more horizontal on its ends.

The first derivative also allows us to identify where \(f(x)\) has its local minimum and maximum values. Because \(f(x)\) is decreasing on \((-\infty,-2]\) and then increasing on \([-2,0]\text{,}\) the graph falls and then rises, meaning that \(f(x)\) has a local minimum at \(x=-2\text{.}\) By similar reasoning, we conclue that \(f(x)\) has a local maximum at \(x=0\) and another local minimum at \(x=2\)

The signs of \(f''(x)\) determine the intervals of concavity. Because \(f''(x)\) measures the rate of change of \(f'(x)\text{,}\) we will determine the signs of \(f''(x)\) by looking at the slope of \(f'(x)\text{.}\) We start by finding \(f''(x)=0\text{,}\) where the tangent line of \(f'(x)\) is horizontal, which occurs at \(x \approx -2.9\text{,}\) \(x \approx -0.9\text{,}\) \(x \approx 0.9\text{,}\) and \(x \approx 2.9\text{.}\) Because \(f'(x)\) is decreasing on \((-\infty,-2.9)\text{,}\) we can see that \(f''(x) \lt 0\) and so \(f(x)\) is concave down on \((-\infty, -2.9]\text{.}\) In a similar way, because \(f'(x)\) has positive slope for \(-2.9 \lt x \lt -0.9\text{,}\) we know \(f(x)\) is concave up on \([-2.9,-0.9]\text{.}\) Using the same reasoning on the remaining intervals, we learn that \(f(x)\) is concave down on \([-0.9,0.9]\text{,}\) concave up on \([0.9,2.9]\text{,}\) and concave up on \([2.9,\infty)\text{.}\) The function \(f(x)\) has inflection points at \(x=-2.9\text{,}\) \(x=-0.9\text{,}\) \(x=0.9\text{,}\) and \(x=2.9\text{.}\)

To create the graph of \(y=f(x)\text{,}\) we use the same ideas as we used in Subsection 2.4.4. When we learn about definite integrals, we will learn how to compute how much a function increases or decreases based on the area between the graph \(y=f'(x)\) and \(y=0\text{,}\) so here we only focus on matching the monotonicity and concavity we have found. In this case, the areas above and below the axis exactly match, so our graph has the same horizontal asymptote on the left as on the right.

Subsection 5.3.3 Antiderivatives and the Power Rule

Given a formula for a function \(f(x)\text{,}\) can we find a formula for an antiderivative \(F(x)\text{,}\) a function so that \(F'(x)=f(x)\text{?}\) Unfortunately, the answer is only sometimes. The rules of differentiation mean that formulas of derivatives need to have patterns that can be created by following these rules. It is actually a challenging task, and often not even possible, to find such a formula.

Polynomials, however, have the nice property that the derivative of a polynomial is another polynomial. In this special case, it is quite easy to find a formula of an antiderivative.

First, recall the pattern of the power rule for a single power along with the constant multiple rule. We know

\begin{equation*} \frac{d}{dx}[c \cdot x^n] = (n \cdot c) \cdot x^{n-1}\text{.} \end{equation*}

That is, the derivative will have a coefficient where the original coefficient is multiplied by the original power and the power will have been reduced by one. If we want to find an antiderivative, we just need to invert this process: restore the power by adding one and then divide by that power.

\begin{equation*} \frac{d}{dx}^{-1}[a \cdot x^p] = (\frac{a}{p+1}) \cdot x^{p+1}\text{.} \end{equation*}

Example 5.3.5.

Find antiderivatives of the following simple power functions.

  • \(\displaystyle 2x^3\)

  • \(\displaystyle 5\sqrt{x}\)

  • \(\displaystyle \frac{3}{4x^4}\)

Solution.

We will consider each power function at a time.

The first function \(2x^3\) has a power \(p=3\text{.}\) For this to be the derivative of another function, the original power must have been \(p+1=4\text{:}\)

\begin{equation*} \frac{d}{dx}[x^4] = 4x^3\text{.} \end{equation*}

Dividing by that original power 4, we find

\begin{equation*} \frac{d}{dx}[{\textstyle \frac{1}{4}} x^4] = {\textstyle \frac{1}{4}} \cdot 4x^3 = x^3\text{.} \end{equation*}

To get the coefficient of 2, we multiply by 2.

\begin{equation*} \frac{d}{dx}[2 \cdot {\textstyle \frac{1}{4}} x^4] = 2 x^3\text{.} \end{equation*}

The second function \(5 \sqrt{x}\) does not initially look like a power until we rewrite the square root \(5 \sqrt{x} = 5 \cdot x^{1/2}\text{.}\) The power of the derivative is \(p = \textstyle \frac{1}{2}\text{,}\) so the power of the original function must have been \(p+1 = \frac{3}{2}\text{:}\)

\begin{equation*} \frac{d}{dx}[x^{3/2}] = {\textstyle \frac{3}{2}} \cdot x^{1/2}\text{.} \end{equation*}

Dividing by the original power is the same as multiplying by its reciprocal to find

\begin{equation*} \frac{d}{dx}[{\textstyle \frac{2}{3}} x^{3/2}] = {\textstyle \frac{2}{3}} \cdot {\textstyle \frac{3}{2}} \cdot x^{1/2} = x^{1/2}\text{.} \end{equation*}

We finish by multiplying by the coefficient:

\begin{equation*} \frac{d}{dx}[{\textstyle 5 \cdot \frac{2}{3}} x^{3/2}] = 5 \cdot x^{1/2} = 5 \sqrt{5}\text{.} \end{equation*}

That is, an antiderivative of \(5 \sqrt{x}\) will be \(\frac{10}{3} x^{3/2} = \frac{10}{3} x \sqrt x\text{.}\)

The third function \(\frac{3}{4x^4}\) is also not immediately in the form of a power function. We can not have division by a power, as we need to have it written as a constant multiplying by a power of \(x\text{.}\) So we rewrite using negative powers, \(\frac{3}{4x^4} = \frac{3}{4} \cdot x^{-4}\text{.}\) The power in the derivative is \(p=-4\text{,}\) so the power in the antiderivative must be \(p+1=-3\text{.}\) In addition, following the pattern of the previous problems, we will need to divide by this power:

\begin{equation*} \frac{d}{dx}[\frac{1}{-3} x^{-3}] = \frac{1}{-3} \cdot -3 x^{-4} = x^{-4}\text{.} \end{equation*}

To get the correct coefficient, we multiply by that constant:

\begin{equation*} \frac{d}{dx}[\frac{3}{4} \cdot \frac{1}{-3} x^{-3}] = \frac{3}{4} x^{-4}\text{.} \end{equation*}

The basic antiderivative of \(\frac{3}{4x^4}\) would be \(\frac{3}{4} \cdot \frac{1}{-3} x^{-3} = \frac{-1}{4} x^{-3}\text{.}\)

Because polynomials are just sums of simple power functions given by constants times powers of \(x\text{,}\) and derivatives have a sum rule, antiderivatives of polynomials and generalized polynomials can be found as a sum of the antiderivatives of each term.

In the introduction of this section, we pointed out that an antiderivative also has an unknown constant that can be added. This constant allows us to apply a vertical shift or translation to the function. This allows us to match any single given point for the antiderivative we are seeking.

Example 5.3.6. Learning Objective A-4.

For each of the following, find the function \(f(x)\) with the given derivative and passing through the given point.

  • \(f'(x) = x^3+2x-5\) with \(f(2)=5\)

  • \(f'(x) = x^2 - x \sqrt{x}\) with \(f(4)= 1\)

Solution.

For each problem, we will start by finding an antiderivative formula that is the sum of the basic antiderivatives of each simple power with an unknown added constant. Then we will evaluate that function at the given point, which will create an equation involving that unknown constant. Solving the equation, we will find the constant so that we can write down the function desired.

Recall that the notation for finding antiderivatives is with the integral operator \(\int \ldots \, dx\text{,}\) so our first function will be

\begin{equation*} f(x) = \int f'(x) \, \mathrm{d}x = \int [x^3+2x-5] \, \mathrm{d}x\text{.} \end{equation*}

To use the antiderivative power rule, we need to think of \(2x = 2x^1\) and \(-5 = -5 x^0\text{.}\) We find

\begin{equation*} f(x) = {\textstyle \frac{1}{4}} x^4 + 2 \cdot {\textstyle \frac{1}{2}} x^2 - 5 {\textstyle \frac{1}{1}} x^1 + C\text{,} \end{equation*}

where \(C\) represents the unknown constant. Simplifying, we get

\begin{equation*} f(x) = \textstyle \frac{1}{4}x^4 + x^2-5x+C\text{.} \end{equation*}

Note that it might be easier to recognize constants as having antiderivatives that are linear with the constant as its slope rather than using the power rule as illustratedabove.

For this first example, we were given that \(f(2)=5\text{.}\) Based on the function we just found, we can find a formula for that same value.

\begin{align*} f(2) &= \frac{1}{4}(2)^4 + (2)^2-5(2)+C\\ &= \frac{1}{4}(16) + 4 - 10 + C\\ &= -2 + C \end{align*}

Knowing that \(f(2)=5\) and \(f(2)=-2+C\text{,}\) we create and then solve the equation \(-2+C=5\) to discover \(C=7\text{.}\) The antiderivative in question was \(f(x)=\frac{1}{4}x^4+x^2-5x+7\text{.}\)

For the second example, we had \(f'(x) = x^2 - x \sqrt{x}\text{.}\) To find the general formula for antiderivatives, we need to rewrite this in the form of a generalized polynomial to get rid of the product and replace it by a single power,

\begin{equation*} f'(x) = x^2 - x^{3/2}\text{.} \end{equation*}

The function \(f(x)\) will be one of the possible antiderivatives,

\begin{align*} f(x) &= \int [x^2 - x^{3/2}] \, \mathrm{d}x \\ &= \frac{1}{3}x^3 - \frac{2}{5} x^{5/2} + C \end{align*}

We were also given \(f(4)=1\text{,}\) so we evaluate this general formula at \(x=4\) to create an equation so that we can find \(C\text{.}\)

\begin{align*} f(4) &= \frac{1}{3} (4)^3 - \frac{2}{5} (4)^{5/2} + C\\ &= \frac{1}{3}\cdot 64 - \frac{2}{5} \cdot 32 + C\\ &= \frac{64}{3} - \frac{64}{5} + C\\ &= \frac{128}{15} + C \end{align*}

This leads to an equation

\begin{equation*} \frac{128}{15} + C = 1 \end{equation*}

which we solve to find \(C = \frac{-113}{15}\text{.}\) The specific antiderivative in question will be

\begin{equation*} f(x) = \frac{1}{3}x^3 - \frac{2}{5} x^{5/2} - \frac{113}{15}\text{.} \end{equation*}

As we wrap up this work, we need to be aware of one power that does not have a simple power function antiderivative. If we think about domain issues and the integration rule

\begin{equation*} \int x^p \, \mathrm{d}x = \frac{1}{p+1} x^{p+1}\text{,} \end{equation*}

we should recognize that division by \(p+1\) will have a problem if \(p=-1\text{.}\) Indeed, the special case of a power function \(x^{p+1} = x^0\) is constant and results in a derivative of 0 rather than a separate power.

We will need an entirely new function to be the antiderivative of \(x^{-1}\text{.}\) We will later learn that this function is the natural logarithm, the transcendental function that is the inverse of the exponential function \(\exp(x) = e^x\text{.}\)
