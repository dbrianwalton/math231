Section 2.1 Function Notation

Function notation provides a means to write about the value of a dependent variable given the value of the independent variable. Suppose we have a function \(x \mapsto y\) with a name of \(f\text{,}\) meaning that knowing the value of \(x\text{,}\) \(f\) allows us to determine the specific value of \(y\text{.}\) We would write

\begin{equation*} y=f(x)\text{.} \end{equation*}

Function notation does not simply mean reading this as “\(y\) is a function of \(x\)”. Rather, this means that \(f\) provides the rule connecting the values of \(x\) and \(y\text{.}\)

Function notation consists of using the name of the function followed immediately by a value or expression inside of parentheses. That value or expression represents the value of the independent variable in the relation. In algebra, a variable name followed by something inside of parentheses would mean there is an implicit (unstated) multiplication. However, for functions, this is not multiplication but mapping. The expression \(f(2)\) refers to “the output value of \(f\) when the input value is 2”, or more succinctly “the value of \(f\) at 2”.

If \(x\) and \(y\) are related by an equation and we can solve that equation so that \(y\) is isolated, the formula on the other side is a formula representation of \(f(x)\text{.}\) So if we started with an equation \(x^2+y=3\text{,}\) this defines a relation. If we solve for \(y\) and get \(y=3-x^2\text{,}\) we realize that knowing \(x\) we can find \(y\) and so this equation defines a function. If we name the function \(f\text{,}\) then we have \(f(x)=3-x^2\text{.}\) We can then directly calculate the value of \(y\) by using the formula we find after substituting the variable \(x\) with its value. This means that \(f(4)\) represents “the value of \(y\) when \(x=4\)” and using substitution we find

\begin{equation*} f(4) = 3-(4)^2 = 3-16 = -13\text{.} \end{equation*}

This section focuses on three learning goals that build on this core understanding.

  • Use function notation on a formula to evaluate and simplify an expression involving substitution with formulas.

  • Use function notation to understand relations between variables represented by tables and graphs.

  • Use function notation to understand relations defined with piecewise formulas.

Subsection 2.1.1

Function notation is often used to describe a summary of mathematical operations applied to a value. A function like \(f(x)=x^2-3x\) could be read as saying, “Given the input value, calculate its square and then subtract from it three times the input value”. If we then had \(f(4)\text{,}\) we would look at this and think, “The input is 4. So our rule says \(f(4)\) will have a value that is the square of 4 minus three times 4.” This is what we mean by \(f(4) = 4^2 - 3(4)\) to get \(f(4)=16-12=4\text{.}\)

In applications, variables represent actual physical quantities. A function relating these variables will need to respect what type of value is used as the input and what type of value is created as an output. For example, if I were doing an electrophoresis gel observation with DNA, then there is a relation between the mass of a DNA segment and the distance it travels through the gel during the observation. If we let \(M\) be the mass of a segment and \(D\) be the distance traveled, then our observation would create a map \(M \mapsto D\text{.}\) If we said \(D = f(M)\text{,}\) then the function \(f\) is expecting mass of a DNA fragment as an input and gives a distance as output. It would not make any sense to talk about \(f(M^2)\) because \(M^2\) is no longer the mass of a fragment.

On the other hand, in mathematics variables represent values in an abstract sense without including the units being measured. If we wrote \(f(x^2)\) for the same function described previously, we would think of \(x\) as a value so that \(x^2\) gives the mass of a DNA fragment. So mathematically, \(f(x^2)\) can make sense.

This idea perhaps feels weirder when we have a function defined in abstract, such as \(f(x)=x^2-3x\) and then write something like \(f(2x+1)\text{.}\) In one sense, it might feel like we are saying \(x\) equals \(2x+1\text{.}\) This is where it is helpful to think of \(f\) as the rule of what we do to the input (and not what we do to \(x\)). We need to square the input \((2x+1)\) and then we need to subtract from that three times the input \((2x+1)\text{.}\)

\begin{equation*} f(2x+1) = (2x+1)^2 - 3(2x+1)\text{.} \end{equation*}

If we compare this to the original formula,

\begin{equation*} f(x) = x^2 - 3 x\text{,} \end{equation*}

we can see that we are substituting the input formula in place of the input variable. Because the input formula represents a single value, we wrap it in parentheses as a group.

Once we make the substitution, we are free to rewrite the formula in more convenient forms. For example, we might want to write the polynomial in standard form so that it is easier to combine with other polynomials later.

\begin{align*} f(2x+1) &= (2x+1)^2 - 3(2x+1)\\ &= (2x+1)(2x+1) - 3(2x+1)\\ &= 4x^2 + 2x + 2x + 1 - 6x - 3\\ &= 4x^2 - 2x - 2 \end{align*}

The representations of the function value are different but they always given the same value.

You are expected to be able to apply the principles of function evaluation using substitution in order to expand and then simplify combinations of functions evaluated with expressions as inputs.

Example 2.1.1. Learning Objective F-1.

Given a function \(f(x) = x^2 - 2x\text{,}\) Compute, expand, and simplify \(f(2x) - f(x-3)\text{.}\)

Solution.

When thinking about a problem like this, I like to imagine blocks representing components of the formula. I might write \(\boxed{f(2x)} - \boxed{f(x-3)}\text{.}\) That is, I should recognize that there are two formulas I need to find, \(f(2x)\) and \(f(x-3)\text{,}\) and I will later subtract them.

One method would be to find the individual formulas first using substitution.

\begin{align*} f(x) &= x^2 - 2x\\ f(2x) &= (2x)^2 - 2(2x)\\ &= (2x)(2x) - 2(2x)\\ &= 4x^2 - 4x\\ f(x-3) &= (x-3)^2 - 2(x-3)\\ &= (x-3)(x-3) - 2(x-3)\\ &= x^2-3x-3x+9-2x+6\\ &= x^2-8x+15 \end{align*}

Once we know the individual formulas, we combine them to create the value in question. We need to be careful to think of these formulas as giving a single value, so we place them inside parentheses to treat the formula as a grouped expression:

\begin{align*} f(2x) - f(x-3) &= (4x^2-4x) - (x^2-8x+15)\\ &= 4x^2-4x - x^2 + 8x - 15\\ &= 3x^2 + 4x - 15 \end{align*}

An alternative method would be to do all of the substitution at once. If this approach is used, extra caution must be taken to ensure that we treat all of the grouped expressions as single values. Use parentheses carefully:

\begin{align*} f(2x) - f(x-3) &= \Big((2x)^2-2(2x)\Big) - \Big((x-3)^2 - 2(x-3)\Big)\\ &= \Big(4x^2-4x\Big) - \Big(x^2-6x+9-2x+6\Big)\\ &= 4x^2-4x - x^2 +8x -15\\ &= 3x^2+4x -15. \end{align*}

Subsection 2.1.2

Understanding the relation between the input and output values of a function are critical in understanding function notation. If \(f\) is a specified function and we are told for some value \(x\) that \(f(x)=4\text{,}\) we should recognize that the symbol \(f(x)\) represents the output value of the function for an input value of \(x\text{.}\) Saying that \(f(x)=4\) means \(x\) is a value for the input such that the output is 4. To apply this understanding to problems involving tables and graphs, we need to know how to interpret input/output pairings for each tool.

A graph representing a function \(f\) includes points \((a,b)\) where \(f(a)=b\text{.}\) That is, the \(x\)-value of a point on the graph is an input value. The \(y\)-value will be the corresponding output value. You should have previously learned that a function must satisfy the vertical line test, which means that for every specific value of \(x\) (a vertical line), there can be at most one point on the graph. Otherwise, knowing the value of the input would not allow us to know the value of the output. If there is no point on the graph for the given value of \(x\text{,}\) then \(f(x)\) will be undefined at that input.

A table representing a function will typically give a column (or row) corresponding to the values of inputs and their corresponding outputs. If the heading to the column of inputs were \(x\text{,}\) then the column of outputs would be \(f(x)\text{.}\)

Example 2.1.2. Learning Objective F-2.

The function \(f\) is represented by a graph and the function \(g\) is represented by a table, shown below. Demonstrate how function notatation allows you to find the requested values.

Find the following values.

  • \(\displaystyle f(-2)\)

  • \(\displaystyle g(2)\)

  • \(x\) so that \(f(x)=g(0)\)

  • \(\displaystyle g(f(-1))\)

Solution.

To find \(f(-2)\text{,}\) we first interpret notation. The symbol \(f(-2)\) stands for the output value of a function \(f\) with an input of \(-2\text{.}\) That is, we want the \(y\)-value of the graph corresponding to the value \(x=-2\text{.}\) The vertical line at \(x=-2\) passes through an open point at \((-2,1)\) and a closed point at \((-2,-1)\text{.}\) The solid point is the point in the relation, so \(f(-2) = -1\text{.}\) The open point represents a limit of the segment of the graph and not an actual point in the graph.

To find \(g(2)\text{,}\) we again interpret notation. The symbol \(g(2)\) is the output value of the map with input of \(2\text{.}\) We look in the table of \(g\text{.}\) The column headed by \(x\) contains input values while the column headed by \(g(x)\) contains corresponding output values. Finding \(x=2\) in the first column is paired with \(g(x)=-4\) in the second column. So \(g(2)=-4\text{.}\)

To find \(x\) so that \(f(x)=g(0)\text{,}\) we need to think through a few steps. The symbol \(g(0)\) represents an output from \(g\) corresponding to an input of \(0\text{.}\) Looking in the table with \(x=0\) we find \(g(0)=1\text{.}\) We replace the symbol \(g(0)\) in the equation with this value to restate our question: find \(x\) so that \(f(x)=1\text{.}\)

Notice that the value 1 and the symbol \(f(x)\) are equal, meaning that we are looking for value(s) \(x\) so that the output value \(f(x)=1\text{.}\) We now look at the graph at a \(y\)-value (horizontal line) \(y=1\) and see what point(s) on the graph match. There are three points where the graph's lines cross \(y=1\text{,}\) at \(x=-5\text{,}\) \(x=2\text{,}\) and \(x=4\text{.}\) The open point at \((-2,1)\) is not an included point of the graph.

To find \(g(f(1))\text{,}\) we need to think carefully about what this means. A function name with parentheses means that whatever is inside the parentheses represents the input value. Both \(f\) and \(g\) are functions here. So what is the input? The function \(g\) (outer function) has parentheses surrounding \(f(1)\text{,}\) so the value of \(f(1)\) is the input. Because \(f\) is also a function, its parentheses indicate that its input is 1.

We start with the innermost piece to find \(f(1)\text{.}\) An input \(x=1\) on the graph is matched with \(y=3\) (the closed point), so \(f(1)=3\text{.}\) Once we know that value, we substitute to discover

\begin{equation*} g(f(1)) = g(3)\text{.} \end{equation*}

The input \(x=3\) for the function \(g\) is paired with an output \(g(x)=-3\text{,}\) so \(g(f(1))=-3\text{.}\)

Subsection 2.1.3

The domain of a function is the set of all valid input values. When we have a formula for a function, the domain is normally understood to be the set of all values for which the formula makes sense (output value is defined). We can restrict the domain of a function by specifying a specific set, usually by giving an interval for the valid input values. In this case, a function requires that we indicate both the formula and the interval.

The notation for specifying a formula on a particular interval is to state the equation for the function's formula along with an inequality for the domain variable. If we want to use \(f(x)=x^2\) but use a domain \((0,3]\text{,}\) we would write

\begin{equation*} f(x) = x^2, \quad 0 \lt x \le 3\text{.} \end{equation*}

For this function, \(f(0)\) would not be defined because \(x=3\) is outside of the domain.

Once we are able to specify a specific formula on an interval in the domain, we can expand this idea to allow different formulas on different intervals. We call such functions piecewise-defined. The notation is to group the collection of rules on individual subsets of the domain with a curly brace on the left.

Example 2.1.5.

Define the function \(f(x)\) piecewise by

\begin{equation*} f(x) = \begin{cases} 3-x, & -2 \le x \lt 0, \\ x^2, & 0 \lt x \le 3. \end{cases} \end{equation*}

The domain has two subsets of interest. On \([-2,0)\text{,}\) the function uses \(f(x)=3-x\text{;}\) on \((0,3]\text{,}\) the function uses \(f(x)=x^2\text{.}\) The overall domain is \([-2,0)\cup(0,3]\text{.}\) The value of \(f(0)\) is undefined because \(x=0\) is not in either domain restriction.

To evaluate a piecewise function at a point, the input value needs to be compared to the restricted domains. Whichever domain restriction the input value belongs determines which formula rule will be used. In the preceding example, to find \(f(2)\text{,}\) we look at the input \(x=2\) and recognize it belongs to the restriction \((0,3]\text{.}\) So the formula will be \(f(x)=x^2\) when \(x=2\) to give \(f(2)=(2)^2 = 4\text{.}\) On the other hand, \(f(-2)\) will use \(f(x)=3-x\) since \(x=-2\) is in the restriction \([-2,0)\text{,}\) giving \(f(-2) = 3-(-2)=5\text{.}\)

An important piecewise function in mathematics is the absolute value function. It is named \(\mathrm{abs}\) but can also be represented by vertical bars around the input, \(\abs(x) = |x|\text{.}\) This function creates an output that matches the input (we call this the identity function) when the input is non-negative. When the input is negative, the output changes sign by multiplying the input by \(-1\text{.}\)

\begin{equation*} \abs(x) = |x| = \begin{cases} x, & x \ge 0, \\ -x, & x \lt 0. \end{cases} \end{equation*}

One strategy for thinking about piecewise functions is to think of the domain on a number line. For each restriction, identify that subset on the number line. Then label the interval with the formula that gets used on that interval.

A graph of a piecewise function follows this same pattern. The graph of the individual parts are restricted based on the domain and combined to a single graph.

You should be comfortable interpreting the restricted domains that make up a piecewise function. In particular, you should be able to recognize where a function is or isn't defined. Pay close attention to whether end-points of intervals are included or excluded in a domain restriction.

Example 2.1.8. Learning Objective F-3.

For the function \(g(x)\) defined piecewise below, evaluate each of the following points, showing clearly how you find the value from the formulas. If the value does not exist, indicate so.

\begin{equation*} g(x) = \begin{cases} x^2-2, & x \le -1, \\ 2+3x, & -1 \lt x \lt 1, \\ 5x-1, & 1 \lt x \lt 3, \\ x^2+2, & x \ge 3 \end{cases} \end{equation*}

  1. \(\displaystyle g(-2)\)

  2. \(\displaystyle g(-1)\)

  3. \(\displaystyle g(0)\)

  4. \(\displaystyle g(1)\)

  5. \(\displaystyle g(3)\)

Solution.

If the idea of visualizing the restricted domain helps you, we might start by creating a number line showing where the different rules are used. This doesn't really add any new information, so this is strictly a personal preference.

We work through each value in turn. First, we look at the value of the input. Then we decide which restricted domain it belongs to. Finally, we use the formula for the restricted domain to find the value.

  1. To find \(g(-2)\text{,}\) we recognize \(x=-2\) is in \((-\infty,-1]\) and so use \(g(x)=x^2-2\text{:}\)

    \begin{equation*} g(-2) = (-2)^2-2 = 4-2 = 2\text{.} \end{equation*}

  2. To find \(g(-1)\text{,}\) we recognize \(x=-1\) is the end-point included in \((-\infty,-1]\) and so again use \(g(x)=x^2-2\text{:}\)

    \begin{equation*} g(-1) = (-1)^2-2 = 1-2 = -1\text{.} \end{equation*}

  3. To find \(g(0)\text{,}\) we recognize \(x=0\) is in \((-1,1)\) and so use \(g(x)=2+3x\text{:}\)

    \begin{equation*} g(0) = 2+3(0) = 2\text{.} \end{equation*}

    \(g(0)\)

  4. To find \(g(1)\text{,}\) we realize \(x=1\) is not in any of the restricted domains. \(g(1)\) does not exist.

  5. To find \(g(3)\text{,}\) we see \(x=3\) is the end-point included in \([3,\infty)\) and uses \(g(x)=x^2+2\text{:}\)

    \begin{equation*} g(3) = (3)^2+2 = 9+2 = 11\text{.} \end{equation*}
