A set can be assigned a name, such as \(P = \{2,3,5,7\}\text{.}\) The number 2 is in \(P\text{,}\) which we write \(2 \in P\text{.}\) The number 4 is not in \(P\text{,}\) which we write \(4 \not \in P\text{.}\) A set that is empty would have no values, \(\{\}\text{,}\) and sometimes is written with a special symbol \(\emptyset\text{.}\)
