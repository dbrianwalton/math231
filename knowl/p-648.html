When using the definition of the derivative \(g'(a)\) on a piecewise function, you need to be careful that the value \(g(a)\) uses the function value at the point of interest and not the formula from the sides. But when calculating \(g(a+h)\text{,}\) we need to consider the different cases for \(h \gt 0\) (used for the right-derivative) and for \(h \lt 0\) (used for the left-derivative).
