Item.

Solve the constraint equation.

We want to solve the constraint equation so that one control variable is a function of the other. In this problem, it will be easier to solve for \(y\) as a function of \(x\text{,}\)

\begin{equation*} y = \frac{2}{x^2} = 2x^{-2}\text{.} \end{equation*}
