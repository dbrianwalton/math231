The behavior of a function is described using the language of monotonicity and concavity. Recall that a function \(f(x)\) is increasing on a set \(S\) if for any \(a,b \in S\text{,}\) if \(a \lt b\) then \(f(b) \gt f(a)\text{.}\) Similarly, \(f(x)\) is decreasing on a set \(S\) if for any \(a,b \in S\text{,}\) if \(a \lt b\) then \(f(b) \lt f(a)\text{.}\)
