An operator is like a function in that it has an input and an output. Instead of taking values as the input, an operator takes an expression or function. The derivative operator takes a function as its input and gives the derivative function as the output. Our earlier work showed that for \(f(x)=x^2\text{,}\) we have \(f'(x)=2x\text{.}\) That is, we think of \(x^2\) as the input of the step and \(2x\) as the output, and write

\begin{equation*} \frac{d}{dx}[x^2] = 2x\text{.} \end{equation*}
