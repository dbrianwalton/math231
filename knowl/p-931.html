A second challenge of application problems is interpreting what information we are given. We are told a value of the radius (in millimeters) and how fast it is changing (in millimeters per second). We need a variable for the radius, say \(r\text{.}\) The given information is the value \(r=0.8\) mm. When we are given a rate of change, that is a derivative, in this case \(\frac{dr}{dt} = -0.05\) mm/s. The value is negative because we know the radius is decreasing.
