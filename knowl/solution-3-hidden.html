The point–slope form makes this very simple. When we recall that \(\Delta y = m \cdot \Delta x\) and then think of \((5,12)\) as our starting point and \((x,y)\) as a second point, we get increments \(\Delta y = y-12\) and \(\Delta x = x-5\text{.}\) The equation can then be immediately written

\begin{equation*} y - 12 = \frac{3}{2} \cdot(x-5)\text{.} \end{equation*}

If we want to write \(y\) as a function of \(x\text{,}\) we isolate the \(y\) to get

\begin{equation*} y = \frac{3}{2} \cdot(x-5) + 12\text{.} \end{equation*}

Compare what would happen if we tried to use the slope–intercept form \(y=mx+b\text{.}\) We know from the problem \(m=\frac{3}{2}\) but we do not yet know the intercept value \(b\text{.}\) We might make use of the information that we do know to get

\begin{equation*} y=\frac{3}{2} x + b\text{.} \end{equation*}

To find \(b\text{,}\) we could use the information that we know \((x,y)=(5,12)\) is a point on the line. That is, substituting values \(x=5\) and \(y=12\) must create a true equation:

\begin{equation*} 12 = \frac{3}{2} \cdot 5 + b\text{.} \end{equation*}

We can now solve for \(b\text{:}\)

\begin{equation*} b = 12 - \frac{15}{2} = \frac{24-15}{2} = \frac{9}{2}\text{.} \end{equation*}

Now that we know the value of the intercept, we can finish the equation,

\begin{equation*} y = \frac{3}{2} x + \frac{9}{2}\text{.} \end{equation*}

This second method is completely valid but requires extra work. Also, students often find the value of \(b\) but then forget to use it to finish stating the equation that was requested.
