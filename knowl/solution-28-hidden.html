Problems like this may feel unnatural because we don't even know what these functions \(f\) and \(g\) might be. Suppose it were a familiar function, say \(\sqrt{2x} = 3\text{.}\) We would want to get rid of the square root and would do this by squaring both sides. That is, we would use an inverse operation. We will do the same thing here, except the inverse operation to \(f\) as a function is the function \(g\text{.}\)

Starting with the original equation, we use composition with the inverse \(g\) on both sides of the equation:

\begin{align*} f(2x) &= 3\\ g(f(2x)) &= g(3). \end{align*}

The composition \(g \circ f\) reduces to the identity function, which just leaves the input expression. The equation therefore becomes:

\begin{align*} g(f(2x)) &= g(3)\\ 2x &= g(3)\\ x &= \frac{1}{2} g(3). \end{align*}

Because we have isolated \(x\) and the value is a formula involving \(g(3)\text{,}\) we are done.
