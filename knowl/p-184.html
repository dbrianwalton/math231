Understanding the relation between the input and output values of a function are critical in understanding function notation. If \(f\) is a specified function and we are told for some value \(x\) that \(f(x)=4\text{,}\) we should recognize that the symbol \(f(x)\) represents the output value of the function for an input value of \(x\text{.}\) Saying that \(f(x)=4\) means \(x\) is a value for the input such that the output is 4. To apply this understanding to problems involving tables and graphs, we need to know how to interpret input/output pairings for each tool.
