Solution 1.4.4.1.

A graph will give us intuition. If we graph \(y=x^4-x^3-8x^2+2\) and \(y=5x^2-x-10\text{,}\) the solution we are looking for is the set of \(x\)-values where the first graph is below (less than) the second graph.

To solve this question directly, we rewrite the inequality as a relation with zero:

\begin{gather*} x^4-x^3-8x^2+2 \lt 5x^2-x-10\\ x^4-x^3-8x^2+2 - 5x^2 + x + 10 \lt 0\\ x^4-x^3-13x^2+ x + 12 \lt 0 \end{gather*}

At this point we need to factor the polynomial. Because this is degree 4, we will use a computer to solve. One approach would be to go to WolframAlpha 1  and type factor x^4-x^3-13x^2+x+12. Another would be to use the free tool SageMath 2 , as illustrated below.

On evaluation of the factoring, we can rewrite our question

\begin{equation*} (x+3)(x+1)(x-1)(x-4) \lt 0\text{.} \end{equation*}

Because this is already factored, we can quickly note that the only roots (zeros) occur at four locations: \(x=-3\text{,}\) \(x=-1\text{,}\) \(x=1\text{,}\) and \(x=4\text{.}\) Polynomials have no discontinuities, so these are the only possible end-points. Using a number line broken at these points, we find five test intervals: \((-\infty,-3)\text{,}\) \((-3,-1)\text{,}\) \((-1,1)\text{,}\) \((1,4)\text{,}\) \((4,\infty)\text{.}\)

We will test each interval by choosing a single value and calculating the sign of the formula. Because the formula is factored, this can be quickly determined by keeping track of the signs of each factor and then counting the number of negative factors. An odd number of negative factors will result in a negative product. An even number of negative factors will result in a positive product.

The table shows our reasoning. It is usually adequate to summarize the results directly on a number line, creating a sign analysis summary. We create a number line and mark each of the possible end-points by solving the equation. Label the value of the variable below the line and why it was included above the line (either a root or discontinuity). Then for each resulting interval choose a test point (label below the interval) and write the sign of the formula above the line.

After we complete the sign analysis, we need to interpret our results. In this question, we were solving for where \(x^4-x^3-13x^2+x+12\) was less than zero, meaning it was negative. We select the set of all values on our number linear that created negative values for the formula. Our solution set is \((-3,-1) \cup (1,4)\text{.}\)
