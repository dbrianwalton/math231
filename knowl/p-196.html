The notation for specifying a formula on a particular interval is to state the equation for the function's formula along with an inequality for the domain variable. If we want to use \(f(x)=x^2\) but use a domain \((0,3]\text{,}\) we would write

\begin{equation*} f(x) = x^2, \quad 0 \lt x \le 3\text{.} \end{equation*}

For this function, \(f(0)\) would not be defined because \(x=3\) is outside of the domain.
