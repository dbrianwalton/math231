To visualize this, let us begin by creating a number line and showing each of the three sets \(A,B,C\text{.}\) To emphasize that these are being considered as different sets, they are colored differently and placed on different layers.

To find \(A \cup B\text{,}\) we need to describe the set of all values that are in \(A\) or \(B\text{.}\) The two sets do not overlap at all, so the union will consist of the same disjoint intervals, but now combined as a single set,

\begin{equation*} A \cup B = (-2,1] \cup [3,5)\text{.} \end{equation*}

To find \(A \cup C\text{,}\) we are interested in values that are in at least one of the sets. Notice that these sets overlap, meaning there are some values that are in both intervals (in their intersection). Consider the boundaries of this overlap. We have \(0 \in A\) and \(0 \not in C\text{,}\) but it is in at least one set and so \(0 \in A \cup C\text{.}\) Similarly, we have \(1 \in A\) and \(1 \in C\text{.}\) By being in both sets, it is necessarily in at least one and so \(1 \in A \cup C\text{.}\) All values between them are in both sets and so are in the union. The union of these overlapping intervals will be a new interval that contains both,

\begin{equation*} A \cup C = (-2,2]\text{.} \end{equation*}

The sets \(B\) and \(C\) do not overlap and so are disjoint. The union is simply written

\begin{equation*} B \cup C = (0,2] \cup [3,5)\text{.} \end{equation*}

Notice that the order of intervals in the union is reading the number line left-to-right, and does not match the order that \(B\) and \(C\) were listed in the union. We could have written

\begin{equation*} B \cup C = [3,5) \cup (0,2]\text{,} \end{equation*}

but this ordering is harder to transfer to a number line and so is not preferred.

The intersections are calculated by finding where the sets overlap (values in both intervals). The only intervals here that overlapped were \(A\) and \(C\text{.}\) As noted earlier \(0 \not in C\text{,}\) so we have

\begin{equation*} A \cap C = (0,1]\text{.} \end{equation*}

The other two intersections are empty:

\begin{gather*} A \cap B = \{\}\\ B \cap C = \{\} \end{gather*}
