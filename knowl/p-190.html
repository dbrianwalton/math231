To find \(g(2)\text{,}\) we again interpret notation. The symbol \(g(2)\) is the output value of the map with input of \(2\text{.}\) We look in the table of \(g\text{.}\) The column headed by \(x\) contains input values while the column headed by \(g(x)\) contains corresponding output values. Finding \(x=2\) in the first column is paired with \(g(x)=-4\) in the second column. So \(g(2)=-4\text{.}\)
