The idea of undoing the function is demonstrated through composition. If we start with an equation

\begin{equation*} y=f(x)\text{,} \end{equation*}

look at what happens if we use composition by \(f^{-1}\) to both sides of the equation:

\begin{equation*} f^{-1}(y)=f^{-1}\big(f(x)\big)\text{.} \end{equation*}

We know that \(f^{-1}(y) = x\) is the inverse map, so the equation becomes

\begin{equation*} x=f^{-1}\big(f(x)\big)\text{.} \end{equation*}

That is, a function composed with its inverse function is the same as doing nothing to the input, an operation called the identity function.
