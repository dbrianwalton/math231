Rather than jump right into the calculation, it is better to think about what we expect so that we can see if our calculated answer makes sense. In this case, a visualization of what we are calculating is helpful. We can do this by graphing the two points (or at least thinking about the graph).

Notice how the point \((3.2,-2)\) is to the right of and below the first point \((1.8,5)\text{.}\) We say that the graph of the line is decreasing. This corresponds to a slope that is negative. When we complete our calculation, we want to be sure that we get a negative slope.

As we calculate the slope, it is possible to do it in a single step. Again, it can be helpful to think about it more deliberately by calculating the increments \(\Delta x\) and \(\Delta y\) separately.

\begin{align*} \Delta x &= x_2 - x_1 = 3.2 - 1.8 = 1.4\\ \Delta y &= y_2 - y_1 = -2 - 5 = -7 \end{align*}

Notice how the definition we are following expects that we have identified what we mean by first and second point. We need to be careful that we calculate both increments as the change going from the first to the second point.

Do the values make sense? The increment \(\Delta x\) is positive because the second point is to the right of the first point. The increment \(\Delta y\) is negative because the second point is below the first point. So we have verified that the signs of the increments are consistent with what we observed in the graph.

The value of the slope (rate of change) can now be directly calculated as the ratio of the increments of change,

\begin{equation*} m = \frac{\Delta y}{\Delta x} = \frac{-7}{1.4} = -5\text{.} \end{equation*}

The slope is negative as we expected.
