However, \(h\) is a common factor in the numerator so that we can rewrite

\begin{align*} \frac{\Delta f}{\Delta x}\Big|_{3,3+h} &= \frac{7h+h^2}{h}\\ &= \frac{h(7+h)}{h}\\ &= 7+h \end{align*}

for \(h \ne 0\text{.}\) Where the original quotient had a discontinuity, the new formula is continuous and we can find the limit using value substitution.

\begin{align*} f'(3) &= \lim_{h \to 0} \frac{f(3+h)-f(3)}{h}\\ &= \lim_{h \to 0} \frac{7h+h^2}{h}\\ &= \lim_{h \to 0} [7+h]\\ &= 7 + 0 = 7 \end{align*}

We replaced the limit in the second to last row by the value of the formula using continuity. So \(f'(3) = 7\text{,}\) and this is not an approximation but an exact value as calculated using the definition.
