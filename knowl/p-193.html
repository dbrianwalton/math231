To find \(g(f(1))\text{,}\) we need to think carefully about what this means. A function name with parentheses means that whatever is inside the parentheses represents the input value. Both \(f\) and \(g\) are functions here. So what is the input? The function \(g\) (outer function) has parentheses surrounding \(f(1)\text{,}\) so the value of \(f(1)\) is the input. Because \(f\) is also a function, its parentheses indicate that its input is 1.
