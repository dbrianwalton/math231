If \(x\) and \(y\) are related by an equation and we can solve that equation so that \(y\) is isolated, the formula on the other side is a formula representation of \(f(x)\text{.}\) So if we started with an equation \(x^2+y=3\text{,}\) this defines a relation. If we solve for \(y\) and get \(y=3-x^2\text{,}\) we realize that knowing \(x\) we can find \(y\) and so this equation defines a function. If we name the function \(f\text{,}\) then we have \(f(x)=3-x^2\text{.}\) We can then directly calculate the value of \(y\) by using the formula we find after substituting the variable \(x\) with its value. This means that \(f(4)\) represents “the value of \(y\) when \(x=4\)” and using substitution we find

\begin{equation*} f(4) = 3-(4)^2 = 3-16 = -13\text{.} \end{equation*}
