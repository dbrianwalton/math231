Definition 5.1.1. Continuity on Intervals.

A function \(f\) is continuous on an open interval \((a,b)\) if \(f\) is continuous at every point \(c \in (a,b)\text{,}\)

\begin{equation*} \lim_{x \to c} f(x) = f(c)\text{.} \end{equation*}

We can include the left end point \(a\) if \(f\) is right-continuous at \(a\text{,}\)

\begin{equation*} \lim_{x \to a^+} f(x) = f(a)\text{.} \end{equation*}

We can include the right end point \(b\) if \(f\) is left-continuous at \(b\text{,}\)

\begin{equation*} \lim_{x \to b^-} f(x) = f(b)\text{.} \end{equation*}
