Solution 4.4.5.1.

A horizontal tangent corresponds to a slope that is zero. We need to find the derivative function \(R'(x)\) and determine values so that \(R'(x)=0\text{.}\) The function \(R(x)\) is a rational function because it is a quotient of two polynomials.

To find \(R'(x)\text{,}\) we need to use the quotient rule because \(R(x)\) itself is a quotient.

\begin{align*} R'(x) &= \frac{d}{dx}\left[ \frac{x^2+2x}{2x^2+1} \right] \\ &= \frac{\frac{d}{dx}[x^2+2x] \cdot (2x^2+1) - (x^2+2x) \cdot \frac{d}{dx}[2x^2+1]}{(2x^2+1)^2} \\ &= \frac{(2x+2)(2x^2+1) - (x^2+2x)(4x)}{(2x^2+1)^2} \\ &= \frac{(4x^3+4x^2+2x+2) - (4x^3+8x^2)}{(2x^2+1)^2} \\ &= \frac{4x^3+4x^2+2x+2 - 4x^3 - 8x^2}{(2x^2+1)^2} \\ &= \frac{-4x^2+2x+2}{(2x^2+1)^2} \end{align*}

Experience has shown me that we always want to rewrite the numerator in hopes of factoring in the future, but almost never want to rewrite the denominator since it is already factored.

Once we know \(R'(x)\text{,}\) we create and solve the equation \(R'(x)=0\text{.}\) Because \(R'(x) = \frac{-4x^2+2x+2}{(2x^2+1)^2}\) is itself a quotient, \(R'(x)= 0\) is solved by determining where the numerator equals zero. In this case, we can factor that resulting polynomial.

\begin{gather*} R'(x) = 0\\ \frac{-4x^2+2x+2}{(2x^2+1)^2} = 0\\ -4x^2+2x+2 = 0\\ -2(2x^2-x-1) = 0\\ -2(2x+1)(x-1) = 0 \end{gather*}

So \(R'(x)=0\) at \(x=-\frac{1}{2}\) and at \(x=1\text{.}\) We can verify whether these values make sense by looking at a graph of \(R(x)\text{.}\)
