If we wanted to be careful, we would also need to learn that there is an order of set operations. Intersection is like multiplication and union is like addition. So if we had three sets \(A\text{,}\) \(B\text{,}\) and \(C\text{,}\) the notation \(A \cup B \cap C\) would mean the union of \(A\) with \(B \cap C\text{.}\) To avoid worrying about this, we will use parentheses to make things clear. For that example, we would have an unambiguous notation by writing \(A \cup (B \cap C)\text{,}\) which is very different from \((A \cup B) \cap C\text{,}\) the intersection of \(A \cup B\) with \(C\text{.}\)
