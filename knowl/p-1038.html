In principle, there is no real domain constraint on \(a\text{.}\) However, it is clear that \(a \lt 0\) or \(a \gt 2\) will not lead to minimum time. So we will use a constrained domain that \(a\) is in \([0,2]\text{.}\) Because \(T(a)\) is continuous on the closed interval \([0,2]\text{,}\) there is guaranteed to be a minimum time path and maximum time path within this domain. We seek the minimum time path.
