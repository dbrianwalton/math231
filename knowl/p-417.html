Suppose we want to know the value of a function \(M\) (for mystery) at a point \(x=0\text{,}\) but the rule we have for evaluating the function has not yet been worked out for that particular value. What might we try to do? If the function is nice, we might try evaluating \(M(0.1)\) thinking that because \(x=0.1\) is close to \(x=0\text{,}\) then \(M(0.1)\) should be close to \(M(0)\text{.}\) That is what continuity is about.
