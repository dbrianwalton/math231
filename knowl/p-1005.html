We found the value of \(x=\sqrt[3]{2}\text{.}\) We also need the value of \(y\text{.}\) From the constraint, we found \(y=2x^{-2}\) so that we can find

\begin{equation*} y = \frac{2}{(\sqrt[3]{2})^2} = \frac{2}{2^{2/3}} = 2^{1/3}\text{.} \end{equation*}

That is, we find \(y=x=\sqrt[3]{2}\text{,}\) and our rectangular box is actualy a cube.
