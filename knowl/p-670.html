The example above showed that for \(g(x)=x^2-2x\text{,}\) we have \(g'(x)=2x-2\text{.}\) This formula can be used now to find the slope of the tangent line at any point. For example, we have \(g'(2) = 2(2)-2 = 2\text{.}\) The tangent line at \(x=2\) has slope 2. Similarly, we have \(g'(3)=2(3)-2 = 4\text{,}\) so that the tangent line at \(x=3\) has slope 4. This one formula allows us to find the slope of any tangent line for \(g(x)\text{.}\)
