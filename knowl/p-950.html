We need to create an equation that relates the height and volume of water in the cup. Using the geometric formula for volume of a circular cone, we start with

\begin{equation*} V = \frac{1}{3} \pi r^2 h\text{.} \end{equation*}

One approach is to use the similarity equation to solve for \(r\) as a function of \(h\text{,}\)

\begin{equation*} r = \frac{4h}{10} = \frac{2}{5} h\text{,} \end{equation*}

so that we can write

\begin{equation*} V = \frac{1}{3} \pi \left(\frac{2}{5} h\right)^2 \cdot h = \frac{4}{75} \pi h^3\text{.} \end{equation*}
