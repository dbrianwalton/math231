What could go wrong? The first thing to watch out for is that some operations change the ordering. For example, you should have learned that multiplying by a negative number reverses order. That is, if we start with \(u \lt v\) and multiply by -2, we have \(-2u \gt -2v\text{.}\) Another example is a reciprocal of two numbers of the same sign. If we start with \(0 \lt u \lt v\) (two positive numbers), then since \(v\) would be a larger denominator, \(0 \lt \frac{1}{v} \lt \frac{1}{u}\text{.}\) Being the same sign is important. If \(u \lt 0 \lt v\text{,}\) then \(\frac{1}{u} \lt 0 \lt \frac{1}{v}\) because the reciprocal does not change the sign, only the magnitude, of the number.
