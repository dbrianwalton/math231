Move the point \(A\) to \(x=-2\text{.}\) You should be sure to select the filled in point as the hollow point is just a limit point of the graph. You can now adjust the slider for \(h\text{.}\) If you approach \(h \to 0^-\) by coming from the left (negative \(h\)), you should see a left-tangent line has a slope that is approximately 2. This allows us to state:

\begin{equation*} \frac{df}{dx}\Big|_{x=-2^-} \approx 2\text{.} \end{equation*}

(Do you see how the tangent line appears like it would go through \((-3,0)\text{?}\) If doing this by hand, you would use that point to calculate a slope.)
