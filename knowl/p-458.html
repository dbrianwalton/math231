Consider the formula

\begin{equation*} f(x) = \frac{(x+3)(x-1)}{(x+2)(x-1)}\text{.} \end{equation*}

Because of how this function is defined, \(x=-2\) and \(x=1\) are both not in the domain. The simplified function

\begin{equation*} g(x) = \frac{x+3}{x+2} \end{equation*}

will have a slightly larger domain, excluding only \(x=-2\text{.}\) For every value \(x \ne 1\text{,}\) the fraction \(\displaystyle \frac{x-1}{x-1}\) will simplify to 1, so that \(g(x) = f(x)\) for all \(x \ne 1\text{.}\) The graphs are the same, except that \(f(x)\) has a hole at \(x=1\text{.}\)
