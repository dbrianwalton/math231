Function notation is often used to describe a summary of mathematical operations applied to a value. A function like \(f(x)=x^2-3x\) could be read as saying, “Given the input value, calculate its square and then subtract from it three times the input value”. If we then had \(f(4)\text{,}\) we would look at this and think, “The input is 4. So our rule says \(f(4)\) will have a value that is the square of 4 minus three times 4.” This is what we mean by \(f(4) = 4^2 - 3(4)\) to get \(f(4)=16-12=4\text{.}\)
