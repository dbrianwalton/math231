The Extreme Value Theorem (see TheoremÂ 5.1.4) provides a guarantee that a function \(f(x)\) will have both a global maximum and a global minimum when \(f(x)\) is continuous and the set in question is a closed interval. If the function has a discontinuity or if the end points of an interval are not included, then we must check limits at the discontinuities and end points. If a limit exceeds that of all the attained values, then the function will not have the corresponding local extreme.
