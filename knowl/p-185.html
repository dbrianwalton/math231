A graph representing a function \(f\) includes points \((a,b)\) where \(f(a)=b\text{.}\) That is, the \(x\)-value of a point on the graph is an input value. The \(y\)-value will be the corresponding output value. You should have previously learned that a function must satisfy the vertical line test, which means that for every specific value of \(x\) (a vertical line), there can be at most one point on the graph. Otherwise, knowing the value of the input would not allow us to know the value of the output. If there is no point on the graph for the given value of \(x\text{,}\) then \(f(x)\) will be undefined at that input.
