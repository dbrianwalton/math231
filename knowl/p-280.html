This inverse function undoes the calculation of \(f\text{.}\) Suppose \(f : x \mapsto y\) has an inverse function \(f^{-1}: y \mapsto x\text{.}\) If we know that \(f(3) = 5\text{,}\) this means \(x=3 \mapsto y=5\text{.}\) The inverse function would map \(y=5 \mapsto x=3\) or \(f^{-1}(5)=3\text{.}\) This can be true when variables are involved as well: \(f(a) = 4\) means \(f^{-1}(4) = a\text{.}\) So if we are solving an equation \(f(x) = 4\text{,}\) the solution would be \(x=f^{-1}(4)\text{.}\)
