Solution 4.2.10.1.

As in the last example, we start by writing down the formula for the slope of a secant line passing through \((3,f(3))\) and \((3+h,f(3+h))\text{.}\) The alternative would be to choose the second point as \((x,f(x))\text{,}\) or use any new variable in place of \(x\text{.}\) The value for \(f(3)\) is \(f(3)=2^3 = 8\text{.}\) Using the second point \((x,f(x))\text{,}\) we have

\begin{equation*} \frac{\Delta f}{\Delta x}\Big|_{3,x} = \frac{f(x)-f(3)}{x-3} = \frac{2^x-8}{x-3}\text{.} \end{equation*}

If we instead used \((3+h,f(3+h))\) as the second point, we would use \(f(3+h) = 2^{3+h} = 2^3 \cdot 2^h = 8 \cdot 2^h\text{,}\) resulting in a formula for the secant line slope given by

\begin{equation*} \frac{\Delta f}{\Delta x}\Big|_{3,3+h} = \frac{f(3+h)-f(3)}{h} = \frac{2^{3+h} - 8}{h}= \frac{8 \cdot 2^h - 8}{h}\text{.} \end{equation*}

Depending on which choice we used for the second point, we either need a limit as \(x \to 3\) or \(h \to 0\text{,}\)

\begin{align*} f'(3) &= \lim_{h \to 0}\frac{f(3+h)-f(3)}{h}\\ &= \lim_{x \to 3} \frac{f(x)-f(3)}{x-3} \end{align*}

Notice how we setup the table of values in our approximation. We will choose points closer and closer to \(x \to 3\) or \(h \to 0\) until it is clear that the decimal values are converging. The entire table that shows the convergence is necessary for showing that your approximation is good. We also need to include enough decimal values in our results to see if the convergence is actually happening.

\(x \to 3\) \(\displaystyle \frac{\Delta f}{\Delta x} = \frac{2^x-8}{x-3}\)
2.9 5.357361
2.99 5.526003
2.999 5.543256
With the data so far, we don't quite see convergence. The values for 2.99 and 2.999 have the first two digits in common of 5.5 for the slope, but the next digit is getting bigger. Sometimes it is easier to discover convergence by checking both sides, this time trying \(x \to 3^+\text{.}\)
\(x \to 3\) \(\displaystyle \frac{\Delta f}{\Delta x} = \frac{2^x-8}{x-3}\)
3.1 5.741877
3.01 5.564440
3.001 5.547100
The slope values for this table are decreasing. We can see that the value should be between 5.543256 and 5.547100. In fact, once we are close on both sides, the average will usually be quite close to the approximate value:

\begin{equation*} f'(3) \approx 5.545178\text{.} \end{equation*}

If we did our calculation using the second point as \((3+h,f(3+h))\text{,}\) we would have a similar table. This time, we will stay on a single side limit with \(h \to 0^+\text{.}\) Because we are not bracketing the value from both sides, we need to use closer and closer values for \(h \to 0\) before we obtain a similar degree of approximation.

\(h \to 0^+\) \(\displaystyle \frac{\Delta f}{\Delta x} = \frac{2^{3+h}-8}{h}\)
0.1 5.741877
0.01 5.564440
0.001 5.547100
0.0001 5.545369
0.00001 5.545197
0.000001 5.545179

This table has six entries, comparable to the previous calculation. The last three entries had slope values with digits 5.545 in common, and the last two had 5.5451 in common. Without additional entries, this is the degree of approximation that we might trust, and we would say

\begin{equation*} f'(3) \approx 5.5451\text{.} \end{equation*}

We don't know if the remaining digits are converging or not, and we don't really even know if the digits we wrote down have converged. This is why the two-sided approach is preferred, but for the assessment, either approach properly demonstrates the idea.
