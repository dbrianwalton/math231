We introduced inflection points as places where a continuous function with a tangent line change concavity. There should be a similar idea for where a function changes monotonicity. Suppose a function is increasing and then at a point \(x=a\) changes to decreasing. If we were to draw that picture, we would realize that \(f(a)\) must be greater than the values to the left (because it is increasing), but also greater than the values to the right (because it is decreasing). That makes the point a maximum value on an interval that extends to the left and right of \(x=a\) (a neighborhood of \(x=a\)).
