How do we find the value of \(\delta\text{?}\)

  • Identify the value of the limit \(L\) and the given value \(\epsilon \gt 0\text{.}\)

  • Calculate thresholds \(y=L-\epsilon\) and \(y=L+\epsilon\text{.}\) The inequality \(|f(x)-L| \lt \epsilon\) is equivalent to saying \(L-\epsilon \lt f(x) \lt L+\epsilon\text{.}\)

  • Find the points where \(f(x)\) crosses the thresholds by solving equations. Graphically, this will be where \(y=f(x)\) intersects the threshold lines or where there are jumps in the graph.

  • Identify an interval \((x_L,x_R)\) surrounding \(x=a\) (not counting at \(x=a\)) that satisfies \(|f(x)-L| \lt \epsilon\text{.}\)

  • Measure the distances from \(x=a\) to each end point:

    \begin{gather*} \delta_{-} = |x_L - a|\\ \delta_{+} = |x_R - a| \end{gather*}

    Choose \(\delta = \min(x_L, x_R)\text{.}\)

  • For all \(x\) in \((a-\delta, a) \cup (a,a+\delta)\text{,}\) we are guaranteed that \(|f(x)-L| \lt \epsilon\text{.}\)
