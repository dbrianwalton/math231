The process of finding the end points \(x_L\) and \(x_R\) will vary based on the particular formula, but otherwise this process is the same for every example. When the limit exists, a value for \(\delta\) will always be found. This is why the definition says that \(\delta\) exists for every \(\epsilon \gt 0\text{.}\) So if there is a value of \(\epsilon \gt 0\) for which no \(\delta\) can be found, the value of \(L\) must not be a limit.
