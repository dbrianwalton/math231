Example 2.4.7.

The following figures illustrate the basic ideas.

The function in Figure 2.4.8 illustrates several intervals of concavity. We start going left to right. When we first encounter the graph around \(x=-5\text{,}\) we can see that the graph is locally bending upwards. That is, the graph is above a tangent line on both sides. The curve continues to bend upward until the corner at \(x=-1\text{.}\) Assuming the graph has a similar behavior to the left of the viewing window, we would say \(f\) is concave up on \((-\infty,-1)\text{.}\) The point \(x=-1\) is not included because there is no well-defined slope at \(x=-1\text{.}\)

Going from \(x=-1\) to \(x=0\text{,}\) the graph is a straight line. That is, \(f\) has no concavity on \((-1,0)\text{.}\) Instead, we say \(f\) is linear on \([-1,0]\text{.}\)

Now, starting at \(x=0\) and going to the right, we see the function is curving downward. Far to the right, however, we see that the curve begins to curve back upward. We need to identify the point where the curving direction switches. In this example, it occurs at \(x=4\text{.}\) If we could draw a tangent line at \(x=4\text{,}\) the graph is below the tangent line for \(x \lt 4\) and above the tangent line for \(x \gt 4\text{.}\) Consequently, we say \(f\) is concave down on \((0,4]\) and concave up on \([4,\infty)\) (assuming the shape pattern continues outside the window).

The function in Figure 2.4.9 provides an additional example. On the far left, the graph is bending upwards, so we know there will be an interval that is concave up. Looking at the graph at \(x=-2\text{,}\) we can see the graph is now bending downwards, so there will be an interval that is concave down.

We check between these points to identify the inflection point. From a graph, we can only give a reasonable guess. The graph appears to be roughly straight for \(x\)-values close to \(x=3\text{.}\) A tangent line at \(x=3\) appears consistent with having the graph above the line on the left and below the line on the right. So \(x=3\) being the inflection point is a reasonable guess. We conclude that \(f\) is concave up on \((-\infty,3]\text{.}\)

We earlier noted that the graph is concave down around \(x=2\text{.}\) Looking further to the right, we should notice that the graph is concave up around \(x=0\text{.}\) It appears that an inflection point is at or near \(x=-1\text{.}\) So we conclude \(f\) is concave down on \([-3,-1]\text{.}\)

Using similar reasoning, we decide there are inflection points at \(x=1\) and at \(x=3\text{.}\) We further conclude that \(f\) is concave up on \([-1,1]\text{,}\) concave down on \([1,3]\text{,}\) and concave up on \([3,\infty)\text{.}\)
