To find \(\displaystyle \lim_{x \to -2} g(x)\text{,}\) we need to look at the left-limit and the right-limit first. Checking our restricted domains, we see that for \(x \to -2\) with \(x \lt -2\text{,}\) we will use \(g(x)=2x+3\) and write:

\begin{equation*} \lim_{x \to -2^-} g(x) = \lim_{x \to -2^-} [2x+3]\text{.} \end{equation*}

That formula is algebraic and so the limit will use the value if it makes sense (continuity), so that we find

\begin{equation*} \lim_{x \to -2^-} g(x) = \lim_{x \to -2^-} [2x+3] = 2(-2)+3 = -1\text{.} \end{equation*}

For \(x \to -2\) with \(x \gt -2\text{,}\) we see we need to use \(g(x)=1-x\text{,}\) which is also algebraic and continuous when defined:

\begin{equation*} \lim_{x \to -2^+} g(x) = \lim_{x \to -2^+}[1-x] = 1-(-2) = 3\text{.} \end{equation*}

At this point, we compare those answers and see that they are not equal, and the two-sided limit does not exist:

\begin{equation*} \lim_{x \to -2} g(x) \; \text{d.n.e.}\text{.} \end{equation*}
