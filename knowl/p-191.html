To find \(x\) so that \(f(x)=g(0)\text{,}\) we need to think through a few steps. The symbol \(g(0)\) represents an output from \(g\) corresponding to an input of \(0\text{.}\) Looking in the table with \(x=0\) we find \(g(0)=1\text{.}\) We replace the symbol \(g(0)\) in the equation with this value to restate our question: find \(x\) so that \(f(x)=1\text{.}\)
