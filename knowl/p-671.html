To find the equation of the tangent line, we would use the point-slope formula, \(y=m(x-a)+b\text{.}\) This requires knowing the point \((a,b)\) and the slope \(m\text{.}\) The derivative has provided the slope. The original function is used to find the \(y\)-value of the point. At \(x=2\text{,}\) we have \(g(2)=2^2-2(2)=0\text{,}\) so our point would be \((2,0)\text{.}\) Using the slope \(g'(2)=2\text{,}\) the tangent line would be

\begin{equation*} y=2(x-2)+0 = 2(x-2)\text{.} \end{equation*}

At \(x=3\text{,}\) we have \(g(3)=3^2-2(3) = 3\) with a corresponding point \((3,3)\) and tangent line

\begin{equation*} y=6(x-3)+3\text{.} \end{equation*}
