So far, we have talked about the derivative as the slope of the tangent line for a function at a particular point. Because a function theoretically could have a tangent line at each point in the domain, we really should think about the derivative as a function. The most common notation for a derivative of a function is to put a prime on the name of the original function using an apostrophe. For a function named \(f\text{,}\) the derivative would be named \(f'\text{.}\) For a function named \(\mathrm{brian}\text{,}\) the derivative would be named \(\mathrm{brian}'\text{.}\)
