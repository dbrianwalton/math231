Function notation consists of using the name of the function followed immediately by a value or expression inside of parentheses. That value or expression represents the value of the independent variable in the relation. In algebra, a variable name followed by something inside of parentheses would mean there is an implicit (unstated) multiplication. However, for functions, this is not multiplication but mapping. The expression \(f(2)\) refers to “the output value of \(f\) when the input value is 2”, or more succinctly “the value of \(f\) at 2”.
