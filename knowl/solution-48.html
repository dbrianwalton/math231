Solution 4.2.9.1.

First, recall that \(f'(3)\) and \(\frac{df}{dx}\Big|_{3}\) are different notations for the same thing. The definition of the derivative tells us we should consider the two points \((3,f(3))\) and \((3+h,f(3+h))\) and compute the slope. We will start by computing the formulas for the function values:

\begin{align*} f(x) &= x^2 + x\\ f(3) &= 3^2 + 3 = 12\\ f(3+h) &=(3+h)^2 + (3+h)\\ &= 9 + 6h + h^2 + 3 + h\\ &= 12 + 7h + h^2 \end{align*}

The slope of the secant line will be:

\begin{align*} \frac{\Delta f}{\Delta x}\Big|_{3,3+h} &= \frac{f(3+h)-f(3)}{h}\\ &= \frac{(12+7h+h^2)-12}{h}\\ &= \frac{7h+h^2}{h} \end{align*}

It is at this point where we would be able to cancel out the factor \(h\text{.}\) For the purpose of numerical approximations, however, cancellation is not required. For more advanced examples, it will not be possible. Instead, what we want to do is create a table of values for progressively smaller values of \(h\) using this slope formula until we see the value converging to a desired degree of accuracy.

One common choice would be to try decreasing powers of 10.

\(h\) \(\frac{\Delta f}{\Delta x}\)
0.1 7.1
0.01 7.01
0.001 7.001
0.0001 7.0001
We should also be in a habit of checking negative powers of 10.
\(h\) \(\frac{\Delta f}{\Delta x}\)
-0.1 6.9
-0.01 6.99
-0.001 6.999
-0.0001 6.9999
In both cases, we can see that the slope of the secant line is converging to a value of \(7\text{,}\) so we would say \(f'(3) \approx 7\text{.}\)

Notice that if we did simplify the formula of the slope of the secant line, we would have \(\frac{\Delta f}{\Delta x} = 7 + h\text{.}\) Can you see how the values calculated in the table relate to this? And can you see how a limit with \(h \to 0\) results in a value of 7?
