In applications, variables represent actual physical quantities. A function relating these variables will need to respect what type of value is used as the input and what type of value is created as an output. For example, if I were doing an electrophoresis gel observation with DNA, then there is a relation between the mass of a DNA segment and the distance it travels through the gel during the observation. If we let \(M\) be the mass of a segment and \(D\) be the distance traveled, then our observation would create a map \(M \mapsto D\text{.}\) If we said \(D = f(M)\text{,}\) then the function \(f\) is expecting mass of a DNA fragment as an input and gives a distance as output. It would not make any sense to talk about \(f(M^2)\) because \(M^2\) is no longer the mass of a fragment.
