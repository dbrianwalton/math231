Solution 1.3.6.1.

Recall that to use factoring, the equation must have a zero on one side. We need to eliminate the 12:

\begin{align*} x^2-4x &= 12\\ x^2-4x -12 &= 0 \end{align*}

We can now identify \(a=1\text{,}\) \(b=-4\text{,}\) and \(c=-12\text{.}\) Notice how we must include the signs as part of the coefficients.

The product \(ac=1(-12) = -12\) needs to be factored. If you can think through factors in your head, that is great. Otherwise, it can be helpful to create a table of possible factors.

\(p\) \(q\) \(p+q\)
-1 12 11
-2 6 4
-3 4 1
-4 3 -1
-6 2 -4
-12 1 -11
Because \(b=-4\text{,}\) we want to use \(p=-6\) and \(q=2\text{.}\)

We will now rewrite the trinomial using four terms and group the resulting pairs, which we use to look for common factors:

\begin{align*} x^2-4x -12 &= 0\\ x^2-6x + 2x -12 &= 0\\ (x^2-6x) + (2x -12) &= 0\\ x(x-6) + 2(x -6) &= 0 \end{align*}

We see that each pair has the common factor \(x-6\text{.}\) Factoring this out, we have our factorization.

\begin{align*} x(x-6) + 2(x -6) &= 0\\ (x+2)(x-6) &= 0 \end{align*}

Once the equation is factored, we can consider each factor separately. One possibility is that \(x+2=0\text{,}\) which has a solution \(x=-2\text{.}\) The other possibility is that \(x-6=0\text{,}\) which has a solution \(x=6\text{.}\) The factoring theorem guarantees that these are the only possible solutions.
