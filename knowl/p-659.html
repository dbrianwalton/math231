When calculating the derivative as a function using the definition, we want to consider an arbitrary value in the domain. Instead of a specific value \(a\text{,}\) we will use a variable \(x\) as a placeholder for this arbitrary value. We will compute expressions for \(f(x)\) and \(f(x+h)\) and again calculate a formula for the slope of the secant line joining these points, after which we will work to evaluate the limit as \(h \to 0\text{.}\)
