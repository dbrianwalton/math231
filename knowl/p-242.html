A composition is when we use the output value of one function as the input value of another function. That is, if \(f\) and \(g\) are functions, then \(f\big(g(x)\big)\) is the composition of \(f\) with \(g\text{.}\) Overall, this constructs a new function \(f \circ g\text{,}\)

\begin{equation*} f \circ g(x) = f\big(g(x)\big)\text{.} \end{equation*}

We are using composition every time we evaluate a function using substitution with a formula as its input. When we identified a formula that was inside of a square root, we were identifying a composition. The skill we want to focus on in this subsection is in recognizing a sequence of operations as a composition of two functions.
