Definition 1.2.1. Intervals.

An interval is a set of real numbers consisting of all numbers between two end-points, possibly including the end-points.

  • \((a,b) = \{x : a \lt x \lt b\}\) is called the open interval and excludes both end-points.

  • \([a,b] = \{x : a \le x \le b\}\) is called the closed interval and includes both end-points.

  • \((a,b] = \{x : a \lt x \le b\}\) is neither open nor closed because it excludes \(a\) and includes \(b\text{.}\)

  • \([a,b) = \{x : a \le x \lt b\}\) is neither open nor closed because it includes \(a\) and excludes \(b\text{.}\)
