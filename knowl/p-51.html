Sets are collections of objects from some universe of possibilities. In calculus, we primarily focus on sets of real numbers. A set can be a described by listing its members, when that is possible. If a set has a finite number of values (meaning the list has a specific number of values), then we can list the included values inside of curly braces, as in \(\{2, 3, 5, 7\}\text{.}\)
