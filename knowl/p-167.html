Function notation provides a means to write about the value of a dependent variable given the value of the independent variable. Suppose we have a function \(x \mapsto y\) with a name of \(f\text{,}\) meaning that knowing the value of \(x\text{,}\) \(f\) allows us to determine the specific value of \(y\text{.}\) We would write

\begin{equation*} y=f(x)\text{.} \end{equation*}

Function notation does not simply mean reading this as “\(y\) is a function of \(x\)”. Rather, this means that \(f\) provides the rule connecting the values of \(x\) and \(y\text{.}\)
