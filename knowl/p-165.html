Functions provide the main tool in mathematics to describe a relationship between two variables. Whenever knowing the value of one variable (independent) can be used to calculate or determine the precise value of th other variable (dependent), that relation is a function. A function is also known as a map and we write \(x \mapsto y\) to say that “\(x\) maps to \(y\)” or “\(y\) is a function of \(x\)”. Like other mathematical objects, a function can be given a name (looks like another variable) and the discipline of mathematics has established function notation to understand how to write about and manipulate functions.
