Suppose that \(f(x)\) has a local maximum at \(x=c\text{.}\) Then for \(c-\delta x \lt c\text{,}\) we know \(f(x) \lt f(c)\) and \(x \lt c\) so that

\begin{equation*} \frac{f(x)-f(c)}{x-c} \gt 0\text{.} \end{equation*}

Using a limit as \(x \to c^-\text{,}\) we find

\begin{equation*} \frac{df}{dx}\Big|_{c^-} = \lim_{x \to c^-} \frac{f(x)-f(c)}{x-c} \ge 0 \end{equation*}

or does not exist. Meanwhile \(c \lt x \lt c+\delta\text{,}\) we know \(f(x) \lt f(c)\) and \(x \gt c\) so that

\begin{equation*} \frac{f(x)-f(c)}{x-c} \lt 0\text{.} \end{equation*}

Using a limit as \(x \to c^+\text{,}\) if it exists, we find

\begin{equation*} \frac{df}{dx}\Big|_{c^+} = \lim_{x \to c^+} \frac{f(x)-f(c)}{x-c} \le 0\text{.} \end{equation*}

If \(f'(c)\) exists, the one-sided derivatives must agree, which can only occur if \(f'(c)=0\text{.}\) If the one-sided derivatives do not exist or if they disagree, then \(f'(c)\) itself does not exist.
