Start by finding the second derivative.

\begin{align*} f(x) &= x^5+2x^4-10x^3-36x^2 + 5x + 10\\ f'(x) &= 5x^4+8x^3-30x^2-72x+5\\ f''(x) &= 20x^3+24x^2-60x-72 \end{align*}

Next, factor (either by grouping or technology) to rewrite:

\begin{equation*} f''(x) = 4(x^2-3)(5x+6)\text{.} \end{equation*}

Setting each factor to zero, we find three points where \(f''(x)=0\text{:}\) \(x=\pm \sqrt{3}\) and \(x=-\frac{6}{5}\text{.}\) We use these points to do sign analysis on the factored version of \(f''(x)\text{.}\)

We finish by interpreting. On intervals where \(f''(x)\) is positive, the original function \(f(x)\) is concave up. On intervals where \(f''(x)\) is negative, the function \(f(x)\) is concave down. Because \(f'(x)\) is defined and continuous everywhere, we can include the end points of the intervals.

  • \(f(x)\) is concave up on \([-\sqrt{3},-\frac{6}{5}]\) and on \([\sqrt{3},\infty)\text{.}\)

  • \(f(x)\) is concave down on \((-\infty,-\sqrt{3}]\) and on \([-\frac{6}{5}, \sqrt{3}]\text{.}\)
