Definition 5.1.5. Local Extreme Values.

A function \(f\) has a local or relative maximum at \(x=c\) if there exists \(\delta \gt 0\) so that for all \(x \in (c-\delta,c+\delta)\text{,}\) \(f(x)\) is defined and \(f(x) \le f(c)\text{.}\)

A function \(f\) has a local or relative minimum at \(x=c\) if there exists \(\delta \gt 0\) so that for all \(x \in (c-\delta,c+\delta)\text{,}\) \(f(x)\) is defined and \(f(x) \ge f(c)\text{.}\)
