When thinking about numbers on the number line, it is useful to first identify their approximate decimal representation so that we can correctly find their position. The value \(\pi\) has an approximate decimal representation of \(\pi \approx 3.14159\text{.}\) When thinking about \(-\pi\text{,}\) we need to be careful that we put this to the left of -3 on the number line.
