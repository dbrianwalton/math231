To find \(f(-2)\text{,}\) we first interpret notation. To find \(g(2)\text{,}\) we again interpret notation. The symbol \(g(2)\) is the output value of the map with input of \(2\text{.}\) We look in the table of \(g\text{.}\) The column headed by \(x\) contains input values while the column headed by \(g(x)\) contains corresponding output values. Finding \(x=2\) in the first column is paired with \(g(x)=-4\) in the second column. So \(g(2)=-4\text{.}\)

To find \(g(f(1))\text{,}\) we need to think carefully about what this means. A function name with parentheses means that whatever is inside the parentheses represents the input value. Both \(f\) and \(g\) are functions here. So what is the input? The function \(g\) (outer function) has parentheses surrounding \(f(1)\text{,}\) so the value of \(f(1)\) is the input. Because \(f\) is also a function, its parentheses indicate that its input is 1.

\begin{equation*} g(f(1)) = g(3)\text{.} \end{equation*}

The input \(x=3\) for the function \(g\) is paired with an output \(g(x)=-3\text{,}\) so \(g(f(1))=-3\text{.}\)
