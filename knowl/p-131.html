We are left using the quadratic formula:

\begin{gather*} x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\\ x = \frac{-(-3) \pm \sqrt{(-3)^2 - 4(3)(-8)}}{2(3)}\\ x = \frac{3 \pm \sqrt{9 + 96}}{6}\\ x = \frac{3 \pm \sqrt{105}}{6} \end{gather*}

We find the solutions \(x = \frac{3 + \sqrt{105}}{6}\) and \(x = \frac{3 - \sqrt{105}}{6}\text{.}\)
