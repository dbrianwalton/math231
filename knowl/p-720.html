The derivative of a function \(f(x)\) at a point \(x=a\) is the slope of the tangent line (linearization) of the function. In order to make sense of what that would mean, we needed to define the derivative as the limit of the slope of a secant line passing through the point \((a,f(a))\) and another point \((a+h, f(a+h))\) as the second point is chosen closer and closer to \(x=a\) by using \(h \to 0\text{.}\) Because this process could be done, in principle, at any point in the domain, this actually defines a function called the derivative function \(f'(x)\text{,}\) defined whenever this limit exists. In certain algebraic cases, we could allow a specific point \(a\) to be replaced by a generic variable \(x\) and still compute the limit.
