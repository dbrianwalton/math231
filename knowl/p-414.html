The first four values are associated with \(x=-1\) or \(x \to -1\text{.}\) Because there is a closed point at \((-1,1)\text{,}\) this value is associated with the value of the function at the point \(x=-1\text{.}\) We have \(f(-1) = 1\text{.}\) The limits are based on the end-points of the segments to the left and right of \(x=-1\text{.}\) To the left, we follow the decreasing linear graph to its end-point, illustrated by the open end-point at \((-1,-1)\text{.}\) The \(y\)-value of this point is the limit of the function using this left segment, and we write:

\begin{equation*} \lim_{x \to -1^-} f(x) = -1\text{.} \end{equation*}

To the right of \(x=-1\text{,}\) the graph uses the increasing linear segment. Again, the end-point of this segment at \(x=-1\) is \((-1,-1)\text{,}\) so that we know

\begin{equation*} \lim_{x \to -1^+} f(x) = -1\text{.} \end{equation*}

Because both segments terminate at the same point, this will be the value of the two-sided limit

\begin{equation*} \lim_{x \to -1} f(x) = -1\text{.} \end{equation*}
