To find \(\delta\text{,}\) we want to see how big an interval we get that is symmetric around \(x=4\) and stays inside of this set. The left end point at \(x=3.61\) is a distance \(|3.61-4| = 0.39\) away from \(x=4\text{.}\) The right end point at \(x=4.41\) is a distance \(|4.41-4| =0.41\) away from \(x=4\text{.}\) Consequently, if we make our symmetric interval extend \(\delta = 0.39\) away from \(x=4\text{,}\) it will stay inside of both sides of the interval. If we instead chose \(\delta = 0.41\text{,}\) the interval will be inside the set on the right, but will go too far on the left.
