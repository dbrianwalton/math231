The function \(f(x)\) will be the formula in the limit, so \(f(x)=\sqrt{x}\) for this problem. The proposed limit is \(L=2\text{.}\) Starting with a graph may be most helpful.

  • Graph \(y=f(x)\text{.}\)

  • Graph \(y=L-\epsilon=2-0.1=1.9\) and \(y=L+\epsilon=2+0.1=2.1\text{.}\)

  • Find where \(f(x)\) intersects or jumps the thresholds to the left and right of \(x=4\text{.}\)

Our algebraic task is to find these points, so we will be solving the equations \(\sqrt{x} = 1.9\) and \(\sqrt{x} = 2.1\text{.}\) The first equation gives \(x=1.9^2 = 3.61\) and the second gives \(x=2.1^2 = 4.41\text{.}\) The inequality \(|f(x)-2| \lt 0.1\) therefore includes the interval \((3.61, 4.41)\text{,}\) which does surround \(x=4\text{.}\)

To find \(\delta\text{,}\) we want to see how big an interval we get that is symmetric around \(x=4\) and stays inside of this set. The left end point at \(x=3.61\) is a distance \(|3.61-4| = 0.39\) away from \(x=4\text{.}\) The right end point at \(x=4.41\) is a distance \(|4.41-4| =0.41\) away from \(x=4\text{.}\) Consequently, if we make our symmetric interval extend \(\delta = 0.39\) away from \(x=4\text{,}\) it will stay inside of both sides of the interval. If we instead chose \(\delta = 0.41\text{,}\) the interval will be inside the set on the right, but will go too far on the left.

We always need to choose the smaller distance, which in this case will be \(\delta = 0.39\text{.}\) The symmetric interval is \((4-0.39,4+0.39) = (3.61,4.39)\text{.}\) The value at \(x=4\) is not required to be in the set for the limit definition to be satisfied, but in this case because \(\sqrt{x}\) is defined at \(x=4\) and satisfies the inequality, it can be included.
