Now, the phrasing that was used involving the idea of “closer” was not quite stated precisely. It is not required that closer values of \(x\) cause the \(y\) values to be progressivly closer. Rather, it is about being able to guarantee closeness. A limit is really about \(y\)-values being guaranteed to be close to the value of the limit so long as the \(x\)-values are close enough. Typically, smaller allowable errors for \(y\) will require smaller deviations for \(x\text{.}\) However, it is typical to use a phrase like “the value of \(f(x)\) approaches \(L\) as \(x\) approaches \(c\)” when stating a limit

\begin{equation*} \lim_{x \to c} f(x) = L\text{.} \end{equation*}
