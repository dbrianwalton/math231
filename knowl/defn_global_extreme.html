Definition 5.1.3. Global Extreme Values.

A function \(f\) with domain \(D\) has a maximum value of \(M\) if there is some value \(x_M \in D\) with \(f(x_M) = M\) and for every other \(x \in D\text{,}\) \(f(x) \le M\text{.}\)

A function \(f\) with domain \(D\) has a minimum value of \(m\) if there is some value \(x_m \in D\) with \(f(x_m) = m\) and for every other \(x \in D\text{,}\) \(f(x) \ge m\text{.}\)
