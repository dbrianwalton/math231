Sign analysis is concerned about where a function is positive or negative. These values are separated by the value \(L=0\text{.}\) The IVT says that a function can switch from positive to negative only by passing through points where \(f(x)=0\) or where \(f(x)\) is not continuous. This is why we mark all points on the number line where \(f(x)=0\) (roots) or where \(f(x)\) is not continuous (such as undefined points).
