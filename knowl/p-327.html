The first group of attributes are classified as describing the monotonicity of a function. The attributes of interest are increasing, decreasing, and constant, and they are properties that a function has on a set. An intuitive understanding of these attributes can come by thinking about moving along a graph of the function restricted to the set in question from left to right. The function is increasing on the set if the \(y\)-values of the graph always go up as you move left to right. The function is decreasing on the set if the \(y\)-values go down. The function is constant on the set if all \(y\)-values are the same.
