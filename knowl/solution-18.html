Solution 2.1.1.1.

When thinking about a problem like this, I like to imagine blocks representing components of the formula. I might write \(\boxed{f(2x)} - \boxed{f(x-3)}\text{.}\) That is, I should recognize that there are two formulas I need to find, \(f(2x)\) and \(f(x-3)\text{,}\) and I will later subtract them.

One method would be to find the individual formulas first using substitution.

\begin{align*} f(x) &= x^2 - 2x\\ f(2x) &= (2x)^2 - 2(2x)\\ &= (2x)(2x) - 2(2x)\\ &= 4x^2 - 4x\\ f(x-3) &= (x-3)^2 - 2(x-3)\\ &= (x-3)(x-3) - 2(x-3)\\ &= x^2-3x-3x+9-2x+6\\ &= x^2-8x+15 \end{align*}

Once we know the individual formulas, we combine them to create the value in question. We need to be careful to think of these formulas as giving a single value, so we place them inside parentheses to treat the formula as a grouped expression:

\begin{align*} f(2x) - f(x-3) &= (4x^2-4x) - (x^2-8x+15)\\ &= 4x^2-4x - x^2 + 8x - 15\\ &= 3x^2 + 4x - 15 \end{align*}

An alternative method would be to do all of the substitution at once. If this approach is used, extra caution must be taken to ensure that we treat all of the grouped expressions as single values. Use parentheses carefully:

\begin{align*} f(2x) - f(x-3) &= \Big((2x)^2-2(2x)\Big) - \Big((x-3)^2 - 2(x-3)\Big)\\ &= \Big(4x^2-4x\Big) - \Big(x^2-6x+9-2x+6\Big)\\ &= 4x^2-4x - x^2 +8x -15\\ &= 3x^2+4x -15. \end{align*}
