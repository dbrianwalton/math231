Let \(f(x)\) be a function defined algebraically, so that it only involves operations of arithmetic (addition, subtraction, multiplication, and division) and powers (including roots). Let \(x=a\) be a value in the domain of \(f\text{.}\) If there exists \(\delta \gt 0\) so that \((a-\delta,a+\delta)\) is a subset of the domain of \(f\text{,}\) then \(f(x)\) is continuous at \(x=a\text{.}\)
