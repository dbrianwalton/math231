Proof

Define the function measuring the difference between \(f(x)\) and \(g(x)\text{,}\)

\begin{equation*} D(x) = g(x)-f(x)\text{.} \end{equation*}

On the interval \(I\text{,}\) \(f\) and \(g\) are differentiable and have the same derivative values, so that we have

\begin{equation*} D'(x) = \frac{d}{dx}[g(x)-f(x)] = g'(x)-f'(x) = 0\text{.} \end{equation*}

The first derivative test for monotonicityÂ 5.2.1, which was an immediate consequence of the Mean Value Theorem, proved that when a function has a derivative equal to zero on an interval, that function must be a constant on that interval, \(D(x)=C\) for some constant \(C\text{.}\) That is \(g(x)-f(x)=C\) or \(g(x)=f(x)+C\text{.}\)
