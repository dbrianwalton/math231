As in the last example, we start by writing down the formula for the slope of a secant line passing through \((3,f(3))\) and \((3+h,f(3+h))\text{.}\) The alternative would be to choose the second point as \((x,f(x))\text{,}\) or use any new variable in place of \(x\text{.}\) The value for \(f(3)\) is \(f(3)=2^3 = 8\text{.}\) Using the second point \((x,f(x))\text{,}\) we have

\begin{equation*} \frac{\Delta f}{\Delta x}\Big|_{3,x} = \frac{f(x)-f(3)}{x-3} = \frac{2^x-8}{x-3}\text{.} \end{equation*}

If we instead used \((3+h,f(3+h))\) as the second point, we would use \(f(3+h) = 2^{3+h} = 2^3 \cdot 2^h = 8 \cdot 2^h\text{,}\) resulting in a formula for the secant line slope given by

\begin{equation*} \frac{\Delta f}{\Delta x}\Big|_{3,3+h} = \frac{f(3+h)-f(3)}{h} = \frac{2^{3+h} - 8}{h}= \frac{8 \cdot 2^h - 8}{h}\text{.} \end{equation*}
