To find \(f(-2)\text{,}\) we first interpret notation. The symbol \(f(-2)\) stands for the output value of a function \(f\) with an input of \(-2\text{.}\) That is, we want the \(y\)-value of the graph corresponding to the value \(x=-2\text{.}\) The vertical line at \(x=-2\) passes through an open point at \((-2,1)\) and a closed point at \((-2,-1)\text{.}\) The solid point is the point in the relation, so \(f(-2) = -1\text{.}\) The open point represents a limit of the segment of the graph and not an actual point in the graph.
