We also previously found

\begin{gather*} \lim_{x \to 1^-} g(x) = 0\\ \lim_{x \to 1^+} g(x) = 0 \end{gather*}

As these have the same value, we know \(\lim_{x \to 1} g(x) = 0\text{.}\) Looking at the restricted domains, we find that \(x=1\) is in the domain \((-2,1]\) so that \(g(1)\) will use the formula \(g(x)=1-x\) to give \(g(1)=1-1=0\text{.}\) Because \(\displaystyle \lim_{x \to 1}g(x) = g(1)\text{,}\) we have verified that \(g(x)\) is continuous at \(x=1\text{.}\)
