Definition 1.2.6. Union of Sets.

Given two sets \(A\) and \(B\text{,}\) the union of \(A\) and \(B\) is a new set \(A \cup B\) defined as all values \(x\) so that \(x \in A\) or \(x \in B\text{.}\)

If we are given more than two sets, the union of the collection is a new set whose values must be in at least one of the sets that were combined.
