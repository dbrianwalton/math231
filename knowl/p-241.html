Once we have considered the restrictions for each risk factor (division and even roots), the domain of the function overall requires that all restrictions are satisfied. If we imagine a number line of potential domain values, we remove \(x=-1\) for the first division, \(x=-4\) and \(x=1\) for the second division, and all values \(-1 \lt x \lt 3\) for the square root. Notice that \(x=1\) is not allowed for two reasons. The domain is the set of remaining values,

\begin{equation*} D = (-\infty, -4) \cup (-4, -1) \cup [3,\infty)\text{.} \end{equation*}
