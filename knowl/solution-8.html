Solution 1.3.1.1.

Isolation is a good method for this problem because the variable \(x\) appears in exactly one location, as part of the expression \(2(3x-1)+4\text{.}\) To use isolation, we look at this expression and identify the last operation (according to the order of operations). The expression involves multiplication by 2 and addition of 4.

Addition occurs after multiplication, so the last operation is the addition of 4. What will cancel adding 4? Subtracting 4, or more precisely, adding -4. Our first step in isolation is to add -4 to both sides of the equation:

\begin{align*} 2(3x-1)+4 &= 8\\ 2(3x-1) &= 8 + -4 = 4 \end{align*}

We now have a new (and simpler) equation that is guaranteed to have the same solutions. We say these equations are equivalent.

The new expression still containing \(x\) is \(2(3x-1)\text{.}\) The last operation is multiplication of \(3x-1\) by 2. The inverse operation is to multiply both sides by \(\frac{1}{2}\text{.}\)

\begin{align*} 2(3x-1) &= 4\\ 3x-1 &= \frac{1}{2}(4) = 2 \end{align*}

The new last operation is adding -1, so the inverse step is adding +1.

\begin{align*} 3x-1 &= 2\\ 3x &= 2+1 = 3 \end{align*}

The only operation left is multiplication by 3, and its inverse is multiplication by \(\frac{1}{3}\text{:}\)

\begin{align*} 3x &= 3\\ x &= \frac{1}{3}(3) = 1 \end{align*}

If we were to rewrite the steps without the explanatory sentences, we would have:

\begin{align*} 2(3x-1)+4 &= 8\\ 2(3x-1) &= 4\\ 3x-1 &= 2\\ 3x &= 3\\ x &= 1 \end{align*}

There is a single solution, \(x=1\text{.}\)
