Visualize this in terms of a graph. We will plot \(y=f(x)\) and the two threshold lines \(y=L-\epsilon\) and \(y=L+\epsilon\text{.}\) We want to identify the set of \(x\)-values so that \(L-\epsilon \lt f(x) \lt L+\epsilon\text{.}\) This set needs to include an interval surrounding but excluding \(x=0\) (called a punctured neighborhood). Looking at FigureÂ 3.5.3, everything looks good when we use \(\epsilon = 0.1\) (left image). If we zoom in further and use \(\epsilon = 0.01\) (right image), the interval barely extends to the right of \(x=0\text{.}\) It is clear that if we choose a smaller value of \(\epsilon\text{,}\) the set of points between the thresholds will be entirely to the left of \(x=0\text{.}\) This is because the limit is not 1.09 but some value slightly greater.
