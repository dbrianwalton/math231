Solution 2.1.8.1.

If the idea of visualizing the restricted domain helps you, we might start by creating a number line showing where the different rules are used. This doesn't really add any new information, so this is strictly a personal preference.

We work through each value in turn. First, we look at the value of the input. Then we decide which restricted domain it belongs to. Finally, we use the formula for the restricted domain to find the value.

  1. To find \(g(-2)\text{,}\) we recognize \(x=-2\) is in \((-\infty,-1]\) and so use \(g(x)=x^2-2\text{:}\)

    \begin{equation*} g(-2) = (-2)^2-2 = 4-2 = 2\text{.} \end{equation*}

  2. To find \(g(-1)\text{,}\) we recognize \(x=-1\) is the end-point included in \((-\infty,-1]\) and so again use \(g(x)=x^2-2\text{:}\)

    \begin{equation*} g(-1) = (-1)^2-2 = 1-2 = -1\text{.} \end{equation*}

  3. To find \(g(0)\text{,}\) we recognize \(x=0\) is in \((-1,1)\) and so use \(g(x)=2+3x\text{:}\)

    \begin{equation*} g(0) = 2+3(0) = 2\text{.} \end{equation*}

    \(g(0)\)

  4. To find \(g(1)\text{,}\) we realize \(x=1\) is not in any of the restricted domains. \(g(1)\) does not exist.

  5. To find \(g(3)\text{,}\) we see \(x=3\) is the end-point included in \([3,\infty)\) and uses \(g(x)=x^2+2\text{:}\)

    \begin{equation*} g(3) = (3)^2+2 = 9+2 = 11\text{.} \end{equation*}
