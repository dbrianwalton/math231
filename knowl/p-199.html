To evaluate a piecewise function at a point, the input value needs to be compared to the restricted domains. Whichever domain restriction the input value belongs determines which formula rule will be used. In the preceding example, to find \(f(2)\text{,}\) we look at the input \(x=2\) and recognize it belongs to the restriction \((0,3]\text{.}\) So the formula will be \(f(x)=x^2\) when \(x=2\) to give \(f(2)=(2)^2 = 4\text{.}\) On the other hand, \(f(-2)\) will use \(f(x)=3-x\) since \(x=-2\) is in the restriction \([-2,0)\text{,}\) giving \(f(-2) = 3-(-2)=5\text{.}\)
