Elementary Limits.

Functions defined by one of the following elementary forms have limits directly justified by the definition of a limit.

  • Constant Functions: \(f(x)=c\) for any real number \(c\text{,}\)

    \begin{equation*} \lim_{x \to a} c = c\text{.} \end{equation*}

  • Identity Function: \(f(x)=x\text{,}\)

    \begin{equation*} \lim_{x \to a} x = a\text{.} \end{equation*}

  • Linear Functions: \(f(x)=mx+b\) for any real numbers \(m,b\text{,}\)

    \begin{equation*} \lim_{x \to a} [mx+b] = ma+b\text{.} \end{equation*}

Ultimately, what these limit rules say is that these elementary functions are continuous.
