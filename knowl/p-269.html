Examples of compositions might include:

  • \(\sqrt{x^2+3x}\text{:}\) The last operation is the square root acting on a formula. So this is a composition \(f \circ g(x)\) of outer function \(f(x)=\sqrt{x}\) with inner function \(g(x)=x^2+3x\text{.}\)

  • \((\sqrt{x}+3)^4\text{:}\) The last operation is the exponent acting on a formula. So this is also a composition \(f \circ g(x)\text{,}\) but with an outer function \(f(x)=x^4\) and an inner function \(g(x)=\sqrt{x}+3\text{.}\)
