Item.

Identify the domain and solve the corresponding global extreme problem.

There were no extra constraints about the values of the variables \(x\) and \(y\text{.}\) There is a physical constraint that the values can not be negative. Our formula for \(y\) has a domain constraint that \(x \ne 0\text{,}\) so we have a domain \((0,\infty)\text{.}\)

This domain is not a closed interval, so global extreme values are not guaranteed. We will need to consider limits \(x \to 0\) and \(x \to \infty\text{.}\)

\begin{align*} \lim_{x \to 0^+} A(x) &= \lim_{x \to 0^+} [2x^2 + 8x^{-1}]\\ &= 0 + \infty = +\infty\\ \lim_{x \to \infty} A(x) &= \lim_{x \to \infty} [2x^2 + 8x^{-1}]\\ &= \infty + 0 = \infty \end{align*}

We can see that there will be no global maximum for the surface area. But we will be able to find a global minimum.

We find critical points by solving \(A'(x) = 0\text{.}\)

\begin{align*} A'(x) &= \frac{d}{dx}[2x^2 + 8x^{-1}]\\ &= 4x - 8x^{-2} = 4x - \frac{8}{x^2}\\ &= \frac{4x^3-8}{x^2} \end{align*}

The critical points will be solutions of \(4x^3-8 = 0\) or \(x^3=2\text{.}\) The only critical point is at \(x=\sqrt[3]{2}\text{.}\) Because \(A''(x) = 4+16x^{-3}\) has \(A''(\sqrt[3]{2}) = 12\) is positive, we know that \(A\) has a local minimum at this point. Because there are no other critical points and we already eliminated both ends of the interval as being unbounded positive values, the box with minimum surface area will be constructed with \(x = \sqrt[3]{2}\text{.}\)
