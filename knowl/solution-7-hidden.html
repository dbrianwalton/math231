The solution follows the same patterns as our example involving individual intervals. The major difference is that we are starting with compound sets. When we graph our representations for \(A\) and \(B\text{,}\) we will want to be clear that each set has multiple parts.

To find the union, we look for all values that are in at least one of the sets. With the number line representations, we can scan from left to right. The set \(A\) includes all values until -1, including that value; the set \(B\) immediately begins, also including -1, and continuing until 1. The union must include all of these values. When we look at the value \(1\text{,}\) we see \(1 \not in A\) and \(1 \not in B\text{,}\) so \(1 \not in A \cup B\text{.}\) But then \(A\) starts up immediately and includes all values until 4. The set \(B\) includes all values from 3 until 5. This overlap reduces to a single interval \((1,5)\text{.}\)

Altogether, this means

\begin{equation*} A \cup B = (-\infty,1) \cup (1,5)\text{.} \end{equation*}

Notice how the union has a gap of a single value.

The intersection of the sets looks for points that are in both sets. We look at the original sets to see where they overlap. Both sets include -1, but none of the values immediately to the left or right are in both sets. This will be an isolated value in our intersection. The value 1 might also look like it is in both sets, but in fact is in neither set. The next set of values in the intersection is the interval from -3 to 4. Because \(-3 \not \in B\text{,}\) it will not be in the intersection. On the other hand, \(4 \in A\) and \(4 \in B\text{,}\) so it will be in the intersection.

Interpreting our results, we have

\begin{equation*} A \cap B = \{-1\} \cup (3,4]\text{.} \end{equation*}
