When a formula or function is a quotient of two polynomials (a rational function), say \(f(x) = p(x)/q(x)\text{,}\) and we are evaluating a limit

\begin{equation*} \lim_{x \to a} f(x) = \lim_{x \to a} \frac{p(x)}{q(x)}\text{,} \end{equation*}

then we can use the following process.

  • Check if it is already continuous. If \(p(a)/q(a)\) is defined (any time \(q(a) \ne 0\)), then

    \begin{equation*} \lim_{x \to a} f(x) = \frac{p(a)}{q(a)}\text{.} \end{equation*}

  • If \(p(a) = 0\) and \(q(a) = 0\) (limit form: \(0/0\)), then \((x-a)\) is a factor of both \(p(x)\) and \(q(x)\text{.}\) Factor both polynomials, cancel any common factors, and then start over using the new formula (leave factored). Use technology if the formulas are hard to factor by hand.

  • If \(p(a) = L \ne 0\) and \(q(a) = 0\) (limit form \(L/0\)), then \(f(x)\) has an infinite discontinuity corresponding to a vertical asymptote. The one-sided limits must be infinite; we just need to determine whether they are \(\pm \infty\text{.}\)

    • Use the factored formulas to do sign analysis. Be sure to mark all zeros of \(p(x)\) (roots) and of \(q(x)\) (discontinuities).

    • Only the intervals immediately to the left and right of \(x=a\) need to be checked. Test a single value in each interval. If the sign is positive, that side limit will be \(+\infty\text{.}\) If the sign is negative, that side limit will be \(-\infty\text{.}\)

    • The results should match the vertical asymptote appearing on a graph.
