Next, a critical understanding requires recognizing that any non-empty interval has infinitely many different numbers inside. Learners often want to think only in terms of integer (whole-number) values, and this interferes with this understanding. For example, some learners want to think of \(0 \lt x \lt 5\) as only including integers \(\{1,2,3,4\}\) or think of \(0 \lt x \lt 1\) as being empty (since there aren't any integers between 0 and 1). Be sure to think about fractional values. For example, \(0.1 \in (0,1)\text{.}\)
