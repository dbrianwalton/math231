Solution 5.5.2.1.

Our function \(f(x)\) is continuous. Because the interval does not include \(x=4\text{,}\) we are not guaranteed to find a global maximum and minimum. Our strategy will be to find any local extremes inside the interval, and then compare the function values at those points with the end point at \(x=-3\) (which is included) and the limit of \(f(x)\) at \(x=4\) (which is not included).

We start by finding the derivative.

\begin{align*} f(x) &= x^3-12x \\ f'(x) &= 3x^2-12 \end{align*}

We find critical points by solving \(f'(x)=0\text{.}\)

\begin{gather*} 3x^2-12 = 0\\ 3(x^2-4) = 0\\ 3(x+2)(x-2) = 0 \end{gather*}

So we find there are critical points at \(x=-2\) and \(x=2\text{.}\) Because \(f''(x)=6x\text{,}\) we can use \(f''(-2)=-12\) and \(f''(2)=12\) to determine that \(f(x)\) has a local maximum at \(x=-2\) and a local minimum at \(x=2\text{.}\)

We will not know if they are global extremes until we compare the value of the function with the end points.

\begin{align*} f(-2) &= (-2)^3-12(-2) = -8+24=16\\ f(2) &= (2)^3-12(2) = 8-24=-16\\ f(-3) &= (-3)^3-12(-3) = -27+36=-1\\ \lim_{x \to 4^-} f(x) &= (4)^3-12(4) = 64-48=16 \end{align*}

The largest value occured at \(x=-2\text{,}\) so \(f(x)\) has an absolute maximum value of 16 at \(x=-2\text{.}\) The absolute minimum value of -16 was attained at \(x=2\text{.}\) We used a limit at \(x=4\) to emphasize that this value is not attained by the function at \(x=4\) since that end point was not included. If our interval had extended any further to the right and still not included the end point, then the limit at the right end point would have been the largest of all the values and we would have had a function where there was not absolute maximum value.
