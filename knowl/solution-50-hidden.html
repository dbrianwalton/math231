If you previously learned calculus, you might be tempted to use derivative rules for the formula that applies with \(x=2\text{.}\) However, because this is a piecewise formula that uses \(x=2\) as a boundary between two rules, this is not appropriate. Instead, we need to compare the left- and right-derivatives.

When using the definition of the derivative \(g'(a)\) on a piecewise function, you need to be careful that the value \(g(a)\) uses the function value at the point of interest and not the formula from the sides. But when calculating \(g(a+h)\text{,}\) we need to consider the different cases for \(h \gt 0\) (used for the right-derivative) and for \(h \lt 0\) (used for the left-derivative).

In this example, the function \(g(x)\) at \(x=2\) uses the formula \(x^2-2x\) to give \(g(2)=2^2-2(2) = 0\text{.}\) The right-derivative uses a secant line based on \(g(2+h)\) with \(h \gt 0\) so that we use the formula for \(x \gt 2\text{,}\)

\begin{equation*} g(2+h) = 2(2+h) = 4+2h\text{.} \end{equation*}

The left-derivative uses a secant line based on \(g(2+h)\) with \(h \lt 0\) to use \(g(x)=x^2-2x\)

\begin{equation*} g(2+h) = (2+h)^2-2(2+h) = 4+4h+h^2-4-2h = 2h+h^2\text{.} \end{equation*}

Once we recognize which formulas and values to use, we calculate the slope of the secant lines and then find the limits.

We start with the right-derivative.

\begin{align*} \frac{dg}{dx}\Big|_{2^+} &= \lim_{h \to 0^+} \frac{g(2+h)-g(2)}{h}\\ &= \lim_{h \to 0^+} \frac{(4+2h) - (0)}{h}\\ &= \lim_{h \to 0^+} \frac{4+2h}{h} \end{align*}

Because \(h\) is not a factor in the numerator, the limit will have \(4+2h \to 4\) and \(h \to 0\text{,}\) giving a limit of the form \(4/0\text{.}\) Our experience with limits tells us that this will be an infinite limit. We need to test the sign. For small positive \(h\text{,}\) the numerator will be close to 4 and the denominator will be a small positive number, so that the quotient will be positive. This means:

\begin{equation*} \frac{dg}{dx}\Big|_{2^+} = +\infty\text{.} \end{equation*}

In the graphical interpretation, this would mean that the graph has a jump discontinuity that goes up on the right so that the limit of the secant line is a vertical line.

Next, we compute the left-derivative.

\begin{align*} \frac{dg}{dx}\Big|_{2^-} &= \lim_{h \to 0^-} \frac{g(2+h)-g(2)}{h}\\ &= \lim_{h \to 0^-} \frac{(2h+h^2) - (0)}{h}\\ &= \lim_{h \to 0^-} \frac{h(2+h)}{h}\\ &= \lim_{h \to 0^-} 2+h \end{align*}

This time the quotient had a common factor of \(h\) that could cancel from the denominator. The new formula is continuous so that the limit can be calculated using continuity to give

\begin{align*} \frac{dg}{dx}\Big|_{2^-} &= \lim_{h \to 0^-} 2+h\\ &= 2+0 = 2 \end{align*}

The function has a left-tangent line with a slope of 2.

We finish by comparing the values. The left- and right-derivatives do not have a common value, so the value \(g'(2)\) does not exist. The function \(g(x)\) is not differentiable at \(x=2\text{.}\)
