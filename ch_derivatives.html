Chapter 4 Derivatives

Derivatives allow us to describe how two variables relate to one another through the idea of rate of change. Consider a linear relation, \(y = mx+b\text{.}\) The value of \(y\) depends on the value of \(x\text{.}\) If we change the value of \(x\text{,}\) the value of \(y\) will necessarily change. The rate of change describe how fast \(y\) changes relative to changes in \(x\text{,}\) and for a linear relation is the slope \(m\text{.}\)

That is, suppose we use a particular example, \(y=3x-2\) and start with \(x=3\text{.}\) The relation requires \(y=3(3)-2 = 7\text{,}\) so that we are starting at a point \((x_0,y_0)=(3,7)\text{.}\) Now, if \(x\) increases by the value \(\Delta x = 0.5\) to the new value \(x = x_0 + \Delta x = 3.5\text{,}\) the value of \(y\) also changes to a new value \(y = y_0 + \Delta y = 3(3.5)-2 = 8.5\text{.}\) Knowing \(y_0 = 7\text{,}\) we find \(\Delta y = 8.5 - 7 = 1.5\text{.}\) The slope give the ratio \(\Delta y / \Delta x = 1.5 / 0.5 = 3\text{.}\) The idea of rate of change is literally a ratio of changes.

Linear relations are linear precisely because the rate of change is constantâ€”the rate of change does not depend on which point is considered. Most relations have varying rates of change (leading to concavity). A derivative provides the mathematical tool to describe rate of change as a varying quantity, and it is defined in terms of a limit of average rates of change.

The learning objectives related to derivatives focus on understanding the relation between the derivative and tangent lines, the attribute of differentiability, working with the definition of the derivative, and applying the rules of differentiation to find derivatives.
