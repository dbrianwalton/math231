Section 2.2 Constructing Functions

One of the most important skills necessary for success in calculations relating to calculus is to understand how functions or expressions written as formulas are constructed out of simpler building blocks. Once we understand the rules of calculus for the individual building blocks, calculus provides rules for how to deal with more complex constructions.

The learning outcomes represented in this section focus on developing this skill.

  • Find the domain of a function. The process of finding a domain given a formula is based on recognizing some steps in construction, specifically relating to the presence of division or square (even) roots. Once these steps are identified, the process reduces to solving equations or inequalities and interpreting the resulting sets.

  • Recognize composition of functions. One of the basic methods of creating a complex function from simple elements, the composition of functions plays a central role in understanding inverse functions and is essential in understanding and using the chain rule and other rules involving substitution.

  • Deconstruct a function. All of the rules of calculus (limits, derivatives, integrals) are based on the structure of a formula. Recognizing how a single formula is constructed from smaller units will be essential for later success.

Subsection 2.2.1 Natural Domains

Learning how to find the domain of an expression involves a subset of the skills relating to how expressions are constructed. While it might be preferable to frame this in terms of deconstruction and composition discussed below, this discussion will try to focus specifically on two operations that impact the domain—division and even roots.

The natural domain of an expression (a function) involving one or more variables is the set of values that can be assigned to the variables such that the expression has a well-defined resulting value. When a function is constructed so that it only involves basic arithmetic operations (addition, subtraction, multiplication, and division) and powers, the only possible problem is division by zero. If a formula has an even root (or powers \(p/q\) with \(q\) an even integer), the resulting root of a negative value is not a real number. Apart from those issues, the formula will create a real-valued answer.

To find the domain of an algebraic function, we need to identify the risky operations (division and even roots) and create inequalities that will ensure they are safe. The set of values that avoid all undefined circumstances will be the domain. Thus, the first skill is to identify where the operations of division occur and the sub-expression that might create a problem.

  1. Is there division? What are the subexpressions representing the numerator and denominator? Values making the denominator zero need to be removed from the domain.

  2. Is there an even root or a power of the form \(p/q\) with \(q\) an even integer? What is the subexpression of the formula inside the root? (This is an input of a composition.) Values making the input expression negative need to be removed from the domain.

The second skill is to apply our previously reviewed principles of solving equations and inequalities to find the sets described above.

Example 2.2.1.

Determine the domain of \(\displaystyle f(x) = 5x + \frac{2x+3}{x^2-4}\text{.}\)

Solution.

When we look at the formula for \(f(x)\text{,}\) we see that the construction involves division,

\begin{equation*} 5x + \boxed{\frac{2x+3}{x^2-4}}\text{.} \end{equation*}

We need to recognize the denominator in that step. The formula \(2x+3\) is divided by \(x^2-4\text{.}\) The domain requires that the denominator \(x^2-4\) is nonzero. We create an equation to see what values create problems by relating the denominator to zero,

\begin{equation*} x^2-4 = 0\text{.} \end{equation*}

We solve the equation using factoring, since a product can only equal zero if one of the factors equals zero.

\begin{gather*} x^2-4 = 0\\ (x+2)(x-2) = 0 \\ x+2 = 0 \quad \text{or} \quad x-2=0\\ x=-2 \quad \text{or} \quad x=2 \end{gather*}

This means \(f(x)\) is defined for all inputs except \(x=-2\) or \(x=2\text{.}\)

To describe the domain using intervals, we think of the real number line and remove \(x=\pm 2\text{.}\) A graphical representation of the set using a number line is shown below. Intervals are read from the line left-to-right. It starts at \(-\infty\) and continues until \(-2\text{,}\) then goes from \(-2\) to \(2\text{,}\) and finally goes from \(2\) until \(+\infty\text{.}\) We write

\begin{equation*} D = (-\infty,-2) \cup (-2, 2) \cup (2,+\infty)\text{.} \end{equation*}

Example 2.2.2.

Find the domain of the function \(\displaystyle g(x)=2x^2 - \sqrt{x^2-x-6}\text{.}\)

Solution.

The expression defining \(g(x)\) includes an even root. To find the domain, we need to identify what expression is the input to that root. The root will only have a real value when the input expression is non-negative. We identify the expression inside the root as \(x^2-x-6\text{.}\) The requirement is that this expression is non-negative. Any values of \(x\) that make \(x^2-x-6 \lt 0\) can not be in the domain.

To find the domain, our task sign analysis of \(x^2-x-6\text{.}\) First, we find possible sign-changing points. The expression is always defined (no discontinuities) so we just solve for zeros \(x^2-x-6=0\text{.}\)

\begin{gather*} x^2-x-6=0 \\ (x-3)(x+2)=0\\ x-3=0 \quad \text{or} \quad x+2=0\\ x=3 \quad \text{or} \quad x=-2 \end{gather*}

Once we know these points, we test each interval to find the sign of the formula \((x-3)(x+2)\text{.}\)

Now we interpret our results. We were finding the domain of \(g(x)=2x^2 - \sqrt{x^2-x-6}\text{,}\) which requires \(x^2-x-6 \ge 0\text{.}\) Based on our summary, we need to find all values which result in the expression \(x^2-x-6\) having a positive sign or zero. Any values the cause the expression to have a negative sign are not in the domain. So our solution is the set

\begin{equation*} D = (-\infty,-2] \cup [3,\infty)\text{.} \end{equation*}

More complicated problems require dealing with multiple issues.

Example 2.2.4. Learning Objective F-4.

Find the domain of \(\displaystyle f(x) = \frac{\sqrt{\frac{x-3}{x+1}}+2}{x^2+3x-4}\text{.}\)

Solution.

Our task is to find the domain of the function. We look at the formula for the potential risk factors: division and even roots. There are two steps involving division and a square root. For each division, we need to look at what is the denominator and ensure it is not zero. For the square root, we need to identify its input and ensure it is not negative.

One division has a denominator of \(x+1\text{.}\) We can't divide by zero, so we identify where a problem would occur by solving \(x+1 = 0\text{.}\) The method of isolation quickly results in \(x=-1\) being excluded from the domain.

The second division has a denominator of \(x^2+3x-4\text{.}\) After factoring, we see this is rewritten \((x+4)(x-1)\text{.}\) A problem occurs if the denominator is zero, so we need to solve \((x+4)(x-1)=0\text{.}\) The two solutions at \(x=1\) and \(x=-4\) are also excluded from the domain.

For the square root, we need to identify the input expression. What formula is inside the root? We need to require \(\displaystyle \frac{x-3}{x+1} \ge 0\) and so will remove any values causing \(\displaystyle \frac{x-3}{x+1} \lt 0\text{.}\) This is already factored, so sign analysis will be quick.

To interpret the results of sign analysis, recall that we were exploring the domain of the square root portion of the formula. That portion is defined where sign analysis reveals values that are positive or zero, which occur for \(x \in (-\infty, -1) \cup [3, \infty)\text{.}\) The interval \([-1,3)\) must all be removed from the domain—removing -1 because it causes division by zero and removing all values \(-1 \lt x \lt 3\) because they put a negative value inside the square root.

Once we have considered the restrictions for each risk factor (division and even roots), the domain of the function overall requires that all restrictions are satisfied. If we imagine a number line of potential domain values, we remove \(x=-1\) for the first division, \(x=-4\) and \(x=1\) for the second division, and all values \(-1 \lt x \lt 3\) for the square root. Notice that \(x=1\) is not allowed for two reasons. The domain is the set of remaining values,

\begin{equation*} D = (-\infty, -4) \cup (-4, -1) \cup [3,\infty)\text{.} \end{equation*}

Subsection 2.2.2 Function Composition

A composition is when we use the output value of one function as the input value of another function. That is, if \(f\) and \(g\) are functions, then \(f\big(g(x)\big)\) is the composition of \(f\) with \(g\text{.}\) Overall, this constructs a new function \(f \circ g\text{,}\)

\begin{equation*} f \circ g(x) = f\big(g(x)\big)\text{.} \end{equation*}

We are using composition every time we evaluate a function using substitution with a formula as its input. When we identified a formula that was inside of a square root, we were identifying a composition. The skill we want to focus on in this subsection is in recognizing a sequence of operations as a composition of two functions.

The idea behind a composition \(f \circ g(x)\) is in thinking of the overall expression or function as a sequence or chain of two maps (functions) that accomplish the overall sequence of operations. We take the variable and perform some of the operations to find \(g(x)\text{.}\) The rest of the calculation then consists of operations that are applied to that intermediate result. It is often the case that we think of this intermediate result as a dependent variable, say \(u=g(x)\text{,}\) and the final result is a function of \(u\text{,}\) \(y=f(u)\text{.}\)

Example 2.2.6.

A drop of water sitting on a flat surface forms the shape of half a sphere. Suppose the radius of the drop, which starts at 3 mm, is decreasing through evaporation at a rate of 0.2 mm/min. The volume of water is a function of the radius of the drop, and the radius of the drop is itself a function of time. This chain of relations allows us to write the volume as a function of time using composition.

We will represent this model using a chain of relations involving time \(t\) (in minutes), radius \(r\) (in millimeters), and volume \(V\) (in cubic millimeters). First, we will find a function representing the map from time to radius, \(t \mapsto r\text{.}\) Then, we will find a function representing the map from radius to volume, \(r \mapsto V\text{.}\) The overall relation between time and volume is found by composition of this chain,

\begin{equation*} t \mapsto r \mapsto V\text{.} \end{equation*}

In the first step, we need the function \(t \mapsto r\) which is modeled by a linear function because of the constant rate of change. That is, we know the slope (negative because decreasing) and an initial point:

\begin{equation*} \frac{\Delta r}{\Delta t} = -0.2, \quad (t,r)=(0,3)\text{.} \end{equation*}

We have \(r=-0.2t+3\text{.}\) This is the first step of the chain, \(t\mapsto r=-0.2t+3\text{,}\) which we could assign to a function \(g(t) = -0.2t+3\text{.}\)

The second step of the chain is based on the relation between radius and volume. Because the drop of water forms just half a sphere, the volume formula is given by

\begin{equation*} V = \frac{1}{2} \cdot \frac{4}{3}\pi r^3 = \frac{2}{3} \pi r^3\text{.} \end{equation*}

This is already in the form of a function, so we have the second step of the chain, \(r \mapsto V = \frac{2}{3} \pi r^3\text{,}\) which we could assign to another function \(f(r) = \frac{2}{3} \pi r^3\text{.}\)

We know that the radius is a function of time \(r=g(t) = -0.2t+3\text{,}\) and volume is a function of radius, \(V=f(r) = \frac{2}{3} \pi r^3\text{.}\) The relation \(t \mapsto V\) is a composition of this chain, \(V = f \circ g(t)\text{.}\) Using substitution, we have

\begin{equation*} V = f(g(t)) = f(-0.2t+3) = \frac{2}{3} \pi (-0.2t+3)^3\text{.} \end{equation*}

The focus of the measured learning objective is in being able to think through this in reverse. Recognizing a composition involves looking for formula patterns that could have come from this type of substitution. Try to find a sub-expression that involves the variable so that the overall expression consists only of further operations on that formula (and no others variables). That sub-expression would be the intermediate variable in the chain, often generically assigned a name \(u\text{.}\) The outer function is identified by replacing that expression by the intermediate variable. There can often be multiple choices for what to call the intermediate variable.

Example 2.2.7. Learning Objective F-5.

Identify non-trivial functions \(f(x)\) and \(g(x)\) (meaning don't choose \(f(x)=x\) or \(g(x)=x\)) so that

\begin{equation*} f(g(x)) = 2(3x+1)^4+6\text{.} \end{equation*}

Show that the substitution in the composition works as required.

Solution.

If we look at the overall expression as a function of \(x\text{,}\) we can think about the operations happening in some sequence. The table below tries to highlight this order:

\begin{align*} 2(3 \boxed{x} + 1)^4 + 6 & & \text{find the variable}\\ 2(\boxed{3 x} + 1)^4 + 6 & & \text{multiply by 3}\\ 2(\boxed{3 x + 1})^4 + 6 & & \text{add 1}\\ 2\boxed{(3 x + 1)^4} + 6 & & \text{apply exponent 4}\\ \boxed{2(3 x + 1)^4} + 6 & & \text{multiply by 2}\\ \boxed{2(3 x + 1)^4 + 6} & & \text{add 6} \end{align*}

In principle, any intermediate step in this example could be used to identify our intermediate variable in the chain. The inner function will capture all of the operations up to that step; the outer function will capture all of the remaining operations.

The most important place to recognize composition is at the step that is not simple arithmetic—at the use of an exponent. The formula \(3x+1\) captures all of the operations preceding the exponent. If we replace that formula by an intermediate variable \(u=3x+1\text{,}\) the overall formula becomes

\begin{equation*} 2u^4+6\text{.} \end{equation*}

(The boxed formula is replaced by \(u\text{.}\)) This formula summarizes all of the remaining operations, starting with the exponent. Using this choice, the inner function (first map) is

\begin{equation*} g(x) = 3x+1\text{,} \end{equation*}

and the outer function (second map) is

\begin{equation*} f(u) = 2u^4+6\text{.} \end{equation*}

Mathematically, when the variable used as input is treated abstractly, we could also write

\begin{equation*} f(x) = 2x^4 + 6\text{.} \end{equation*}

The composition works because substitution gives

\begin{equation*} f \circ g(x) = f(\boxed{g(x)}) = 2 (\boxed{3x+1})^4 + 6\text{.} \end{equation*}

However, other composition choices could have been made. If we chose to do all steps including the exponent as the first step, we would have instead found \(u = g(x) = (3x+1)^4\) and \(y = f(u) = 2u + 6\) or \(f(x)= 2x+6\text{.}\) With this choice, composition gives the same overall result:

\begin{equation*} f \circ g(x) = f(\boxed{g(x)}) = 2 \boxed{(3x+1)^4} + 6\text{.} \end{equation*}

Subsection 2.2.3 Expression Deconstruction

We are now ready to look at the full picture of how all algebraic functions and expressions are constructed. Every formula consists of a collection of operations that take simpler expressions to create more complex expressions. The simplest expressions would be numbers and variables (which are symbols representing numbers). All of the operations can be summarized as arithmetic or composition.

Arithmetic operations are the four basic operations of addition, subtraction, multiplication, and division. These operations are called binary operations because they use two expressions as inputs to create a new complex expression. The associative and commutative properties of addition and multiplication make it so that we could think of addition or multiplication of more than two values at a time, but then the way we talk about things gets more complicated. Subtraction and division do not conveniently allow for more than two values. So we will always think of all of the arithmetic operations as being binary.

Every other operation is represented by a function. Is something raised to a power? That's a function. Do we use an absolute value? That's a function. For each new operation, we have additional functions. When we use the operation to another formula, we have composition of functions.

To understand the rules of calculus, we must be able to look at a formula and understand how that formula is constructed and the order in which the operations are used. This is where we really need the order of operations. Some teachers use the acronym PEMDAS for (1) Parentheses, (2) Exponents, (3) Multiplication, (4) Division, (5) Addition, and (6) Subtraction. In reality, multiplication and division are the same thing, so they happen at the same priority. Similarly, addition and subtraction are really the same thing and have the same priority. When there are multiple items with the same priority, the understanding is that they are completed from left to right.

We need a slightly broader view in order to account for other operations. Because an exponent can be interpreted as a function, we put all functions at that same priority order. We could think of the E in the acronym as meaning every function.

Parentheses are not a separate type of operation. They are listed first because they are used to change the priority to complete everything inside. For example, if we want to be sure that we add two values before a multiplication, we write \((a+b) \cdot c\) because without parentheses \(a+b\cdot c\) would be understood to complete multiplication \(b \cdot c\) before we add \(a\text{.}\) Functions use parentheses to emphasize that everything is inside needs to be evaluated before using the function's operation.

When you look at a formula, whether it is an expression standing alone or a function, you should be able to quickly identify the last operation so that you can deconstruct the formula into an operation involving two simpler expressions or functions. You could think of this as looking for the lowest order of operation in the expression. If there are not two clear parts, then you are most likely facing a composition of an operation with some input formula.

We will look at some examples of formula construction.

Example 2.2.8.

How is \(\displaystyle 3x^4(2x+1)\) made from two simpler expressions?

Solution.

This formula involves exponents, multiplication, and addition. Addition would normally be last, but it is occuring inside parentheses which raises the priority. So the last operation is a multiplication, and to be more precise is the right-most multiplication:

\begin{equation*} \boxed{3x^4} \cdot \boxed{(2x+1)}\text{.} \end{equation*}

We say the expression is a product of the simpler expressions \(3x^4\) and \(2x+1\text{.}\)

Example 2.2.9.

How is \(\displaystyle 2x^3 + \frac{2x-1}{x^2+5}\) made from two simpler expressions?

Solution.

This formula involves exponents, multiplication, division, addition, and subtraction. One aspect of quotients (division) written in fraction form is that the numerator and denominator don't show parentheses but it is as if they were. The formula might also have been written

\begin{equation*} 2x^3 + (2x-1)/(x^2+5)\text{.} \end{equation*}

The product \(2x^3\) and the quotient \((2x-1)/(x^2+5)\) both have higher priority than the addition that joins them. So the last operation is the addition:

\begin{equation*} \boxed{2x^3} + \boxed{\frac{2x-1}{x^2+5}}\text{.} \end{equation*}

We say the expression is a sum of the simpler expressions \(2x^3\) and \(\displaystyle \frac{2x-1}{x^2+5}\text{.}\)

Examples of compositions might include:

  • \(\sqrt{x^2+3x}\text{:}\) The last operation is the square root acting on a formula. So this is a composition \(f \circ g(x)\) of outer function \(f(x)=\sqrt{x}\) with inner function \(g(x)=x^2+3x\text{.}\)

  • \((\sqrt{x}+3)^4\text{:}\) The last operation is the exponent acting on a formula. So this is also a composition \(f \circ g(x)\text{,}\) but with an outer function \(f(x)=x^4\) and an inner function \(g(x)=\sqrt{x}+3\text{.}\)

The learning outcome focuses on identifying the structure at multiple layers. Once you have identified the last operation and the two simpler expressions or functions involved, each of those expressions are themselves either elementary or can be further deconstructed. Repeat the process on the individual parts.

Example 2.2.10. Learning Objective F-6.

Deconstruct the function \(f(x)=\sqrt{x^2+3x} - 3x^4(2x+1)\text{.}\)

Solution.

The last operation is subtraction:

\begin{equation*} f(x) = \boxed{\sqrt{x^2+3x}} - \boxed{3x^4(2x+1)}\text{.} \end{equation*}

We can label the simpler expressions with subscripts rather than creating new function names:

\begin{align*} f(x) &= f_1(x) - f_2(x) & \text{difference}\\ f_1(x) &= \sqrt{x^2+3x}\\ f_2(x) &= 3x^4(2x+1) \end{align*}

We need to deconstruct \(f_1(x)\) and \(f_2(x)\text{.}\) The last operation for \(f_1(x)\) is the square root with a formula as its input. So we have a composition:

\begin{align*} f_1(x) &= f_3 \circ f_4(x) & \text{composition}\\ f_3(x) &= \sqrt{x}\\ f_4(x) &= x^2+3x \end{align*}

The last operation of \(f_2(x)\) is multiplication:

\begin{align*} f_2(x) &= f_5(x) \cdot f_6(x) & \text{product}\\ f_5(x) &= 3x^4\\ f_6(x) &= 2x+1 \end{align*}

We would then continue for each of the functions. The function \(f_3(x) = \sqrt{x}\) is a simple function acting on \(x\text{,}\) so this has no further deconstruction. The function \(f_4(x) = x^2 + 3x\) is a sum:

\begin{align*} f_4(x) &= f_7(x) + f_8(x) & \text{sum}\\ f_7(x) &= x^2\\ f_8(x) &= 3x \end{align*}

Note that \(f_7(x)\) is another simple function and requires no further deconstruction, while \(f_8(x)\) is a product of elementary expressions \(3\) and \(x\text{.}\) The function \(f_5(x) = 3x^4\) is a product of \(f_9(x)=3\) (elementary expression) and \(f_{10}(x)=x^4\) (elementary function). And the function \(f_6(x)=2x+1\) is a sum of \(f_{11}(x)=2x\) (a product) and \(f_{12}(x)=1\) (elementary expression).

One way to visualize the construction of an expression is with a graphical structure called a tree. The original expression or function is the root of the tree and it is associated with the last operation. Each of the two component expressions that are combined become branches of the tree. We keep branching each branch until we end up at a number, a variable, or an elementary function.
