Section 3.5 Reasoning About Limits

Recall that mathematics as a discipline seeks to understand patterns. Mathematical definitions provide formal statements that codify the idea of a pattern, and deductive reasoning looks for consequences that are provable based on the attributes an object might possess. A function having a limit is one such pattern; a function being continuous is another. Mathematicians have created a definition for a function limit that captures the intuitive ideas about limits we have used relating to graphs and values being approximated from one side or the other. Simple functions can be showed to have limits directly using the definition. Theorems applying the definition of a limit to more complex formulas constructed from simpler formulas that have limits allow us to apply our ideas about continuity to these complex formulas. Together, these theorems justify our ability to compute limits of functions without needing to verify the definition each time.

Definition 3.5.1. Limit of a Function.

Let \(f\) be a function, and \(a\) and \(L\) represent real numbers. “The limit of \(f(x)\) as \(x \to a\) is \(L\)”, \(\displaystyle \lim_{x \to a} f(x) = L\text{,}\) means: For all \(\epsilon \gt 0\) (a required approximation tolerance for \(y\)), there exists \(\delta \gt 0\) (a corresponding acceptable variation for \(x\)) so that

\begin{equation*} |f(x)-L| \lt \epsilon \quad \text{whenever} \quad 0 \lt |x-a| \lt \delta \text{.} \end{equation*}

Learning goals related to deductive reasoning about limits include:

  • Given a limit and a required tolerance \(\epsilon\text{,}\) find a value for \(\delta\) that will guarantee the function approximation stated in the limit definition.

  • Apply limit theorems to justify limit calculations on complex algebraic formulas in terms of elementary limits.

Subsection 3.5.1 Understanding Limits

The definition of a limit of a function attempts to formalize the idea that values to the left and the right of \(x=a\) result in values of \(f(x)\) that are close to \(L\) through the use of inequalities and two tolerance values, traditionally called epsilon \(\epsilon\) and delta \(\delta\text{.}\) We want the meaning to be something like “if you can get close enough to \(x=a\text{,}\) then \(f(x)\) will be close to its limit \(L\)”.

The value of epsilon \(\epsilon\) describes a tolerance (maximum acceptable error) between \(f(x)\) and \(L\text{,}\) while the value of delta \(\delta\) describes how close the value of \(x\) needs to be to \(a\) in order to guarantee we are within this tolerance. The value of \(\epsilon\) is chosen first. Given the value of \(\epsilon\text{,}\) the value of \(\delta\) is then chosen and typically depends on \(\epsilon\text{.}\) The inequalities establish the connection between \(\epsilon\) and \(\delta\text{.}\)

How do we find the value of \(\delta\text{?}\)

  • Identify the value of the limit \(L\) and the given value \(\epsilon \gt 0\text{.}\)

  • Calculate thresholds \(y=L-\epsilon\) and \(y=L+\epsilon\text{.}\) The inequality \(|f(x)-L| \lt \epsilon\) is equivalent to saying \(L-\epsilon \lt f(x) \lt L+\epsilon\text{.}\)

  • Find the points where \(f(x)\) crosses the thresholds by solving equations. Graphically, this will be where \(y=f(x)\) intersects the threshold lines or where there are jumps in the graph.

  • Identify an interval \((x_L,x_R)\) surrounding \(x=a\) (not counting at \(x=a\)) that satisfies \(|f(x)-L| \lt \epsilon\text{.}\)

  • Measure the distances from \(x=a\) to each end point:

    \begin{gather*} \delta_{-} = |x_L - a|\\ \delta_{+} = |x_R - a| \end{gather*}

    Choose \(\delta = \min(x_L, x_R)\text{.}\)

  • For all \(x\) in \((a-\delta, a) \cup (a,a+\delta)\text{,}\) we are guaranteed that \(|f(x)-L| \lt \epsilon\text{.}\)

The process of finding the end points \(x_L\) and \(x_R\) will vary based on the particular formula, but otherwise this process is the same for every example. When the limit exists, a value for \(\delta\) will always be found. This is why the definition says that \(\delta\) exists for every \(\epsilon \gt 0\text{.}\) So if there is a value of \(\epsilon \gt 0\) for which no \(\delta\) can be found, the value of \(L\) must not be a limit.

Example 3.5.2.

Consider \(\displaystyle \lim_{x \to 0} \frac{3^x-1}{x}\text{.}\) Clearly, this formula does not allow \(x=0\) (division). And it doesn't have any obvious simplifications that might cancel the \(x\) in the denominator because the numerator's formula is not algebraic. One strategy to think about such limits is to try values of \(x\) that get closer and closer to \(x=0\) and see if we can decide what value might be the limit.

Define \(\displaystyle f(x) = \frac{3^x-1}{x}\) and look at the values for a sequence of values of \(x\text{:}\)

\(x\) \(f(x)\)
0.1 1.16123
0.01 1.10467
0.001 1.09922
What might we predict based on this small sample of values? The values of \(f(x)\) are decreasing. We might want to assume that the limit goes to something nice that is below 1.099, and a graph might suggest trying \(f(x) \to 1.09\text{.}\)

Let us now look at what the definition of a limit would require for us to say \(\displaystyle \lim_{x \to 0} f(x) = 1.09\text{.}\) For any value of \(\epsilon \gt 0\) that we are given, it must be possible to find \(\delta \gt 0\) so that whenever \(0 \lt |x-0| \lt \delta\text{,}\) it follows that \(|f(x)-1.09| \lt \epsilon\text{.}\)

Visualize this in terms of a graph. We will plot \(y=f(x)\) and the two threshold lines \(y=L-\epsilon\) and \(y=L+\epsilon\text{.}\) We want to identify the set of \(x\)-values so that \(L-\epsilon \lt f(x) \lt L+\epsilon\text{.}\) This set needs to include an interval surrounding but excluding \(x=0\) (called a punctured neighborhood). Looking at Figure 3.5.3, everything looks good when we use \(\epsilon = 0.1\) (left image). If we zoom in further and use \(\epsilon = 0.01\) (right image), the interval barely extends to the right of \(x=0\text{.}\) It is clear that if we choose a smaller value of \(\epsilon\text{,}\) the set of points between the thresholds will be entirely to the left of \(x=0\text{.}\) This is because the limit is not 1.09 but some value slightly greater.

The actual limit for this question is an irrational number whose decimal representation begins \(L=1.098612\ldots\text{.}\)

A limit requires that no matter how small \(\epsilon\) is chosen, the set of values for \(x\) so that \(|f(x)-L| \lt \epsilon\) must extend slightly to the left of \(x=a\) and slightly to the right of \(x=a\text{.}\) For many limits, a finite decimal representation will not be possible. Once \(\epsilon\) is chosen to be smaller than the place value of the last digit chosen, the representation is likely to break down. In that case, we would say we know an approximation to the limit within that value of \(\epsilon\text{.}\)

For our class, you are expected to be able to construct an appropriate graphical representation of the limit thresholds, find the set of \(x\)-values resulting in \(|f(x)-L| \lt \epsilon\text{,}\) and determine the value of \(\delta\) that would have been used in the definition for that choice of \(\epsilon>\text{.}\) The formulas appearing in assessment questions will allow you to solve for the end-points of the interval exactly.

Example 3.5.4. Learning Objective L-8.

We know that \(\lim_{x \to 4} \sqrt{x} = 2\text{.}\)

For \(\epsilon = 0.1\text{,}\) find \(\delta\) so that \(|f(x)-L| \lt \epsilon\) whenever \(0 \lt |x-4| \lt \delta\text{,}\) for the appropriate \(f(x)\) and \(L\) for this limit. Illustrate the role of \(\epsilon\) and \(\delta\) by including an appropriate graph.

Solution.

The function \(f(x)\) will be the formula in the limit, so \(f(x)=\sqrt{x}\) for this problem. The proposed limit is \(L=2\text{.}\) Starting with a graph may be most helpful.

  • Graph \(y=f(x)\text{.}\)

  • Graph \(y=L-\epsilon=2-0.1=1.9\) and \(y=L+\epsilon=2+0.1=2.1\text{.}\)

  • Find where \(f(x)\) intersects or jumps the thresholds to the left and right of \(x=4\text{.}\)

Our algebraic task is to find these points, so we will be solving the equations \(\sqrt{x} = 1.9\) and \(\sqrt{x} = 2.1\text{.}\) The first equation gives \(x=1.9^2 = 3.61\) and the second gives \(x=2.1^2 = 4.41\text{.}\) The inequality \(|f(x)-2| \lt 0.1\) therefore includes the interval \((3.61, 4.41)\text{,}\) which does surround \(x=4\text{.}\)

To find \(\delta\text{,}\) we want to see how big an interval we get that is symmetric around \(x=4\) and stays inside of this set. The left end point at \(x=3.61\) is a distance \(|3.61-4| = 0.39\) away from \(x=4\text{.}\) The right end point at \(x=4.41\) is a distance \(|4.41-4| =0.41\) away from \(x=4\text{.}\) Consequently, if we make our symmetric interval extend \(\delta = 0.39\) away from \(x=4\text{,}\) it will stay inside of both sides of the interval. If we instead chose \(\delta = 0.41\text{,}\) the interval will be inside the set on the right, but will go too far on the left.

We always need to choose the smaller distance, which in this case will be \(\delta = 0.39\text{.}\) The symmetric interval is \((4-0.39,4+0.39) = (3.61,4.39)\text{.}\) The value at \(x=4\) is not required to be in the set for the limit definition to be satisfied, but in this case because \(\sqrt{x}\) is defined at \(x=4\) and satisfies the inequality, it can be included.

Subsection 3.5.2 Limit Theorems

For a typical question, relying on the definition of a limit to justify that the value of a limit is valid is much too complex. Theorems provide a connection between calculating limits using various rules and justifying those limits in terms of the definition. Limit theorems start by establishing certain elementary formulas whose limits can be directly justified by the definition. Additional theorems then establish rules about limits based on arithmetic combining those formulas.

Elementary Limits.

Functions defined by one of the following elementary forms have limits directly justified by the definition of a limit.

  • Constant Functions: \(f(x)=c\) for any real number \(c\text{,}\)

    \begin{equation*} \lim_{x \to a} c = c\text{.} \end{equation*}

  • Identity Function: \(f(x)=x\text{,}\)

    \begin{equation*} \lim_{x \to a} x = a\text{.} \end{equation*}

  • Linear Functions: \(f(x)=mx+b\) for any real numbers \(m,b\text{,}\)

    \begin{equation*} \lim_{x \to a} [mx+b] = ma+b\text{.} \end{equation*}

Ultimately, what these limit rules say is that these elementary functions are continuous.

Algebraic functions are those functions defined by combining these elementary functions using arithmetic operations (sum, difference, product, and quotient), including powers (repeated multiplication). Limit rules for functions constructed from simpler parts are based on the idea that if we know the limits of the parts, then we can guarantee the limit of the combination has a value calculated from those simpler limits.

Limit Rules for Combinations.

Suppose that \(f(x)\) and \(g(x)\) are functions with known limits as \(x \to a\text{:}\)

\begin{gather*} \lim_{x \to a} f(x) = L\\ \lim_{x \to a} g(x) = M \end{gather*}

The following combinations of formulas also have limits:

  • Limit of Constant Sum: For any real number \(k\text{,}\) \(\displaystyle \lim_{x \to a} [f(x)+k] = L + k\text{.}\)

  • Limit of Constant Multiple: For any real number \(k\text{,}\) \(\displaystyle \lim_{x \to a} [k \cdot f(x)] = k \cdot L\text{.}\)

  • Limit of Reciprocal (Multiplicative Inverse): If \(M \ne 0\text{,}\) \(\displaystyle \lim_{x \to a} [\frac{1}{g(x)}] = \frac{1}{M}\text{.}\)

  • Limit of Sum: \(\displaystyle \lim_{x \to a} [f(x) + g(x)] = L+M\text{.}\)

  • Limit of Difference: \(\displaystyle \lim_{x \to a} [f(x) - g(x)] = L-M\text{.}\)

  • Limit of Product: \(\displaystyle \lim_{x \to a} [f(x) \cdot g(x)] = L \cdot M\text{.}\)

  • Limit of Quotient: If \(M \ne 0\text{,}\) \(\displaystyle \lim_{x \to a} [\frac{f(x)}{g(x)}] = \frac{L}{M}\text{.}\)

The first three rules that only involve one function are listed separately because using those rules only requires knowing the limit of one function in advance. This simplifies the process of justification in that we don't need to separately make note of the limits of the individual constants involved. Otherwise, to justify a limit using a limit rule, we first show the limits of the parts are separately justified, and then we apply the limit rule to justify the combined limit is justified.

The process of justification of a limit using limit rules starts by thinking through the deconstruction of a formula, as addressed in our earlier learning outcomes in Subsection 2.2.3. Each operation that combines two formulas to create a slightly more complex formula will correspond to using a limit rule in a limit justification. Once we have the deconstruction, the elementary functions will be the leaves of the tree. We start by justifying the limits of these functions using the elementary limit rules. Then we work our way up the tree, combining branches one at a time using limit rules of combination, until we have the limit of interest. We do need beware that we don't try to combine limits involving division by zero.

Example 3.5.6.

Find and justify the limit \(\displaystyle \lim_{x \to 2} 5x^3(x^2+3x)\text{.}\)

Solution.

Justifying the limit requires more than just plugging the value into the formula. That would be evaluating the function, not finding the limit, and their equivalence is a consequence of continuity. In fact, limit justification for this example is really justification that the function is continuous. Instead, we need to look at how the function \(f(x)=5x^3(x^2+3x)\) is constructed from elementary functions and then use limit rules based on that deconstruction.

The elementary building blocks for which direct limits are justified are constant functions, the identity function (\(x\) all alone), and linear functions. Our formula involves powers of \(x\text{.}\) We will need to think of those as repeated multiplication: \(x^2 = x \cdot x\) and \(x^3 = x \cdot x \cdot x = x^2 \cdot x\text{.}\) In principle, we could create an entire decontruction using a tree.

Each leaf on this tree is an elementary formula (constant, identity or linear function), and we can state their limits as being justified by elementary limit rules. For reference later in our justification, each limit that is justified will be numbered. Constants that will be used in a constant multiple rule or constant sum rule are not required to justify limits separately as the limit rule has that built-in. The two elementary limits that are needed are for \(x\) (identity) and \(3x\) (linear).

Step Limit Justification
1 \(\displaystyle \lim_{x \to 2} x = 2\) Limit of Identity
2 \(\displaystyle \lim_{x \to 2} 3x = 3 \cdot 2 = 6\) Limit of Linear Function
Alternatively, we could have used the constant multiple rule since \(3x = 3 \cdot x\text{.}\)

We now start justifying limits of intermediate formulas, working from the leaves upward in the tree. Notice that \(x^2\) appears twice in this tree, but the limit only needs to be justified once. Then we can build \(x^3\) as a product and \(x^2+3x\) as a sum. The remaining steps involved in justification are listed below.

Step Limit Justification
3
\begin{align*} \displaystyle \lim_{x \to 2} x^2 &= \lim_{x \to 2} x \cdot \lim_{x \to 2} x\\ &= 2 \cdot 2 = 4 \end{align*}
Limit of Product
4
\begin{align*} \displaystyle \lim_{x \to 2} x^3 &= \lim_{x \to 2} x \cdot \lim_{x \to 2} x^2\\ &= 2 \cdot 4 = 8 \end{align*}

5
\begin{align*} \displaystyle \lim_{x \to 2} x^2+3x &= \lim_{x \to 2} x^2 + \lim_{x \to 2} 3x\\ &= 8 + 6 = 14 \end{align*}

\begin{align*} \displaystyle \lim_{x \to 2} 5x^3 &= 5 \cdot \lim_{x \to 2} x^3\\ &= 5 \cdot 8 = 40 \end{align*}

Limit of Product

This sure feels like a lot of work. It is because it is a methodical justification of why we are allowed to evaluate an algebraic formula at a value when calculating a limit, which is formally defined through a statement about inequalities. Any formula constructed only involving arithmetic (including powers that can be rewritten as repeated multiplication) can always be justified following this pattern. The steps will vary for different formulas, but the process is the same. This is why basic algebraic formulas are all continuous.

The process also works for limits that are not based on continuous functions, so long as we have rules for the limits that correspond to discontinuities. This is where assessment questions related to the learning outcome will focus.

Example 3.5.8. Learning Objective L-9.

Suppose that we know \(\displaystyle \lim_{x \to 3} \frac{f(x)}{x-3} = 5\text{.}\) Find and justify the limit \(\displaystyle \lim_{x \to 3} \frac{2f(x)(x^2+1)}{x^2-x-6}\text{.}\)

Solution.

Notice that if we use \(x=3\) in the denominator \(x^2-x-6\) we get \(3^2-3-6 = 0\text{.}\) The limit rule for a quotient does not apply when the denominator results in division by zero. This is why we needed to be told separately the value of a limit that had division by zero in the formula.

In order to use limit rules to justify a limit for this formula, we need to rewrite the formula so that we get a product with \(\displaystyle \frac{f(x)}{x-3}\text{.}\) Factoring the denominator, we have \(x^2-x-6 = (x-3)(x+2)\text{.}\) The factor \(x-3\) can be group with the numerator factor \(f(x)\) so that we rewrite our limit in question:

\begin{equation*} \lim_{x \to 3} \frac{2f(x)(x^2+1)}{x^2-x-6} = \lim_{x \to 3} \frac{f(x)}{x-3} \cdot \frac{2(x^2+1)}{x+2}\text{.} \end{equation*}

The factor on the left has a given limit. We will use limits rules with deconstruction to find the limit of the formula on the right. Then we will use the limit of a product to justify a limit of the entire formula.

See if you can deconstruct the formula on the right on your own. You should then be able to follow the steps of limit justification shown below. Notice that multiplication by 2 is a constant multiple and adding the constant 1 to \(x^2\) is a constant sum, allowing us to avoid needing to find limits of the constants on their own.

2 \(\displaystyle \lim_{x \to 3} x = 3\) Limit of Identity
3 \(\displaystyle \lim_{x \to 3} x+2 = 5\) Limit of a Linear Function
4
