Section 3.1 Intuition about Limits and Continuity

When we graph a function defined piecewise, the graph has different segments on different subsets of the domain. Because the domains of functions we are interested in studying involve a continuum of values on the number line, the end-point plays an interesting role. If the end-point of an interval subset of the restricted domain is not included, there are still infinitely many values in the interval that get infinitely close to the end-point that are in the domain. We usually graph that restricted piece of the function with a curve and put an open point on the graph to illustrate that that value is not used, but the graph uses all values to the side that approach that point. This is, in essence, the idea of a limit. A function is continuous the graph to the left and the right of a point both approach the actual value of the function.

This section focuses on applying these ideas to calculate limits of piecewise functions, defined graphically or with piecewise formulas, and then using these limits to provide reasoning for whether the function is or is not continuous.

Subsection 3.1.1 Limit Intuition from Graphs

When a graph is shown with a segment of the graph ending in a point, that point might be either open (hollow) or closed (filled in). A closed end-point means that the point is included in the graph. An open end-point means that the point is not included in the graph but that the adjacent segment of the graph approaches that point.

In Figure 3.1.1, the graph of a function, which we will call \(f\text{,}\) has a break at \(x=-1\text{.}\) There are two points indicated at this break. The point at \((-1,2)\) is open while the point at \((-1,0)\) is closed.

The open point at \((-1,2)\)is not associated with the value of the function at \(x=-1\text{,}\) but is instead associated with the segment of the graph with domain restriction \((-\infty,-1)\text{.}\) It means that the closer values \(x \lt -1\) are to \(x=-1\text{,}\) the closer the values of \(f(x)\) will be to the \(y\)-value of \(y=2\text{.}\) We write a limit to represent this idea:

\begin{equation*} \lim_{\substack{x \rightarrow -1 \\ x \lt -1}} f(x) = 2\text{.} \end{equation*}

Alternatively, below the “lim” operator, we usually just write \(x \to -1^-\text{,}\) where the superscript - indicates we are on the left of \(x=-1\text{,}\) to write

\begin{equation*} \lim_{x \rightarrow -1^-} f(x) = 2\text{.} \end{equation*}

The closed point represents the actual value of the function, \(f(-1)=0\text{.}\) The fact that the point is also connected to the graph segment to the right means that it can also be used as a limit. The closer values \(x \gt -1\) are to \(x=-1\text{,}\) the closer the values of \(f(x)\) will be to the \(y\)-value of \(y=0\text{.}\) That is, we have another limit:

\begin{equation*} \lim_{\substack{x \rightarrow -1 \\ x \gt -1}} f(x) = 0\text{,} \end{equation*}

which could also be written with a superscript on the limit

\begin{equation*} \lim_{x \rightarrow -1^+} f(x) = 0\text{.} \end{equation*}

In Figure 3.1.1, the limit on the left and the limit on the right had different values. This happens whenever the segments lead to different points. We say that the two-sided limit \(\displaystyle \lim_{x \to -1} f(x)\) does not exist. This is two-sided because the limit does not specify \(x \lt -1\) or \(x \gt -1\text{.}\) When segments on both sides of a point approach the same point, whether it is included or not, the two-sided limit would equal that common value.

Now, the phrasing that was used involving the idea of “closer” was not quite stated precisely. It is not required that closer values of \(x\) cause the \(y\) values to be progressivly closer. Rather, it is about being able to guarantee closeness. A limit is really about \(y\)-values being guaranteed to be close to the value of the limit so long as the \(x\)-values are close enough. Typically, smaller allowable errors for \(y\) will require smaller deviations for \(x\text{.}\) However, it is typical to use a phrase like “the value of \(f(x)\) approaches \(L\) as \(x\) approaches \(c\)” when stating a limit

\begin{equation*} \lim_{x \to c} f(x) = L\text{.} \end{equation*}

Example 3.1.2. Learning Objective L-1.

Given the graph of \(g(x)\) shown in Figure 3.1.3, determine each of the following values. If a value does not exist, explain why.

  1. \(\displaystyle \displaystyle \lim_{x \to -1^-} f(x)\)

  2. \(\displaystyle \displaystyle \lim_{x \to -1^+} f(x)\)

  3. \(\displaystyle \displaystyle \lim_{x \to -1} f(x)\)

  4. \(\displaystyle \displaystyle f(-1)\)

  5. \(\displaystyle \displaystyle \lim_{x \to 2^-} f(x)\)

  6. \(\displaystyle \displaystyle \lim_{x \to 2^+} f(x)\)

  7. \(\displaystyle \displaystyle \lim_{x \to 2} f(x)\)

  8. \(\displaystyle \displaystyle f(2)\)

Solution.

The first four values are associated with \(x=-1\) or \(x \to -1\text{.}\) Because there is a closed point at \((-1,1)\text{,}\) this value is associated with the value of the function at the point \(x=-1\text{.}\) We have \(f(-1) = 1\text{.}\) The limits are based on the end-points of the segments to the left and right of \(x=-1\text{.}\) To the left, we follow the decreasing linear graph to its end-point, illustrated by the open end-point at \((-1,-1)\text{.}\) The \(y\)-value of this point is the limit of the function using this left segment, and we write:

\begin{equation*} \lim_{x \to -1^-} f(x) = -1\text{.} \end{equation*}

To the right of \(x=-1\text{,}\) the graph uses the increasing linear segment. Again, the end-point of this segment at \(x=-1\) is \((-1,-1)\text{,}\) so that we know

\begin{equation*} \lim_{x \to -1^+} f(x) = -1\text{.} \end{equation*}

Because both segments terminate at the same point, this will be the value of the two-sided limit

\begin{equation*} \lim_{x \to -1} f(x) = -1\text{.} \end{equation*}

The second four values are associated with \(x=2\) or \(x \to 2\text{.}\) The closed point at \((2,2)\) gives the value of the function. The end-points of the segments of the graph to the left and right of \(x=2\) give the limits of the function from the sides. We have

\begin{gather*} f(2)=2\\ \lim_{x \to 2^-} f(x) = 2\\ \lim_{x \to 2^+} f(x) = 1 \end{gather*}

Because the left and right segments do not terminate at the same point, the two-sided limit does not exist. This is sometimes written in an abbreviated form

\begin{equation*} \lim_{x \to 2} f(x) \text{ d.n.e.}\text{.} \end{equation*}

(Notice there is no equality here.)

Subsection 3.1.2 Continuity

As mentioned in the introduction, continuity is a mathematical idea that on one hand captures the idea of connectedness. However, it is more importantly about how the values of the function change near a point.

Suppose we want to know the value of a function \(M\) (for mystery) at a point \(x=0\text{,}\) but the rule we have for evaluating the function has not yet been worked out for that particular value. What might we try to do? If the function is nice, we might try evaluating \(M(0.1)\) thinking that because \(x=0.1\) is close to \(x=0\text{,}\) then \(M(0.1)\) should be close to \(M(0)\text{.}\) That is what continuity is about.

Now \(x=0.1\) might not be close enough to \(x=0\) to create a good approximation. Maybe \(x=0.01\) or \(x=0.001\) would be better. Or maybe we need to go even closer before the approximation works out well. It might be that none of \(M(0.1)\text{,}\) \(M(0.01)\) or \(M(0.001)\) are individually close enough (by our judgment) to the value of \(M(0)\text{,}\) but we want to have assurances that if we just kept getting closer to \(x=0\text{,}\) we would eventually get good enough approximations. This is where the need for a limit comes from.

Definition 3.1.4. Continuity at a Point.

Suppose \(f\) is a function. “\(f\) is continuous at a point \(x=a\)” means:

\begin{equation*} \lim_{x \to a} f(x) = f(a)\text{.} \end{equation*}

Note that this requires \(f(a)\) is defined, the limit exists, and the limit and value of the function are the same.

There are one-sided versions:

  1. \(f\) is left-continuous at \(x=a\text{:}\)

    \begin{equation*} \lim_{x \to a^-} f(x) = f(a)\text{.} \end{equation*}

  2. \(f\) is right-continuous at \(x=a\text{:}\)

    \begin{equation*} \lim_{x \to a^+} f(x) = f(a)\text{.} \end{equation*}

A function that is continuous will be both left-continuous and right-continuous.

Notice how the natural idea of connectedness is related to continuity. A piecewise function that is disconnected has a break or jump. On one side or the other (or both), the value of the function won't agree with the limit because the segments of the graph don't both meet at the value of the function.

To verify that a function is continuous, we need to show that the defining statement (the equation) is true. That is, we need to know \(\displaystyle \lim_{x \to a} f(x)\) and \(f(a)\) and show that they are the same value. To verify that a function is not continuous, we need to show that the defining statement is false. Either the limit or the value of the function fail to exist or they have different values. This type of exercise is a practice in calculation and communication.

Example 3.1.5. Learning Objective L-2.

Consider the graph of the function from Example 3.1.2 shown again in Figure 3.1.6. Determine whether the function is continuous at \(x=-1\text{,}\) \(x=1\text{,}\) and \(x=2\text{,}\) justifying your reasoning in terms of the definition of continuity.

Solution.

Much of the computational work was done earlier in Example 3.1.2. We know:

\begin{gather*} \lim_{x \to -1} f(x) = -1\\ f(-1) = 1 \end{gather*}

Because \(\displaystyle \lim_{x \to -1} f(x) \ne f(-1)\text{,}\) we know that \(f(x)\) is not continuous at \(x=-1\text{.}\) We know:

\begin{gather*} \lim_{x \to 2^-} f(x) = 2\\ \lim_{x \to 2^+} f(x) = 1\\ f(2) = 2 \end{gather*}

Because \(\displaystyle \lim_{x \to 2^-} f(x) \ne \lim_{x \to 2^+} f(x)\text{,}\) we know \(\displaystyle \lim_{x \to 2} f(x)\) does not exist and thus can not equal \(f(2)\text{,}\) so \(f(x)\) is not continuous at \(x=2\text{.}\) Note that \(f(x)\) is left-continuous at \(x=2\text{.}\)

The previous example did not do any calculations for \(x=1\text{.}\) The graph shows it is smoothly connected here, in the middle of a continuous segment. That is, we can read from the graph \(f(1)=1\text{,}\) and because the graph is connected from the left and the right at this point, we also have

\begin{gather*} \lim_{x \to 1^-} f(x) = 1\\ \lim_{x \to 1^+} f(x) = 1 \end{gather*}

so that \(\displaystyle \lim_{x \to 1} f(x) = 1\text{.}\)

The last calculation in the example, testing the continuity at \(x=1\text{,}\) feels like circular logic. We wanted to show it is continuous and then used the idea that it was continuous to find the limits required to show continuity. Ultimately, the circularity depends on how you think through it. If you say, “The graph is obviously continuous, so all the values are the same”, then that is circular and bypasses some of the understanding. If instead you say, “To find the limit from the left, I need to look at the segment to the left of \(x=1\) and see where it approaches as we go back to \(x=1\)”, and it naturally goes back to \(f(1)\text{,}\) then you are thinking through the idea of a limit rather than just relying on broken vs unbroken graphs.

Ultimately, most limits that we evaluate will fall back to the idea of continuity. That is, given a limit of interest, we find a continuous function that will have the same limit and evaluate that function at the point of interest. This makes it feel like limits are just a complicated version of plugging in the value. Try to keep in mind that a limit is about coming from the left or right and seeing what those values approach, and that continuity means this is the same as the value at the point.
