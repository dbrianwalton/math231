Section 2.3 Inverse Functions

In the context of covarying variables, a function is a rule (operation) that predicts the value of the dependent variable (output) given the value of the independent variable (input). The algebra equation \(y=f(x)\) indicates that an independent variable \(x\) can lead to the value of the dependent variable \(y\) using the function \(f\text{.}\) We think of this as a map \(x \mapsto y\text{.}\)

We might be interested in predicting the reverse relation \(y \mapsto x\text{.}\) That is, it may also be the case that knowing \(y\text{,}\) we can calculate the value of \(x\text{.}\) Algebra and pre-calculus courses teach the idea that this is only possible when \(f\) is one-to-one, meaning that the value of \(y\) can only come from a single value of \(x\text{.}\) When this is true for a function \(f\text{,}\) we can talk about \(f^{-1}\) as a new function, called the inverse function.

This inverse function undoes the calculation of \(f\text{.}\) Suppose \(f : x \mapsto y\) has an inverse function \(f^{-1}: y \mapsto x\text{.}\) If we know that \(f(3) = 5\text{,}\) this means \(x=3 \mapsto y=5\text{.}\) The inverse function would map \(y=5 \mapsto x=3\) or \(f^{-1}(5)=3\text{.}\) This can be true when variables are involved as well: \(f(a) = 4\) means \(f^{-1}(4) = a\text{.}\) So if we are solving an equation \(f(x) = 4\text{,}\) the solution would be \(x=f^{-1}(4)\text{.}\)

You may remember learning something about the graph of an inverse function being a reflection of the graph of the original. If a point \((a,b)\) is on the graph \(y=f(x)\text{,}\) this means \(f(a)=b\text{.}\) The inverse function has \(f^{-1}(b)=a\) so that \((b,a)\) is on the graph of \(f^{-1}\text{.}\) Every point on the graph of \(f\) corresponds to a point on the graph of \(f^{-1}\) where the coordinates have reversed. Visually, this creates a reflection across the line \(y=x\text{.}\)

The idea of undoing the function is demonstrated through composition. If we start with an equation

\begin{equation*} y=f(x)\text{,} \end{equation*}

look at what happens if we use composition by \(f^{-1}\) to both sides of the equation:

\begin{equation*} f^{-1}(y)=f^{-1}\big(f(x)\big)\text{.} \end{equation*}

We know that \(f^{-1}(y) = x\) is the inverse map, so the equation becomes

\begin{equation*} x=f^{-1}\big(f(x)\big)\text{.} \end{equation*}

That is, a function composed with its inverse function is the same as doing nothing to the input, an operation called the identity function.

Definition 2.3.1. Identity Function.

The identity function is a function that returns the same value as its input, \(\mathrm{id}(x) = x\text{.}\)

Because some functions may not be represented using formulas, inverse functions need a definition other than rules on how to manipulate formulas. Mathematicians have found that the idea of composition creating the identity function is useful and have adopted that as the official definition of an inverse function.

Definition 2.3.2. Inverse Function.

A function \(f\) with domain \(D_f\) and a function \(g\) with domain \(D_g\) are inverse functions (to each other) if:

  • For every \(x \in D_f\text{,}\) \(g \circ f(x) = x\)

  • and for every \(x \in D_g\text{,}\) \(f \circ g(x) = x\text{.}\)

We have three learning objectives relating to inverse functions:

  • Use inverse functions as a way to cancel a function in an equation or expression.

  • Verify whether two functions are inverses to each other by computing and simplifying their compositions and determining if the result in each case is the identity function.

  • Starting with the formula of an invertible function, find the formula of its inverse function by finding the inverse map.

Subsection 2.3.1 Using Inverse Functions

The fundamental idea of inverse functions is that whenever we have composition \(f \circ f^{-1}\) or \(f^{-1} \circ f\text{,}\) this is identical to the identity function. We explore this idea by solving an equation where some of the operations are not arithmetic but are invertible functions.

Recall that the method of isolation is a strategy to solve an equation by applying the same operation on both sides of an equation to cancel away operations until the unknown variable is isolated. Early examples of this strategy focused on operations involving basic arithmetic. We can now add to this strategy the use of inverse functions to cancel away operations represented by functions.

Example 2.3.3.

Suppose that \(f\) and \(g\) are inverse functions. Solve the equation \(f(2x) = 3\) in terms of values of \(f\) and \(g\text{.}\)

Solution.

Problems like this may feel unnatural because we don't even know what these functions \(f\) and \(g\) might be. Suppose it were a familiar function, say \(\sqrt{2x} = 3\text{.}\) We would want to get rid of the square root and would do this by squaring both sides. That is, we would use an inverse operation. We will do the same thing here, except the inverse operation to \(f\) as a function is the function \(g\text{.}\)

Starting with the original equation, we use composition with the inverse \(g\) on both sides of the equation:

\begin{align*} f(2x) &= 3\\ g(f(2x)) &= g(3). \end{align*}

The composition \(g \circ f\) reduces to the identity function, which just leaves the input expression. The equation therefore becomes:

\begin{align*} g(f(2x)) &= g(3)\\ 2x &= g(3)\\ x &= \frac{1}{2} g(3). \end{align*}

Because we have isolated \(x\) and the value is a formula involving \(g(3)\text{,}\) we are done.

Sometimes we need to use additional operations in order to isolate the function of an expression before using the inverse function.

Example 2.3.4.

Suppose that \(S\) and \(T\) are inverse functions. Solve the equation \(3T(2x+1) - 4 = 1\) in terms of values of \(S\) and \(T\text{.}\)

Solution.

The method of isolation should work because the equation involves the variable \(x\) in only one place. We will identify the last operation on the side involving the variable and apply inverse operations until we have finished isolating \(x\text{.}\)

Equation Last Operation
\(3T(2x+1)-4 = 1\) Add: \(-4\)
\(3T(2x+1) = 1+4 = 5\) Multiply: \(3\)
\(T(2x+1) = \frac{1}{3} \cdot 5 = \frac{5}{3}\) Function: \(T\)
At this point, we will use the inverse function \(S\text{.}\)
Equation Last Operation
\(T(2x+1) = \frac{1}{3} \cdot 5 = \frac{5}{3}\) Function: \(T\)
\(2x+1 = S(\frac{5}{3})\) Add: \(1\)
\(2x = S(\frac{5}{3}) - 1\) Multiply: \(2\)
\(x = \frac{1}{2}\big(S(\frac{5}{3}) - 1\big)\) Isolated
We have found the solution \(x = \frac{1}{2}\big(S(\frac{5}{3}) -1\big)\text{.}\)

Notice that once we had \(S(\frac{5}{3})\) as a term on the right hand side, that represented a single number. None of the remaining steps had any interaction with the value \(\frac{5}{3}\) since that was the input to the expression, not the value of the function's output.

Subsection 2.3.2 Verifying Inverse Functions

The learning objective that this subsection focuses on is on the what it means to verify that two functions satisfy the definition of being inverses. The emphasis in your past was likely about finding an inverse function. In mathematics, definitions play a critical role in creating the language we use to describe objects. A definition is a statement that can be true or false. To show that something satisfies a definition, we need to show that the statement in the definition is true.

Consequently, to verify that two functions \(f\) and \(g\) are inverses, we need to construct the compositions \(f \circ g(x)\) and \(g \circ f(x)\text{.}\) If both of these compositions reduce to the identity function, then they satisfy the definition. Along the way, we need to practice simplify formulas that involve compositions. This is the second reason that we have this learning objective—practicing thinking about the structure of formulas and how the order of operations impacts what does or does not simplify.

As we look at a formula, we should identify the order in which the different operations take place. When we find operations that are inverses one after another, they can be canceled. But operations that are inverses of one another but occur out of sequence will not cancel.

Example 2.3.5.

Suppose \(h\) is an invertible function. Simplify each of the following formulas as much as possible:

  • \(\displaystyle h(h^{-1}(x-1))+1\)

  • \(\displaystyle \frac{1}{2} h(h^{-1}(2x-1))+1\)

Solution.

Consider the formula \(h(h^{-1}(x-1))+1\text{.}\) We can follow the same process as when we wrote a formula as a composition in Example 2.2.7 to work on Learning Objective F-5.

\(h(h^{-1}(\boxed{x}-1))+1\) find the variable
\(h(h^{-1}(\boxed{x-1}))+1\) add -1
\(h(\boxed{h^{-1}(x-1)})+1\) composition with \(h^{-1}\)
\(\boxed{h(h^{-1}(x-1))}+1\) composition with \(h\)
\(\boxed{h(h^{-1}(x-1))+1}\) add 1
Notice that in one spot we add -1 and in another we add +1. These are inverse operations. However, they were not applied in sequence so they do not cancel at this point.

On the other hand, the composition with \(h\) and \(h^{-1}\) do occur in sequence after one another and so they cancel to leave the identity function:

\begin{align*} h(h^{-1}(x-1))+1 &= \boxed{h(h^{-1}(x-1))}+1\\ &= \boxed{\mathrm{id}(x-1)}+1\\ &= (x-1)+1\\ &= x \end{align*}

When we now consider the second expression, \(\frac{1}{2} h(h^{-1}(2x-1))+1\text{,}\) we look for inverse operations that are sequentially in order. We could think through the order of operations as above. Or we can directly look for sequential operations that cancel. There are three potential operations that cancel:

  • multiplication by \(\frac{1}{2}\) and by 2,

  • addition with \(-1\) and \(+1\text{,}\)

  • composition of \(h\) and \(h^{-1}\text{.}\)

However, notice that the operations of multiplication by 2 and addition of -1 are inside the composition with \(h^{-1}\text{:}\)

\begin{equation*} \frac{1}{2} h(h^{-1}(\boxed{2x-1}))+1\text{.} \end{equation*}

This prevents them being canceled. However, the composition of \(h\) and \(h^{-1}\) are back-to-back and cancel to an identity function,

\begin{align*} \frac{1}{2} h(\boxed{h^{-1}(2x-1)})+1 &= \frac{1}{2} (\boxed{2x-1})+1\\ &= \frac{1}{2}(2x-1)+1 \end{align*}

Now we distribute the multiplication by \(\frac{1}{2}\) over the addition in \(2x-1\text{.}\)

\begin{align*} \frac{1}{2}(2x-1)+1 &= \frac{1}{2}(2x) - \frac{1}{2}(1) + 1\\ &= x - \frac{1}{2} + 1\\ &= x + \frac{1}{2}. \end{align*}

When verifying if two functions satisfy the definition of being inverses, we must first create the two compositions. Then we use the process illustrated in the previous example to simplify as much as possible. If the formulas of the compositions both simplify to just \(x\text{,}\) then we have the composition being equal to the identity function as required by the definition. Otherwise, they do not satisfy the definition and are not inverses.

Example 2.3.6. Learning Objective F-8.

Verify whether \(f(x)\) and \(g(x)\) satisfy the definition of inverse functions, where each is defined in terms of an invertible function \(h\) by:

\begin{align*} f(x) &= 5 h^{-1}\left(\frac{x+1}{2}\right) + 3\\ g(x) &= 2 h\left(\frac{x-3}{5}\right) - 1 \end{align*}
Solution.

Let us start by clearing identifying our goal. The functions satisfy the definition of inverse functions if it is true that \(f \circ g(x) = x\) and \(g \circ f(x) = x\text{.}\) Consequently, we need to create the formulas for \(f \circ g(x) = f \big(g(x)\big)\) and \(g \circ f(x) = g \big(f(x)\big)\text{.}\) Then we will need to simplify the formulas, canceling terms where appropriate, and see if they reduce to \(x\) or something more complicated.

We start with the first composition to find

\begin{align*} f \circ g(x) &= f\left(g(x)\right)\\ &= 5 h^{-1}\left(\frac{\boxed{g(x)}+1}{2}\right) + 3\\ &= 5 h^{-1}\left(\frac{\boxed{2 h\left(\frac{x-3}{5}\right) - 1}+1}{2}\right) + 3 \end{align*}

We are looking for inverse operations that are back-to-back steps. Notice that the last operation in \(g(x)\) was adding -1 and the first operation in \(f(x)\) was adding 1. These cancel one another:

\begin{align*} f \circ g(x) &= 5 h^{-1}\left(\frac{2 h\left(\frac{x-3}{5}\right) - \cancel{1}+\cancel{1}}{2}\right) + 3\\ f \circ g(x) &= 5 h^{-1}\left(\frac{2 h\left(\frac{x-3}{5}\right)}{2}\right) + 3 \end{align*}

Now we see that in the input to \(h^{-1}\text{,}\) there is multiplication by 2 and division by 2 back-to-back that cancel:

\begin{align*} f \circ g(x) &= 5 h^{-1}\left(\frac{\cancel{2} h\left(\frac{x-3}{5}\right)}{\cancel{2}}\right) + 3\\ f \circ g(x) &= 5 h^{-1}\left( h\left(\frac{x-3}{5}\right)\right) + 3 \end{align*}

Now we have a composition of inverses \(h^{-1} \circ h\) that reduces to the identity, after which we use algebra to simplify:

\begin{align*} f \circ g(x) &= 5 h^{-1} \circ h\left(\frac{x-3}{5}\right) + 3\\ &= 5 \left(\frac{x-3}{5}\right) + 3\\ &= \cancel{5} \left(\frac{x-3}{\cancel{5}}\right) + 3\\ &= x-3 + 3\\ &= x \end{align*}

We have shown that \(f \circ g(x) = x\text{,}\) which is half of the required work to show they are inverses.

We need to check the reverse process.

\begin{align*} g \circ f(x) &= g\left(f(x)\right)\\ &= 2 h\left(\frac{\boxed{f(x)}-3}{5}\right) - 1\\ &= 2 h\left(\frac{\boxed{5 h^{-1}\left(\frac{x+1}{2}\right) + 3}-3}{5}\right) - 1\\ &= 2 h\left(\frac{5 h^{-1}\left(\frac{x+1}{2}\right) + \cancel{3} -\cancel{3}}{5}\right) - 1\\ &= 2 h\left(\frac{\cancel{5} h^{-1}\left(\frac{x+1}{2}\right)}{\cancel{5}}\right) - 1\\ &= 2 h\left( h^{-1}\left(\frac{x+1}{2}\right)\right) - 1\\ &= 2 h \circ h^{-1}\left(\frac{x+1}{2}\right) - 1\\ &= \cancel{2} \left(\frac{x+1}{\cancel{2}}\right) - 1\\ &= x+1 - 1\\ &= x \end{align*}

We have now also shown that \(g \circ f(x) =x\text{.}\)

Because \(f \circ g(x)=x\) and \(g \circ f(x)=x\text{,}\) the functions \(f\) and \(g\) satisfy the definition of inverse functions. We now know that \(f^{-1} = g\) and \(g^{-1}=f\text{.}\)

Subsection 2.3.3 Finding Inverse Functions

Because an inverse function \(f^{-1}\) is designed specifically to undo the operations of \(f\text{,}\) we can often find a formula for \(f^{-1}\) by either literally reversing the operations or by solving an equation.

Example 2.3.7.

Find a formula for \(f^{-1}(x)\) given the function

\begin{equation*} f(x) = \frac{2}{3x+1} - 4\text{.} \end{equation*}
Solution.

We know that if \(y=f(x)\text{,}\) then an inverse function would give \(x=f^{-1}(y)\text{.}\) So one approach to finding a formula is to solve the equation to isolate \(x\) and use the formula in terms of \(y\) as the inverse formula. We start with the equation

\begin{equation*} y = f(x) = \frac{2}{3x+1} - 4\text{.} \end{equation*}

We now try to isolate the variable \(x\text{:}\)

\begin{align*} y &= \frac{2}{3x+1} - 4\\ y + 4 &= \frac{2}{3x+1} - \cancel{4} + \cancel{4}\\ y + 4 &= \frac{2}{3x+1} \end{align*}

At this point, it is easiest to apply the reciprocal (multiplicative inverse) to both sides:

\begin{align*} y + 4 &= \frac{2}{3x+1}\\ \frac{1}{y + 4} &= \frac{3x+1}{2}\\ 2 \cdot \frac{1}{y + 4} &= \cancel{2} \cdot \frac{3x+1}{\cancel{2}}\\ \frac{2}{y + 4} &= 3x+1\\ \frac{2}{y + 4} - 1 &= 3x\\ \frac{1}{3} \left(\frac{2}{y + 4} - 1\right) &= x \end{align*}

Because we finished solving for \(x\) and have a formula in terms of \(y\text{,}\) that formula can be used to define \(f^{-1}(y)\text{.}\) If we think of \(x\) as an abstract variable and not a physical quantity, we can use that in place of \(y\) to get

\begin{equation*} f^{-1}(x) = \frac{1}{3} \left( \frac{2}{y+4} - 1\right)\text{.} \end{equation*}

Alternatively, the inverse function can be constructed by literally undoing the operations occuring in the function \(f\text{.}\) The operations applied by \(f\text{:}\)

\(f(x) = 2 \cdot \frac{1}{3 \boxed{x}+1} - 4\) start with input
\(f(x) = 2 \cdot \frac{1}{\boxed{3x}+1} - 4\) multiply by 3
\(f(x) = 2 \cdot \frac{1}{\boxed{3x+1}} - 4\) add 1
\(f(x) = 2 \cdot \boxed{\frac{1}{3x+1}} - 4\) reciprocal
\(f(x) = 2 \cdot \boxed{\frac{1}{3x+1}} - 4\) multiply by 2
\(f(x) = 2 \cdot \boxed{\frac{1}{3x+1}} - 4\) subtract 4
The operations applied by \(f^{-1}\) need to undo the most recent operations:
start with input x
add 4 x+4
divide by 2 \frac{x+4}{2}
reciprocal \frac{2}{x+4}
subtract 1 \frac{2}{x+4} - 1
divide by 3 \frac{1}{3}\left(\frac{2}{x+4} - 1\right)
This last formula is \(f^{-1}(x)\text{.}\) Notice that solving an equation by isolation is exactly the same process as reversing step-by-step operations. If we use different methods to solve the equation, we are likely to find a different representation.

To reinforce the abstract notation of unknown functions, the learning objective assessments will typically involve calculations involving some invertible function. This will later be useful when you are working with exponential functions and their inverses (logarithms) and trigonometric functions and their inverses.

Example 2.3.8. Learning Objective F-9.

Suppose that \(h\) is an invertible function and that

\begin{equation*} f(x) = \frac{5}{h(2x)+1}-3\text{.} \end{equation*}

Find a formula for \(f^{-1}\) in terms of \(h\) and \(h^{-1}\text{.}\)

Solution.

The major difference in this example to the previous example is the presence of the function \(h\text{.}\) We will follow the same process: create an equation \(y=f(x)\text{,}\) and because \(x=f^{-1}(y)\text{,}\) we will use the method of isolation to solve for \(x\text{.}\)

\begin{gather*} y = \frac{5}{h(2x)+1}-3\\ y+3= 5 \cdot \frac{1}{h(2x)+1}\\ \frac{y+3}{5} = \frac{1}{h(2x)+1}\\ \frac{5}{y+3} = h(2x)+1\\ \frac{5}{y+3} - 1 = h(2x) \end{gather*}

We could combine the terms on the left to a single fraction if we use a common denominator:

\begin{gather*} \frac{5}{y+3} - \frac{y+3}{y+3} = h(2x)\\ \frac{5-(y+3)}{y+3} = h(2x)\\ \frac{2-y}{y+3} = h(2x) \end{gather*}

At this point, we need to cancel the composition with \(h\) and do this by using a composition with \(h^{-1}\) on both sides.

\begin{gather*} h^{-1}\left(\frac{2-y}{y+3}\right) = h^{-1} \circ h(2x)\\ h^{-1}\left(\frac{2-y}{y+3}\right) = 2x\\ \frac{1}{2} h^{-1}\left(\frac{2-y}{y+3}\right) = x \end{gather*}

This formula gives us the relation \(y \mapsto x\) which must be \(f^{-1}\text{:}\)

\begin{equation*} f^{-1}(y) = \frac{1}{2} h^{-1}\left(\frac{2-y}{y+3}\right)\text{.} \end{equation*}

If we considered the variables \(x\) and \(y\) as abstract values rather than physical measurements, we can replace \(y\) with any variable:

\begin{equation*} f^{-1}(x) = \frac{1}{2} h^{-1}\left(\frac{2-x}{x+3}\right)\text{.} \end{equation*}
