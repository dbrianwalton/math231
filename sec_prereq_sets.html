Section 1.2 Sets and Interval Notation

In typical algebra classes, sets are mainly used in describing solutions to equations or inequalities. This will still be true in calculus, but sets will play a more important role than just summarizing solutions. As we learn more about functions, we will learn that functions are defined in terms of sets. In addition, functions will also be described by various attributes (adjectives) that will be characterized by sets. In preparation, we will need to become practiced in describing and working with sets.

Subsection 1.2.1

Sets are collections of objects from some universe of possibilities. In calculus, we primarily focus on sets of real numbers. A set can be a described by listing its members, when that is possible. If a set has a finite number of values (meaning the list has a specific number of values), then we can list the included values inside of curly braces, as in \(\{2, 3, 5, 7\}\text{.}\)

A set can be assigned a name, such as \(P = \{2,3,5,7\}\text{.}\) The number 2 is in \(P\text{,}\) which we write \(2 \in P\text{.}\) The number 4 is not in \(P\text{,}\) which we write \(4 \not \in P\text{.}\) A set that is empty would have no values, \(\{\}\text{,}\) and sometimes is written with a special symbol \(\emptyset\text{.}\)

Calculus is more interested in intervals. The real numbers form a continuum of values. Between any two real numbers, there is always another number (e.g., the midpoint). In fact, there are infinitely many numbers. Some of them are rational numbers, meaning a ratio of integers. There are also even more irrational numbers, numbers that can never be written as a ratio of integers.

Definition 1.2.1. Intervals.

An interval is a set of real numbers consisting of all numbers between two end-points, possibly including the end-points.

  • \((a,b) = \{x : a \lt x \lt b\}\) is called the open interval and excludes both end-points.

  • \([a,b] = \{x : a \le x \le b\}\) is called the closed interval and includes both end-points.

  • \((a,b] = \{x : a \lt x \le b\}\) is neither open nor closed because it excludes \(a\) and includes \(b\text{.}\)

  • \([a,b) = \{x : a \le x \lt b\}\) is neither open nor closed because it includes \(a\) and excludes \(b\text{.}\)

What are some important take-away messages about how intervals are defined?

The end-points of the intervals always need to be ordered with the greater value listed second. So \((3,5)\) is a valid interval but \((5,3)\) is not. If an interval goes on forever, such as \(x \gt 2\text{,}\) we use infinity as an end-point, as in \((2,\infty)\text{.}\) Similarly, \(x \le -1\) would create the interval \((-\infty,-1]\) Be aware that infinity is never included as an end-point because it is not actually a real number.

Next, a critical understanding requires recognizing that any non-empty interval has infinitely many different numbers inside. Learners often want to think only in terms of integer (whole-number) values, and this interferes with this understanding. For example, some learners want to think of \(0 \lt x \lt 5\) as only including integers \(\{1,2,3,4\}\) or think of \(0 \lt x \lt 1\) as being empty (since there aren't any integers between 0 and 1). Be sure to think about fractional values. For example, \(0.1 \in (0,1)\text{.}\)

We can visualize intervals and sets graphically by highlighting the points on a number line. Since the number line itself is a line, we often draw the interval or set as a thicker line over or next to the number line. Individual points are indicated with solid points. End-points that are not included are indicated with open circles or parentheses. It is also helpful to label the locations of the end-points, especially if the number line itself does not have the location clearly marked.

Example 1.2.2. Learning Outcome P-3.

Illustrate the intervals \((-4,-\pi]\text{,}\) \([0,\sqrt{5})\text{,}\) and \([\sqrt{2},\infty)\text{,}\) and, for each interval, identify at least one non-integer real number that is in the interval.

Solution.

When thinking about numbers on the number line, it is useful to first identify their approximate decimal representation so that we can correctly find their position. The value \(\pi\) has an approximate decimal representation of \(\pi \approx 3.14159\text{.}\) When thinking about \(-\pi\text{,}\) we need to be careful that we put this to the left of -3 on the number line.

For the other non-integer values, we have \(\sqrt{2} \approx 1.41421\) (rounds down) and \(\sqrt{5} \approx 2.23607\) (rounds up). I made mention of rounding because this technically means that \(\sqrt{2}\) is between 1.41421 and 1.41422, while \(\sqrt{5}\) is between 2.23606 and 2.23607.

Once we know where to place the numbers, we can draw our number line. We really only need to include that part of the number line that is useful to illustrate our interval. For example, the interval \((-4,-\pi]\) might include a few integers to the left and right of the interval but does not need more than that.

What numbers are inside the interval? What numbers are outside? You should be able to think about this critically. Because \(-\pi \approx -3.14159\text{,}\) you should be able to reason that \(-\pi\) is between -3.2 and -3.1. Consequently, -3.2 is inside the interval but -3.1 is outside the interval. Similarly, -3.15 is inside the interval while -3.14 is outside.

The other two intervals are constructed similarly. Be careful about end-points. The value \(\sqrt{2}\) is in the interval \([\sqrt{2],\infty)\) because the end-point is included. However, if we used a decimal approximation \(\sqrt{2} \approx 1.414\text{,}\) the value 1.414 is not in the interval because this is rounding down from the exact value and so is to the left of the actual end-point.

Subsection 1.2.2

Intervals and singletons (sets with a single value) form the elementary building blocks for more complicated sets. Suppose we were looking at the graph \(y=x^2-1\) and wanted to describe the set of values \(x\) so that the graph had \(y \gt 0\text{.}\) There are two disjoint pieces of the graph that have \(y \gt 0\text{.}\) One piece is where \(x \lt -1\) and the other is where \(x \gt 1\text{.}\) This is one set, but it is composed of two disjoint intervals. To make a statement about \(x\) being in the set, we would need to say \(x \lt -1\) or \(x \gt 1\text{.}\) Mathematically, we say that the set is a union of intervals and write \((-\infty,-1) \cup (1,\infty)\text{.}\)

Definition 1.2.6. Union of Sets.

Given two sets \(A\) and \(B\text{,}\) the union of \(A\) and \(B\) is a new set \(A \cup B\) defined as all values \(x\) so that \(x \in A\) or \(x \in B\text{.}\)

If we are given more than two sets, the union of the collection is a new set whose values must be in at least one of the sets that were combined.

A related concept to union is intersection. An intersection considers where sets overlap and includes shared values from the sets.

Definition 1.2.7. Intersection of Sets.

Given two sets \(A\) and \(B\text{,}\) the intersection of \(A\) and \(B\) is a new set \(A \cap B\) defined as all values \(x\) so that \(x \in A\) and \(x \in B\text{.}\)

If we are given more than two sets, the intersection of the collection is a new set whose values must be in all of the sets that were combined.

If we wanted to be careful, we would also need to learn that there is an order of set operations. Intersection is like multiplication and union is like addition. So if we had three sets \(A\text{,}\) \(B\text{,}\) and \(C\text{,}\) the notation \(A \cup B \cap C\) would mean the union of \(A\) with \(B \cap C\text{.}\) To avoid worrying about this, we will use parentheses to make things clear. For that example, we would have an unambiguous notation by writing \(A \cup (B \cap C)\text{,}\) which is very different from \((A \cup B) \cap C\text{,}\) the intersection of \(A \cup B\) with \(C\text{.}\)

When working with sets defined by intervals, it can help greatly to put the sets side by side with a common number line. If you have access to color, that can help you keep track of the sets as well. We will see that if intervals overlap, the union will be a new interval that contains both parts.

Example 1.2.8.

Given sets \(A = (-2,1]\text{,}\) \(B = [3,5)\) and \(C = (0,2)\text{,}\) find the unions \(A \cup B\text{,}\) \(A \cup C\) and \(B \cup C\text{.}\) Then find the intersections \(A \cap B\text{,}\) \(A \cap C\) and \(B \cap C\text{.}\) Write the sets using simplified unions of disjoint sets.

Solution.

To visualize this, let us begin by creating a number line and showing each of the three sets \(A,B,C\text{.}\) To emphasize that these are being considered as different sets, they are colored differently and placed on different layers.

To find \(A \cup B\text{,}\) we need to describe the set of all values that are in \(A\) or \(B\text{.}\) The two sets do not overlap at all, so the union will consist of the same disjoint intervals, but now combined as a single set,

\begin{equation*} A \cup B = (-2,1] \cup [3,5)\text{.} \end{equation*}

To find \(A \cup C\text{,}\) we are interested in values that are in at least one of the sets. Notice that these sets overlap, meaning there are some values that are in both intervals (in their intersection). Consider the boundaries of this overlap. We have \(0 \in A\) and \(0 \not in C\text{,}\) but it is in at least one set and so \(0 \in A \cup C\text{.}\) Similarly, we have \(1 \in A\) and \(1 \in C\text{.}\) By being in both sets, it is necessarily in at least one and so \(1 \in A \cup C\text{.}\) All values between them are in both sets and so are in the union. The union of these overlapping intervals will be a new interval that contains both,

\begin{equation*} A \cup C = (-2,2]\text{.} \end{equation*}

The sets \(B\) and \(C\) do not overlap and so are disjoint. The union is simply written

\begin{equation*} B \cup C = (0,2] \cup [3,5)\text{.} \end{equation*}

Notice that the order of intervals in the union is reading the number line left-to-right, and does not match the order that \(B\) and \(C\) were listed in the union. We could have written

\begin{equation*} B \cup C = [3,5) \cup (0,2]\text{,} \end{equation*}

but this ordering is harder to transfer to a number line and so is not preferred.

The intersections are calculated by finding where the sets overlap (values in both intervals). The only intervals here that overlapped were \(A\) and \(C\text{.}\) As noted earlier \(0 \not in C\text{,}\) so we have

\begin{equation*} A \cap C = (0,1]\text{.} \end{equation*}

The other two intersections are empty:

\begin{gather*} A \cap B = \{\}\\ B \cap C = \{\} \end{gather*}

To understand union and intersection and to reinforce the interpretation of individual intervals, you should be able to think through the union and intersection of compound sets.

Example 1.2.12. Learning Outcome P-4.

Given the sets \(A = (-\infty,-1] \cup (1,4]\) and \(B = [-1,1) \cup (3,5)\text{,}\) find \(A \cup B\) and \(A \cap B\text{,}\) writing them as a union of disjoint sets.

Solution.

The solution follows the same patterns as our example involving individual intervals. The major difference is that we are starting with compound sets. When we graph our representations for \(A\) and \(B\text{,}\) we will want to be clear that each set has multiple parts.

To find the union, we look for all values that are in at least one of the sets. With the number line representations, we can scan from left to right. The set \(A\) includes all values until -1, including that value; the set \(B\) immediately begins, also including -1, and continuing until 1. The union must include all of these values. When we look at the value \(1\text{,}\) we see \(1 \not in A\) and \(1 \not in B\text{,}\) so \(1 \not in A \cup B\text{.}\) But then \(A\) starts up immediately and includes all values until 4. The set \(B\) includes all values from 3 until 5. This overlap reduces to a single interval \((1,5)\text{.}\)

Altogether, this means

\begin{equation*} A \cup B = (-\infty,1) \cup (1,5)\text{.} \end{equation*}

Notice how the union has a gap of a single value.

The intersection of the sets looks for points that are in both sets. We look at the original sets to see where they overlap. Both sets include -1, but none of the values immediately to the left or right are in both sets. This will be an isolated value in our intersection. The value 1 might also look like it is in both sets, but in fact is in neither set. The next set of values in the intersection is the interval from -3 to 4. Because \(-3 \not \in B\text{,}\) it will not be in the intersection. On the other hand, \(4 \in A\) and \(4 \in B\text{,}\) so it will be in the intersection.

Interpreting our results, we have

\begin{equation*} A \cap B = \{-1\} \cup (3,4]\text{.} \end{equation*}
